## 외형

```
체형:
외모 특징:
주로 입는 옷/스타일:

```

---

## 성격 및 말투

### 핵심 성격 키워드 (3~5개)
```
예: 냉철함 / 의리 / 자존심 강함 / 내면은 따뜻함
```

### 말투 특징
```
- 존댓말/반말 여부:
- 자주 쓰는 표현이나 말버릇:
- 감정이 격해질 때 달라지는 점:
- 절대 쓰지 않는 표현:
```

### 대화 예시 (AI 학습용)
```
평소:    "..."
화났을 때: "..."
당황했을 때: "..."
좋아하는 사람에게: "..."
```

## 보이스 시트 (Voice Sheet)

> 산문 스타일리스트 에이전트가 이 캐릭터의 대사를 쓸 때 참조합니다.
> 캐릭터의 '목소리'를 일관되게 유지하는 핵심 데이터입니다.

```
문장 리듬: (예: 짧고 끊김 / 길고 유려함 / 불규칙)
자주 쓰는 표현 / 말버릇: (예: "됐어", 말 끝에 "...이랄까")
절대 쓰지 않는 표현:
격식체 / 반말 / 혼용 여부:
감정이 격해질 때 달라지는 것: (예: 말수가 줄어듦, 존댓말로 전환)
```

### 상황별 대사 예시 (AI 학습용)
```
평소:           "..."
화났을 때:       "..."
당황했을 때:     "..."
좋아하는 사람에게: "..."
극도로 위기일 때:  "..."
```
