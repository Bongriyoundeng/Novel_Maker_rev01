# 캐릭터 프로필 템플릿

> 캐릭터마다 별도 파일로 복사해서 사용하세요. AI에게 캐릭터 파일을 제공하면 말투와 행동 일관성이 유지됩니다.
> **인물 형상화 원칙:** 모든 설정은 '스토리의 중심축(서사)'과 직접 연결되어 사건을 유발하거나 해결하는 데 기여하는 것만 최우선으로 배치합니다.

---

## 인물 형상화 6가지 특징 체크

> 독자의 뇌리에 인물을 각인시키기 위해 전략적으로 설계합니다.
> **전부 채울 필요 없음 — 서사와 연결된 것만 우선순위를 두세요.**

```
[ ] 고정관념 활용: 대중이 이 캐릭터 유형에 가진 기대치 (비틀 수 있음)
    →

[ ] 신체적 특징: 독자가 한 번에 기억할 시각적 포인트 1가지
    →

[ ] 말투와 행동: 이 인물만의 고유 습관·언변 (대사만 봐도 알 수 있어야 함)
    →

[ ] 상황 설정: 인물이 처한 특수한 환경 (행동을 제약하거나 가능하게 함)
    →

[ ] 특별한 능력: 비일상적 힘 — 반드시 진짜 욕망/두려움과 연결되어 등장
    →

[ ] 태생적 특징: 이름·혈통·출신에 운명이 담긴 경우
    →
```


## 성격 및 말투

### 핵심 성격 키워드 (3~5개)
```
예: 냉철함 / 의리 / 자존심 강함 / 내면은 따뜻함
```

### 말투 특징
```
- 존댓말/반말 여부:
- 자주 쓰는 표현이나 말버릇:
- 감정이 격해질 때 달라지는 점:
- 절대 쓰지 않는 표현:
```

### 대화 예시 (AI 학습용)
```
평소:    "..."
화났을 때: "..."
당황했을 때: "..."
좋아하는 사람에게: "..."
```

## 배경 스토리

```
과거:
현재 상황:
숨겨진 비밀:
트라우마:
```

> **과거 설정의 목적:** 인물의 돌발 행동에 "저 인물이라면 과거의 상처 때문에 충분히 그럴 수 있다"는 개연성을 부여합니다.

## 양면성 & 입체성

```
외적 면모 (겉으로 보이는 모습):

내적 과거 (숨겨진 상처·결핍):

충돌 지점 (외적 면모와 내적 과거가 충돌할 때 어떤 행동이 나오는가):
```

### 히어로-빌런 양면성 (악역 또는 갈등 인물에 해당 시 작성)

```
공유하는 트라우마 (주인공과 같은 상처를 가지는가):

운명을 가른 선택 (같은 상황에서 왜 다른 선택을 했는가):

빌런의 논리 (독자에게 설득력을 주는 빌런만의 정당성):
```

## 성장 아크 (12단계 서사 기준)

```
[변화 전]
1단계 (일상 세계) 상태:
2단계 (소명) 계기:
3단계 (조건 부재) 무력한 상태:

[변화 과정]
4~6단계 (전개) 상태:
7~8단계 (위기·시련) 상태:
  - 이 시점 변화 저항(항상성):

[변화 후]
9~10단계 (보상·정면승부) 상태:
11단계 (각성) 깨달은 것:
12단계 (귀환) 결말 상태:
```

## 보이스 시트 (Voice Sheet)

> 산문 스타일리스트 에이전트가 이 캐릭터의 대사를 쓸 때 참조합니다.
> 캐릭터의 '목소리'를 일관되게 유지하는 핵심 데이터입니다.

```
문장 리듬: (예: 짧고 끊김 / 길고 유려함 / 불규칙)
자주 쓰는 표현 / 말버릇: (예: "됐어", 말 끝에 "...이랄까")
절대 쓰지 않는 표현:
격식체 / 반말 / 혼용 여부:
감정이 격해질 때 달라지는 것: (예: 말수가 줄어듦, 존댓말로 전환)
```

### 상황별 대사 예시 (AI 학습용)
```
평소:           "..."
화났을 때:       "..."
당황했을 때:     "..."
좋아하는 사람에게: "..."
극도로 위기일 때:  "..."
```
