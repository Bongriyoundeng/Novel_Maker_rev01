---

## 성격 및 말투

### 핵심 성격 키워드 (3~5개)
```
예: 냉철함 / 의리 / 자존심 강함 / 내면은 따뜻함
```

### 말투 특징
```
- 존댓말/반말 여부:
- 자주 쓰는 표현이나 말버릇:
- 감정이 격해질 때 달라지는 점:
- 절대 쓰지 않는 표현:
```

### 대화 예시 (AI 학습용)
```
평소:    "..."
화났을 때: "..."
당황했을 때: "..."
좋아하는 사람에게: "..."
```
