# 장면 4단계 레이어 템플릿

> **산문 스타일리스트 에이전트**가 각 장면을 집필할 때 사용합니다.
> 한 번에 모든 요소를 처리하지 않고 4단계로 나눠 각 요소의 밀도를 높입니다.
> AI가 한 번에 너무 많은 정보를 처리할 때 발생하는 '지능의 분산'을 방지합니다.

---

## 장면 기본 정보

```
장면 번호:

POV (시점 캐릭터):
장소:
시간 (낮/밤, 계절, 날씨):
등장인물:
이 장면의 서사 목적: (비트 시트의 해당 Scene에서 가져오기)
```

## 1단계: 감각적 배경 설정

> **규칙:** 캐릭터의 행동이나 감정은 아직 쓰지 않습니다. 오직 환경만 묘사합니다.
> 독자가 이 장소에 '와 있는' 느낌을 받도록 오감을 활용합니다.

```
[산문 스타일리스트에게 - 1단계]

이 장면의 물리적 환경만 묘사해줘. 캐릭터 행동/감정은 아직 쓰지 마.

시각 (조명, 색감, 공간 크기, 시선이 가는 곳):
청각 (소리, 정적, 멀리서 들리는 것):
후각 / 촉각 (필요한 경우):
분위기 (공기의 질감, 온도, 긴장감의 물리적 현시):
```

## 2단계: POV 캐릭터 내면 목소리

> **규칙:** 위 배경 안에서 POV 캐릭터가 느끼는 것을 추가합니다.
> "그는 두려웠다" 같은 직접 서술은 금지. 신체 반응 + 내적 독백으로만 표현합니다.

```
[산문 스타일리스트에게 - 2단계]

1단계 배경 묘사에, POV 캐릭터의 심리 반응과 내적 독백을 추가해줘.
감정을 직접 말하지 말고, 신체 반응과 내적 독백으로만 표현해.

캐릭터의 현재 욕망 (이 장면에서 원하는 것):
캐릭터의 현재 두려움 (이 장면에서 잃을 수 있는 것):
이 장면에서의 심리 상태 키워드:
```

## 3단계: 대화 & 행동

> **규칙:** 사건을 촉발하거나 정보를 전달하는 대화와 행동을 추가합니다.
> "그가 말했다" 대화 태그는 금지. 제스처·행동·표정으로 대체합니다.

```
[산문 스타일리스트에게 - 3단계]

2단계 초안에, 이 장면의 핵심 대화와 행동을 추가해줘.
대화 태그("그가 말했다")는 제스처나 행동으로 대체해.

핵심 대화 목적: [갈등 유발 / 정보 전달 / 관계 변화 중 선택]
이 장면의 클리프행어 또는 전환점:
각 캐릭터 말투: (보이스 시트에서 가져오기)
```

## 4단계: 최종 산문 합성

> **규칙:** 1~3단계 요소를 하나의 리듬 있는 산문으로 통합합니다.
> 이 단계에서 '베이지색 표현' 제거, 속도감 조절, 구어체 삽입을 완성합니다.

```
[산문 스타일리스트에게 - 4단계]

지금까지의 3단계 내용을 하나의 완성된 장면으로 통합해줘.

- 긴장 구간: 문장을 짧게 끊어 속도감 높이기 (1~5단어 문장 혼용)
- 감정 구간: 감각 묘사를 풍부하게, 내적 독백 활용
- 베이지색 표현("갑자기", "그리고 나서", "~라고 느꼈다") 제거
- 구어체 및 신조어 적절히 삽입
- 목표 분량: 약 _자
```
