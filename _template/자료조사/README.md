# 자료조사 폴더 — [프로젝트명]

> **목적:** 이 소설의 현실감과 개연성을 높이는 **프로젝트 전용 참고 자료**를 보관합니다.
> **참조 방법:** 비트 시트 작성 시 이번 화와 관련된 파일 경로를 명시하고, 에이전트에게 붙여넣어 제공합니다.

---

## 이 폴더의 사용 원칙

```
[언제 자료를 추가하는가]
→ 특정 직업·전문 분야를 묘사해야 할 때
→ 실제 장소나 공간을 묘사해야 할 때
→ 특정 시대·문화·제도를 정확하게 담아야 할 때
→ 장르 특화 전문 지식이 필요할 때 (스포츠 규정, 법률, 의학 등)

[어떻게 에이전트에게 제공하는가]
→ beat_sheet.md의 '이번 화 참조 자료' 섹션에 파일 경로 기록
→ 산문 스타일리스트 호출 시 해당 파일 내용을 붙여넣기
→ 세계관 제작자 호출 시 관련 자료 파일을 함께 제공

[관리 원칙]
→ 파일명은 내용이 한눈에 보이도록 (예: 소방관_현장_절차.md)
→ 출처(링크·서적 등)를 파일 상단에 기록
→ 소설에 사용한 내용은 ✅로 표시
```

## 폴더 구조 예시

```

├── README.md            ← 이 파일 (인덱스)
├── 인물_직업/           ← 인물·직업·전문분야
│   └── 직업명.md
├── 장소_공간/           ← 실제 장소, 공간 묘사
│   └── 장소명.md
├── 시대_문화/           ← 시대적 배경, 문화·제도

    └── 주제.md
```
