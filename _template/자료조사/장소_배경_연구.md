# 장소·배경 연구 노트

> **용도:** 특정 장소나 공간을 현실감 있게 묘사하기 위한 참고 자료
> **출처:**

---

## 대상 장소

```
장소명:
소설 내 등장 목적:
관련 화수:
```

---

## 공간 개요

```
실제 위치 / 주소 (해당 시):
규모 / 면적:
운영 시간 / 계절 변화:
접근 방법:
```

---

## 감각 묘사 자료

> Show Don't Tell에 직접 활용할 수 있는 감각 정보

```
[시각]
→ 낮:
→ 밤:
→ 계절별:

[청각]
→ 평상시:
→ 특수 상황 시:

[후각]
→

[촉각 / 온도]
→

[공간만의 독특한 분위기]
→
```

---

## 장소에서의 인물 행동 패턴

```
[이 장소에서 사람들이 하는 행동]


[이 장소에서 금지되거나 어색한 행동]


[이 장소에 오래 있으면 나타나는 변화 (소음, 피로 등)]

```

---

## 소설 활용 포인트

```
[갈등 유발 요소]
→ 이 장소의 어떤 특성이 인물의 목표를 방해하는가?

[사이다/고구마 장치로 활용 가능한 요소]
→

[클리프행어 세팅으로 활용 가능한 요소]
→
```
