# 사령관 운영 프로토콜

> 사용자(사령관)는 직접 집필하지 않습니다. **명령·승인·검수**만 담당합니다.
> 모든 에이전트는 사령관의 지시 없이 결정 경계 항목을 독자적으로 실행할 수 없습니다.

---

## 사령관의 역할 정의

| 권한 | 내용 |
|------|------|
| 목표 설정 | 결말, 테마, 마일스톤 확정 |
| 에이전트 지휘 | 각 에이전트에게 과업 부여 및 결과물 승인 |
| 결정 경계 집행 | 아래 '반드시 승인 필요 사항' 실행 전 최종 결정 |
| 품질 검수 | 서사 감사관의 보고를 받아 수정 또는 통과 결정 |

---

## 표준 에이전트 워크플로우

```
[프로젝트 시작]
플롯 건축가 → 시놉시스 & 마일스톤 초안
                    ↓ [사령관 승인]
세계관 제작자 → 세계관 규칙 & 스토리 바이블 초안
                    ↓ [사령관 승인]
──────────────────────────────────────────
[화별 반복 사이클]
플롯 건축가 → 화별 비트 시트 작성
                    ↓ [사령관 승인]
산문 스타일리스트 → 4단계 레이어 기반 장면 집필
                    ↓
서사 감사관 → 품질 검수 & 설붕 체크 보고
                    ↓ [사령관 최종 확정]
세계관 제작자 → 스토리 바이블 업데이트
                    ↓
마이크로 요약 작성 → 다음 화 루프 시작
```

---

## 4단계 승인 절차

### 1단계 — 시놉시스 & 캐릭터 설정 승인
- **담당 에이전트:** 플롯 건축가 + 세계관 제작자
- **검토 포인트:** 결말과의 일치성, 마일스톤 인과 관계, 주인공 아크
- **승인 시:** story_bible.md 초안 확정

### 2단계 — 화별 비트 시트 승인
- **담당 에이전트:** 플롯 건축가
- **검토 포인트:** '하지만/그러므로' 인과 구조, 클리프행어 강도, 이번 화 목표 달성
- **승인 시:** 산문 스타일리스트에게 집필 지시

### 3단계 — 장면 초안 검수
- **담당 에이전트:** 산문 스타일리스트 → 서사 감사관 검수
- **검토 포인트:** Show/Don't Tell, 캐릭터 보이스 일치, 베이지색 산문 여부
- **승인 시:** 최종 원고 확정 또는 수정 지시

### 4단계 — 최종 원고 확정 & 바이블 업데이트
- **담당 에이전트:** 세계관 제작자
- **검토 포인트:** 마이크로 요약 작성 여부, 스토리 바이블 업데이트 항목
- **승인 시:** 해당 화 완료, 다음 화 루프 시작

## 결정 경계 (반드시 사령관 승인 필요)

아래 항목은 에이전트가 독자적으로 결정할 수 없습니다.

- [ ] 주요 캐릭터의 사망 또는 영구 퇴장
- [ ] 스토리 바이블 세계관 규칙 변경
- [ ] 고정 결말 방향과 충돌하는 전개
- [ ] 신규 주요 캐릭터 추가
- [ ] 기존 캐릭터의 핵심 성격 반전
- [ ] 10화 이상의 플롯 방향 전환
- [ ] 독자 피드백이 결말과 충돌할 때

## 킬 스위치 프로토콜

에이전트는 아래 상황 발생 시 즉시 집필을 중단하고 사령관에게 보고합니다.

```
⚠️ 킬 스위치 발동

상황: [발생한 상황 설명]
충돌 요소: [무엇과 충돌하는지 - 결말 / 세계관 규칙 / 캐릭터 설정]
제안 옵션:
  A) [옵션 1 - 현재 방향 유지하되 조정]
  B) [옵션 2 - 방향 전환]
사령관 결정 요청
```

## 사령관 검수 체크리스트

### 시놉시스 & 비트 승인 시
- [ ] 결말 방향과 일치하는가?
- [ ] 각 마일스톤에 '하지만/그러므로' 인과 관계가 있는가?
- [ ] 주인공의 내·외적 성장 아크가 명확한가?
- [ ] 독자가 매 화 이탈하지 않을 훅이 있는가?

### 장면 초안 검수 시
- [ ] 'Show, Don't Tell'이 적용되었는가?
- [ ] 캐릭터 말투가 보이스 시트와 일치하는가?
- [ ] 클리프행어가 충분히 강렬한가?
- [ ] 베이지색 산문('매끄럽지만 지루한 문장')이 없는가?

### 최종 원고 확정 시
- [ ] 서사 감사관의 '통과' 판정을 받았는가?
- [ ] 스토리 바이블 업데이트 필요 항목이 반영되었는가?
- [ ] 마이크로 요약(micro_summary.md)이 작성되었는가?


## 산문 스타일리스트 특별 지침 (유성의 궤적 전용)

```
[문체 원칙 — 엽기적인그녀 참조]
- 구어체 서술. 독자에게 말 거는 느낌으로 쓸 것.
- 황당한 상황을 과장 없이 담담하게 묘사할 것 (그게 더 웃김).
- 내면 독백: 짧고 건조하게. "아, 이거 진짜인 것 같은데." 수준.
- 감동 장면도 절대 과장하지 않음. 독자가 스스로 느끼게 둠.
- 템포: 빠름. 군더더기 없이 장면→장면.

[대화 원칙 — 숏박스 스타일 참조]
- 중학생 친구 사이 반말. "야", "아씨", "진짜", "미쳤냐" 자연스럽게.
- 짧은 대사 위주. 한 캐릭터가 3줄 이상 말하는 대사는 최대한 쪼갤 것.
- 진지한 순간에도 누군가 분위기를 깨는 한 마디 추가.
- 중학생 일상 (급식, 학원, 통금, 부모님)이 자연스럽게 녹아들어야 함.

[농구 묘사 원칙 — 규정 파일 참조]
- 대회 출전 절차(스포츠클럽 창설, 선수 등록, 최저학력제)를 현실감 있게 묘사.
- 농구 용어는 실제 경기에서 쓰는 것만. 과장된 필살기 이름 없음.
- 능력 묘사: 과학 설명 없이 감각 중심으로. "공중에서 멈춘 것 같았다" 수준.

[절대 금지]
- 등장인물을 성인처럼 말하게 할 것.
- 능력을 영웅적으로 과장 묘사할 것.
- 설명형 독백 ("그것은 바로 Court Vision이라는 능력으로서..." 금지).
```
