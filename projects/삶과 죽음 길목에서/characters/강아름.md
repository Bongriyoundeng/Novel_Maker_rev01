# 캐릭터 프로필 — 강아름 (EP3 핵심 조력자)

> 에피소드 3 "천 원짜리 물 한 병"의 핵심 조력자. **제3의 조력자 / 선택된 가족.**

---

## 인물 형상화 6가지 특징 체크

```
[v] 고정관념 활용: '고아원 출신 신입 사원' = 불쌍하고 눈치 보는 캐릭터 기대 전복.
    → 오히려 당당하고 자기 확신이 있음. '불쌍함'으로 대우받는 걸 거부함.

[v] 신체적 특징: 항상 지퍼백을 가방에 넣고 다님 (고아원에서 물건 정리하던 습관).
    급박한 상황에서도 손이 먼저 정리·준비를 시작함.

[v] 말투와 행동: 직접적이고 단호함. 보고서를 빨간 줄 없이 완벽하게 내는 유일한 사람.
    선희에게 처음으로 "틀렸어요"라고 말할 수 있는 사원.

[v] 상황 설정: 버스비 없을 정도로 경제적으로 빠듯하지만, 그게 당연한 일상이라 비참하지 않음.
    마지막 천 원으로 선희에게 물을 사다주는 장면이 서사의 핵심 전환점.

[v] 특별한 능력: 타인의 진심을 행동으로 먼저 반응하는 능력.
    말보다 행동이 먼저. 선희가 쓰러졌을 때 아무도 움직이지 않을 때 먼저 달려감.

[v] 태생적 특징: 이름 '아름' — 아름다움. 고아원에서 원장선생님이 지어준 이름.
    "네가 이 세상에서 가장 아름다운 걸 해주길 바란다"는 뜻.
```

## 외형

```
체형: 165cm, 보통 체형. 항상 단정하지만 비싸지 않은 옷.
외모 특징: 짧은 머리. 안경. 가방 안에 항상 지퍼백이 몇 개 있음.
주로 입는 옷/스타일: 체인점 정장 또는 깔끔한 캐주얼. 고가품 없음.
특징적인 외형 포인트: 지퍼백. 어디서나 뭔가를 지퍼백에 넣어 정리한다.
                       이 습관을 부끄럽게 생각하지 않는다.
```

## 성격 및 말투

### 핵심 성격 키워드

```
당당한 자립 / 행동 선행 / 직접성 / 감사 표현 / 두려움 없음
```

### 말투 특징

```
- 존댓말/반말: 선희 회장에게 존댓말이지만 '예쁜 존댓말'이 아님. 사실 그대로.
- 자주 쓰는 표현: "확인했습니다." / "그건 제가 잘못 알았습니다." / "해볼게요."
- 감정이 격해질 때: 더 직접적이 됨. 우회 없음.
- 절대 쓰지 않는 표현: "불쌍해서 도운 게 아니에요" — 이 말 자체를 하지 않음.
  행동으로만 증명함.
```

### 대화 예시 (AI 학습용)

```
평소:         "확인했습니다."
선희 회장에게: "그 부분은 회장님이 틀리셨습니다. 근거는 이겁니다."
당황했을 때:   "...잠깐만요." (지퍼백 꺼내 뭔가 정리 시작)
선희 쓰러질 때: (아무말 없이 먼저 달려감. 물 사옴. 그냥 내밈.)
극도로 위기일 때: "저 지금 버스비밖에 없는데요. 이거 쓸게요." (천 원짜리 꺼내며)
```

## 배경 스토리

```
과거:
  - 서울 서부권 A고아원 출신 (오선희가 20년간 익명 후원한 곳 - 72화까지 비밀)
  - 장학금으로 경영학과 졸업
  - 아르바이트를 쉬지 않고 하면서 스펙 쌓음 (불쌍함으로 성공한 게 아님 - 능력으로)
  - 오선희 그룹에 지원한 이유: "회장님이 요양 사업도 하신다고 해서요. 그 분야에 관심 있습니다."

현재 상황:
  - 신입 사원 첫 달. 버스비를 아껴서 생활.
  - 선희 회장이 차갑지만 능력 있다는 것을 이미 간파함.
  - 선희가 자신을 '불쌍한 고아'로 보지 않는다는 것에 오히려 안심.

숨겨진 비밀:
  - 오선희 그룹이 후원한 고아원이 자신이 자란 곳임을 나중에 알게 됨.
  - 고아원 원장이 선희에 대해 "익명의 후원자가 있었는데 당신에게 잘하라고 했다"고 말한 것을 기억함.

트라우마:
  - 고아원에서 '불쌍해서' 도움받는 것이 싫었음.
    특히 어른들이 자신을 볼 때 나오는 '연민의 눈빛'이 가장 싫었음.
  - 선희 회장의 눈에는 연민이 없다. 그게 처음으로 편안한 이유.
```

## 욕망과 두려움

```
표면적 욕망: 이 회사에서 실력으로 인정받는 것.
진짜 욕망:   선희 회장에게 "네가 내 딸 같다"는 말을 듣는 것이 아니라,
             "네가 일을 잘한다"는 말을 듣는 것. (딸로 대우받고 싶지 않음. 동료로 인정받고 싶음.)
             → 이것이 결말에서 선희와의 관계를 특별하게 만드는 포인트.
가장 큰 두려움: 자신이 선의를 베풀었을 때 그것이 '고아라서 그런 거지' 로 해석되는 것.
```

## 성장 아크 (12단계 서사 기준)

```
[변화 전]
1단계: 오선희 그룹 첫 출근. 실수로 선희 커피를 쏟음.
2단계: 선희가 "실력으로 증명해봐"라고 함. (소명 = 도전)

[변화 과정]
4~6단계: 빨간 줄 없는 보고서. 선희의 신뢰 쌓기 시작.
          선희가 자신의 고아원 출신을 알게 됨 → 아름은 "그래서요?"라고 함.
7~8단계: 선희 쓰러짐. 버스비 천 원으로 물 사옴.
  - 항상성 없음. 아름은 주저하지 않는다. 이것이 아름의 가장 큰 특징.

[변화 후]
9~11단계: 선희의 아들 재원이 등장. 아름은 재원의 '계산'을 느낌.
           선희와 재원 사이에서 아름은 선희에게 "그건 회장님이 결정하세요"라고 물러남.
12단계: 물병을 내밀고 선희의 "고맙다"를 듣는 순간.
        아름은 처음으로 우는 것을 참지 않는다.
```
