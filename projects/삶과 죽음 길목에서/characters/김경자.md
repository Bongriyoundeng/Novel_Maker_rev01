# 캐릭터 프로필 — 김경자 (EP2 조력자)

> 에피소드 2 "가죽 손가방"의 조력자. **마지막 증인 / 남편의 기록을 받아든 아내.**

---

## 인물 형상화 6가지 특징 체크

```
[v] 고정관념 활용: '노인 남편을 뒤에서 내조하는 아내' = 수동적 조력자 기대 전복.
    → 경자는 50년간 태석 옆에서 모든 것을 '보아온' 사람.
    → 말은 없지만 태석이 기록을 멈추지 않은 이유 중 하나가 경자 때문임.

[v] 신체적 특징: 항상 손수건을 들고 다님 (어린 시절부터의 습관).
    남편 병실에 올 때마다 손수건이 구겨져 있음 — 오는 길에 울었다는 신호.

[v] 말투와 행동: 말수가 매우 적음. 하지만 남편에게만 이름을 부름 "태석 씨."
    이 두 글자가 50년의 관계를 요약함.

[v] 상황 설정: 아들 덕수에게 연금 통장을 '관리 명목'으로 빼앗긴 상태.
    남편에게 병문안 오고 싶어도 덕수의 눈치를 봐야 함.
    → 면회를 못 오는 것이 '무관심'이 아님을 태석은 알고 있음.

[v] 특별한 능력: 남편의 말 없는 신호를 읽는 능력.
    50년을 함께 살며 태석의 침묵이 무엇을 뜻하는지 안다.

[v] 태생적 특징: 이름 '경자(慶子)' — 경사스러운 자녀. 부모가 기쁘게 맞이한 딸.
    태석의 장부 첫 페이지가 결혼 첫날이고, 마지막 페이지가 경자의 이름으로 끝나는 것의 의미.
```

## 외형

```
체형: 157cm, 작고 마른 체형. 나이에 비해 정정한 모습.
외모 특징: 흰 머리. 단정하지만 소박한 차림. 손수건은 항상 손에 쥐거나 주머니에.
주로 입는 옷/스타일: 수수한 한복 또는 생활 한복. 화려함 없음.
특징적인 외형 포인트: 구겨진 손수건.
                       병실 문을 열기 전 잠깐 멈추는 습관 — 마음을 가다듬는 시간.
```

## 성격 및 말투

### 핵심 성격 키워드

```
묵묵한 동반자 / 참아온 서러움 / 남편에 대한 신뢰 / 아들에 대한 복잡한 감정 / 증인
```

### 말투 특징

```
- 존댓말/반말: 남편에게는 "태석 씨" (이름 + 씨). 50년 내내 이 호칭.
  → 아들에게는 반말이지만, 이 아들에게는 아무 말 없음이 더 많음.
- 자주 쓰는 표현: "괜찮아요." (실제로는 괜찮지 않음) / "태석 씨." (이 두 글자로 모든 것을 전달)
- 감정이 격해질 때: 울지 않음. 더 조용해짐. 손수건을 더 세게 쥠.
- 절대 쓰지 않는 표현: 아들 덕수에 대한 직접적인 비난 — 입에 담지 않음. 단지 침묵.
```

### 대화 예시 (AI 학습용)

```
남편에게:      "태석 씨. 왔어요." (문을 열며. 두 마디.)
아들 앞에서:   (아무 말 없이 태석의 손을 잡음)
나현수에게:    "우리 영감, 밥은 먹어요?" (첫 마디가 항상 밥.)
장부 받을 때:  "이걸 왜 나한테 줘요?" (처음으로 목소리가 흔들림)
태석 눈 감을 때: "태석 씨." (이 두 글자만. 그것으로 충분.)
```

## 배경 스토리

```
과거:
  - 20대에 태석과 결혼. 가난하게 시작했지만 불평 없이 살아옴
  - 두 아들을 키우며 태석의 사업을 옆에서 지켜봄
  - 태석이 장부를 쓰기 시작할 때부터 그것의 의미를 알고 있었음
  - 차남 민수에게 생활비를 몰래 보내는 태석을 알고 있었음 (덕수에게 말하지 않음)

현재 상황:
  - 덕수가 연금 통장을 '관리 명목'으로 가져간 상태
  - 병문안 올 때마다 덕수의 일정과 겹치지 않도록 시간을 맞춤
  - 남편이 기록을 이어가고 있다는 것을 알고, 그것이 끝날 때를 기다리고 있음

숨겨진 내면:
  - 태석이 자신의 이름을 부를 때마다 50년의 모든 시간이 돌아옴
  - 장부 마지막 페이지에 자신의 이름이 있다는 것을 결말에 처음 알게 됨
  - 경자는 그 순간 처음으로 손수건을 꺼내지 않음 — 울지 않음. 이미 충분하기 때문.
```

## 양면성 & 입체성

```
외적 면모: 조용하고 순응적인 노인. 아들에게 끌려다니는 어머니처럼 보임.

내적 과거: 50년간 모든 것을 참고 지켜봤지만, 한 번도 포기하지 않은 사람.
           덕수의 행동이 틀렸다는 것을 알면서도, 직접 막을 힘이 없었음.
           그 무력함이 경자의 가장 큰 고통.

충돌 지점: 태석이 장부를 경자의 손에 쥐어주는 순간.
           "이제 당신이 증인이야" — 이 말이 50년의 기다림에 대한 응답임을 경자만 알 수 있음.
```

---

## 욕망과 두려움

```
표면적 욕망: 남편이 편하게 지내는 것.
진짜 욕망:   남편 곁에서 끝을 맞이하는 것. 혼자 남겨지지 않는 것.
가장 큰 두려움: 남편이 아들에게 상처를 받은 채로 눈을 감는 것.
               자신이 증인이 되기 전에 태석이 먼저 가는 것.
```

---

## 주인공과의 관계

```
초기 관계: 면회가 어려운 아내. 거리감 있어 보이지만 실제로는 가장 가까운 사람.
갈등 요인: 덕수의 연금 통제로 인한 면회 제약
관계 변화 방향: 장부가 법원에 제출되며 태석의 계획이 완성됨을 목격
결말에서의 최종 관계: 장부를 받아든 경자. "이제 당신이 증인이야."
                      창밖에 벚꽃이 진다. 두 사람 사이에는 50년치의 침묵이 흐름.
```
