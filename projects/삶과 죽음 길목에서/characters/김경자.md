# 캐릭터 프로필 — 김경자 (EP2 핵심)

> 에피소드 2 "모두가 원하는 죽음"의 핵심. **죽고 싶은데 죽지 못하는 아내.**

---

## 인물 형상화 6가지 특징 체크

```
[v] 고정관념 활용: '치매 환자' = 아무것도 모르는 상태라는 기대 전복.
    → 경자는 가끔 정신이 돌아온다(알츠하이머형 치매의 '좋은 날' — 실제 임상에서 발생).
    → 그 순간 자신이 짐이 되고 있다는 것을 선명하게 안다. 그 앎이 더 고통스럽다.
    → 임종 직전에는 '말기 명징(Terminal Lucidity)'이 나타날 수 있다 — 수년간 반응 없던
       환자가 임종 수 시간 전 갑자기 명징해지는 현상. 1~2% 사례 보고. 경자의 마지막 장면 근거.

[v] 신체적 특징: 치매가 진행될수록 눈에서 빛이 사라지는 날이 많아진다.
    하지만 '좋은 날'에는 처음 만났을 때의 눈빛이 돌아온다.
    → 이 눈빛이 돌아오는 날이 이태석이 가장 기다리는 것이자 가장 두려운 것.

[v] 말투와 행동: 치매가 진행된 상태에서는 단편적인 말만 한다.
    하지만 정신이 돌아오는 날에는 조용하고 명징하게 말한다.
    "나 이제 그만 가고 싶어요."

[v] 상황 설정: 암을 이겨냈는데 치매가 찾아왔다. 자신의 의지로 이길 수 없는 병.
    암은 싸울 수 있었지만, 치매는 자신이 사라지는 것을 지켜봐야 하는 병이다.

[v] 특별한 능력: 정신이 돌아오는 날의 명징함.
    태석이 고생하고 있다는 것, 덕수가 지쳐가고 있다는 것을 정확히 안다.
    그래서 "나 이제 그만 가고 싶어요"라는 말이 포기가 아니라 사랑에서 나온다.

[v] 태생적 특징: 이름 '경자(慶子)' — 경사스러운 자녀. 부모가 기쁘게 맞이한 딸.
    이태석에게도 경사스러운 존재였다. 그래서 이 사람이 떠난다는 것이 이 소설에서 가장 큰 상실.
```

## 외형

```
체형: 157cm, 작고 마른 체형. 치매로 인한 영양 불균형으로 더 줄었다.
외모 특징: 흰 머리. 좋은 날에는 단정하게 보이려 애쓴다. 나쁜 날에는 눈이 공허하다.
주로 입는 옷/스타일: 요양병원 환자복. 태석이 가져다준 좋아하는 색의 카디건을 걸칠 때도 있다.
특징적인 외형 포인트:
  - 좋은 날: 태석이 들어오면 눈이 먼저 반응한다. 입이 열리기 전에 눈이 먼저 웃는다.
  - 나쁜 날: 태석이 들어와도 모른다. 낯선 사람을 보듯 바라본다.
```

## 성격 및 말투

### 핵심 성격 키워드

```
(치매 상태) 단편적·반복적 / (좋은 날) 명징하고 조용함 / 태석에 대한 사랑 / 짐이 되는 것에 대한 두려움
```

### 말투 특징

```
- 치매 진행 상태:
  단편적인 말만 한다. "밥은?" / "추워." / "여기가 어디야."
  같은 말을 반복하기도 한다. "태석 씨 왔어?" (태석이 옆에 있어도.)

- 정신이 돌아오는 날 (좋은 날):
  조용하고 명징하다. 평소보다 말이 많아진다.
  태석의 손을 잡고 조용히 말한다. "나 이제 그만 가고 싶어요."
  태석의 눈이 빨개지는 것을 보고 더 말하지 않는다.

- 아들 덕수에게:
  좋은 날: "덕수야, 밥은 먹었냐." (어머니의 본능.)
  나쁜 날: 아들을 알아보지 못하기도 한다.

- 절대 하지 않는 말: "왜 나를 여기 보냈냐" — 원망하지 않는다.
```

### 대화 예시 (AI 학습용)

```
좋은 날, 태석이 들어올 때:   (눈이 먼저 웃는다.) "왔어요."
좋은 날, 태석 손 잡고:       "나 이제 그만 가고 싶어요." (조용하고 명징하게)
좋은 날, 덕수에게:           "덕수야, 밥은 먹었냐."
나쁜 날:                     "여기가 어디야." / "태석 씨 왔어?" (태석이 옆에 있어도)
마지막 명징한 순간:           "당신, 고생 많이 했어요. 나도 알았어요."
                              (잠깐의 침묵 후) "이제 보내줘도 돼요."
```

## 배경 스토리

```
과거:
  - 이태석과 결혼 후 소박하고 행복하게 살았다.
  - 아들 덕수 하나를 키우며 전업주부로 생활.
  - 태석이 회사에서 힘든 날에는 말없이 맛있는 밥상을 차렸다.
    태석이 술 마시고 늦게 들어오면 "왔어요?"라고만 했다.
  - 둘이 은퇴 후 부산 여행을 가기로 약속했었다.

발병 과정:
  - 60대 후반에 위암 발견. 수술 + 항암 치료.
    산정특례로 본인부담률은 낮아졌지만, 비급여 항목(영양제, 일부 약제)에 상당한 비용.
  - 암 완치 판정 후 1년: 기억 이상 증상. 알츠하이머형 치매 진단.
  - 치매 진행 속도가 비교적 빠른 편. 진단 3년 만에 장기요양 시설 입소 필요 수준.
  - 여명의원 304호 요양 병동 입소. 태석이 매일 면회 (오전 일찍 — 배우자 면회는 시간 무관).
  - 요양병원 월 비용: 본인부담금 100만 원 + 공동 간병비 70만 원 + 소모품 = 월 185~200만 원.
    비급여 약제 추가 시 230만 원 이상. 태석의 국민연금으로는 전액 감당 불가.
  - 연명의료: '좋은 날'에 사전연명의료의향서 작성 여부 고려 가능 설정.
    경자가 명징한 날 자신의 의사를 표명하는 것이 법적 효력을 가짐.

현재 상황:
  - 좋은 날과 나쁜 날이 교차한다.
  - 좋은 날: 태석과 덕수를 알아보고, 자신이 처한 상황을 명확히 인식한다.
             "내가 이러고 있으면 안 되는데"라는 자책이 이 날의 주된 감정.
  - 나쁜 날: 태석을 알아보지 못하거나, 혼란스러운 상태.

내면:
  - 좋은 날 경자가 느끼는 것: 죽고 싶다. 이게 더 계속되면 안 된다.
    남편이 자신을 간호하다 먼저 쓰러질 것 같다.
    아들이 자신 때문에 더 이상 행복하게 살지 못하는 것 같다.
  - 이 마음이 스스로를 포기한 것이 아니라 사랑에서 나온 것임을
    경자 자신은 알고 있지만, 태석과 덕수에게 말하면 상처가 될 것 같아서 말을 아낀다.
```

## 양면성 & 입체성

```
외적 면모: 치매 환자. 요양병원 침대에 누워있는 노인.
           때로는 남편을 알아보지 못하는, 도움이 필요한 상태.

내적 과거: 50년을 태석 옆에서 살면서 말보다 행동으로 사랑을 표현해온 사람.
           좋은 날의 명징함이 치매의 잔인함을 더욱 극명하게 보여준다.
           아는데 잊어가는 것이 모르는 것보다 더 무섭다.

충돌 지점:
  마지막 명징한 순간. "당신, 고생 많이 했어요. 나도 알았어요."
  이 말이 경자가 태석에게 50년 동안 하지 못한 모든 말의 압축이다.
  그리고 "이제 보내줘도 돼요."
  이 말을 들은 이태석이 처음으로 "응, 가도 돼"라고 말한다.
  → 이 순간이 EP2의 클라이맥스.

임종 과정 묘사 근거 (현실 반영):
  - 말기 명징(Terminal Lucidity): 임종 수 시간~수 일 전, 수년간 반응 없던 환자가
    갑자기 명징해지는 현상. 경자의 마지막 대사 장면이 이것에 해당.
    이 장면 직후 경자 상태 급격히 악화 → 임종으로 이어지는 서사 구조.
  - 청각 마지막 유지: 임종 직전 의식이 없어도 청각은 마지막까지 남음 (임상 관찰).
    태석이 손을 잡고 "내가 여기 있어"라고 말하는 것이 의미 있는 이유.
  - 임종의 신체적 모습: 조용히 호흡이 느려지다 멈춤. 극적인 소리 없음.
    이태석이 한참 동안 손을 놓지 않는 장면.
```

## 욕망과 두려움

```
표면적 욕망: 태석과 덕수가 편안하게 사는 것.
진짜 욕망:   자신이 없는 자리에서 두 사람이 실제로 편안해지는 것.
             그것을 확인할 수 없다는 것이 마지막 두려움.
가장 큰 두려움: 자신이 기억을 완전히 잃기 전에 태석에게 하고 싶은 말을 못 하는 것.
               나쁜 날이 좋은 날보다 많아지는 것.
```

## 주인공(이태석)과의 관계

```
초기 관계: 치매가 진행 중이지만, 좋은 날엔 50년 부부의 교감이 살아있다.
갈등 요인: 경자의 "나 이제 그만 가고 싶어요" — 태석이 이 말을 받아들이지 못하는 것.
관계 변화 방향: 경자의 좋은 날이 줄어들수록 태석의 말 못할 생각이 커짐.
결말에서의 관계:
  경자의 마지막 명징한 순간 — "이제 보내줘도 돼요."
  이태석의 처음이자 마지막 허락 — "응, 가도 돼."
  경자가 눈을 감는다.
  이것이 두 사람의 마지막 대화.
```
