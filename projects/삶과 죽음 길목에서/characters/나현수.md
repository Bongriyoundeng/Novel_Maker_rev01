# 캐릭터 프로필 — 나현수 (연결 캐릭터)

> 세 에피소드를 연결하는 **프레임 서술자 / 장례지도사.**
> 세 이야기 모두에 조력자 또는 관찰자로 등장.

---

## 인물 형상화 6가지 특징 체크

```
[v] 고정관념 활용: '장례지도사' = 죽음을 다루는 음울한 직업이라는 편견 전복.
    → 오히려 가장 냉정하게 삶을 바라볼 수 있는 사람. 죽음의 전문가가 곧 삶의 관찰자.

[v] 신체적 특징: 항상 검은 정장이지만 주머니에 사탕을 넣고 다님.
    노인들이 약 먹을 때 쓴맛을 달래주기 위해 (자신의 어머니에게 하던 습관).

[v] 말투와 행동: 조용하고 중립적. 어떤 상황에서도 판단하는 말을 하지 않음.
    하지만 필요할 때 정확한 한 마디를 함. "지금 어떠세요?"

[v] 상황 설정: 자신의 어머니가 여명의원 인근 병원에 입원 중.
    직업상 매일 죽음을 다루면서, 정작 자기 어머니의 임종은 마주하지 못할까봐 두려움.

[v] 특별한 능력: 세 가족의 공통점을 먼저 감지하는 관찰력.
    직업 특성상 사람의 죽음 전후를 많이 봐와서 패턴을 읽음.

[v] 태생적 특징: 이름 '현수(賢秀)' — 현명하고 뛰어난 사람. 직업적 현명함과 일치.
```

## 외형

```
체형: 163cm, 보통 체형. 언제나 단정하고 절제된 외모.
외모 특징: 검은 정장 + 흰 와이셔츠. 화장은 최소화.
주로 입는 옷/스타일: 직업 특성상 항상 검은 정장. 하지만 주머니 속 사탕이 유일한 컬러.
특징적인 외형 포인트: 왼쪽 정장 주머니에 항상 사탕 몇 개.
                       대화 중 상대가 불안해 보이면 말없이 사탕을 꺼내 테이블에 올려놓음.
```

## 성격 및 말투

### 핵심 성격 키워드

```
중립적 관찰자 / 깊은 공감 / 절제된 감정 / 연결자 / 내면의 두려움
```

### 말투 특징

```
- 존댓말/반말: 모든 상대에게 존댓말. 직함 없이 "어르신" / 이름+씨 호칭.
- 자주 쓰는 표현: "지금 어떠세요?" / "괜찮으세요?" / (사탕을 내밀며) "드세요."
- 감정이 격해질 때: 말이 아닌 행동. 상대의 손을 잡음.
- 절대 쓰지 않는 표현: "잘 됐다" / "다행이다" — 상실 앞에서 위로처럼 들리는 말을 하지 않음.
```

## 배경 스토리

```
과거:
  - 20년 전 아버지의 임종을 홀로 지키며 장례지도사 직업을 선택
  - 여명의원에서 일한 지 12년
  - 비밀: 어린 시절 서울 서부권 어느 집에서 세 들어 살 때 집주인 이태석에게
    은혜를 입은 기억이 있음 (아버지 병원비 문제). 80화까지 비밀.

현재 상황:
  - 어머니가 여명의원 인근 병원에 입원 중 (치매 초기)
  - 매일 다른 사람의 임종을 도우면서 자신의 어머니를 어떻게 보내야 할지 모름
  - 세 가족(박기태·이태석·오선희)의 사정을 서서히 파악해가며 조력자 역할

숨겨진 비밀:
  - 80화에서 밝혀짐: 이태석의 집에 세 들어 살던 나현수 어머니가 병원비 위기일 때
    이태석이 모르는 척 집세를 안 받았음. 그것이 현수에게 유일한 은인.
  - 이 비밀이 밝혀질 때 나현수가 이태석을 위해 결정적 도움을 주는 행동의 이유가 됨.
```

## 욕망과 두려움

```
표면적 욕망: 세 가족이 후회 없는 작별을 할 수 있도록 돕는 것.
진짜 욕망:   자신의 어머니와의 작별을 제대로 할 수 있는 용기를 얻는 것.
가장 큰 두려움: 직업적으로는 수백 번의 임종을 봤지만,
               어머니의 임종 앞에서 자신이 장례지도사가 아닌 딸이 될 수 있을지.
```
