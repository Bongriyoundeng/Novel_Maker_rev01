# 캐릭터 프로필 — 박기태 (EP1 주인공)

> 에피소드 1 "900억보다 무거운 것"의 주인공. **시험하는 군주.**

---

## 인물 형상화 6가지 특징 체크

```
[v] 고정관념 활용: 권위적 대기업 총수의 전형 — 하지만 실체는 자식의 사랑을 갈구하는 나약한 노인.
    → 독자가 처음엔 차갑고 계산적인 인물로 오해했다가 내면이 드러나며 역전 공감.

[v] 신체적 특징: 굽은 허리와 거친 손마디 — 30년 전 막노동판에서 시멘트를 나른 흔적.
    아무리 정장을 입어도 손을 숨길 수 없다.

[v] 말투와 행동: 말수가 극도로 적음. 꼭 필요한 말만 짧게. 하지만 수아 앞에선 말이 끊어짐.
    → 혼자일 때는 생일 케이크에 대고 "잘 있었냐"고 중얼거린다.

[v] 상황 설정: 6개월 시한부 진단(독자는 45화까지 모름). 진짜 유언 대신 '살아있는 시험'을 선택.

[v] 특별한 능력: 사람 읽는 능력 — 평생 사업에서 단련된 눈으로 누가 진심인지 즉시 감지.
    → 하지만 수아 앞에서는 이 능력이 작동하지 않는다. 딸 앞에선 맹인이 된다.

[v] 태생적 특징: 이름 '기태(基泰)' — 터를 닦아 태평을 이룬다는 뜻.
    평생 남을 위한 기반을 쌓았지만 정작 자기 가정의 터는 혼자 흔들리고 있었다.
```

## 성격 및 말투

### 핵심 성격 키워드

```
전략적 냉혹함 / 억눌린 부성애 / 자존심 / 고독 / 기다림
```

### 말투 특징

```
- 존댓말/반말 여부: 모두에게 반말. 단, 수아 앞에선 말이 나오지 않아 침묵이 반말 역할.
- 자주 쓰는 표현: "됐다." / "그래서?" / "알아서 해." — 모든 감정을 3음절 이하로 압축.
- 감정이 격해질 때: 반대로 말이 더 줄어듦. 극도로 조용해진다.
  → 완전히 감정이 터질 때만 존댓말: "내가... 잘못했습니다." (단 한 번, 수아 앞에서)
- 절대 쓰지 않는 표현: "사랑한다" — 평생 한 번도 자녀에게 말한 적 없음.
```

### 대화 예시 (AI 학습용)

```
평소:       "됐어. 나가."
화났을 때:  (말 없음. 찻잔을 내려놓는 소리만.)
당황했을 때: "...그래서, 뭐가 필요한 거야."
수아에게:   "...밥은 먹었냐." (3초 침묵 후)
시한부 알고 있을 때: "괜찮다. 오래 살 것도 없어."
```

## 배경 스토리

```
과거:
  - 10살에 아버지를 잃고 막노동으로 가족을 먹여 살림
  - 30대에 기태건설 창업, 50대에 그룹으로 성장
  - 자식들에게 '성공'만 가르쳤고, '감정'은 사치라 생각했음
  - 10년 전, 막내딸 수아가 '보잘것없는 남자'와 결혼하겠다 했을 때 의절 선언

현재 상황:
  - 여명의원 302호 입소 (가짜 파산 수행 중)
  - 실제로는 심장 질환 + 암 초기 진단. 6개월 시한부 (독자에게 45화까지 비밀)
  - 장남 민혁이 재산 은닉 중이라는 것을 알면서도 지켜보는 중

숨겨진 비밀:
  - 시한부 선고를 받은 후, 처음으로 수아에게 연락하기로 결심한 것이 파산 연극의 진짜 동기
  - 지갑 속 가족사진: 수아 얼굴 부분만 가위로 오려낸 자리가 있다. 10년 전 의절 직후 오렸는데,
    사진 뒷면에 수아의 출생 당시 체중이 볼펜으로 적혀 있다. 버리지 못한 이유.

트라우마:
  - 어머니를 일찍 여의고 "다시는 내 사람을 잃지 않겠다"고 다짐했지만,
    결국 자기 방식으로 자식들을 밀어낸 아이러니
```

## 양면성 & 입체성

```
외적 면모 (겉으로 보이는 모습):
  냉혹한 전략가. 900억 자산가. 침묵으로 명령하는 가부장.

내적 과거 (숨겨진 상처·결핍):
  막노동으로 가족을 먹인 10살짜리 소년이 어른이 된 것뿐.
  '약한 모습을 보이면 무너진다'는 생존 철학이 감정 표현을 차단함.

충돌 지점:
  수아의 반지하 방에서 죽은 사위의 영정 사진을 발견하는 순간.
  "내 딸이 과부가 된 것도 몰랐다" — 가부장적 철학이 처음으로 완전히 무너짐.
  이 순간 이후, 그는 '시험관'이 아니라 '아버지'가 되기 시작한다.
```

### 히어로-빌런 양면성

```
공유하는 트라우마 (장남 민혁과):
  둘 다 "약해 보이면 진다"고 배움. 기태는 빈곤에서, 민혁은 기태의 양육 방식에서.

운명을 가른 선택:
  기태 → 가난을 극복하되 자식에게 물질로 보상했다 (감정은 차단)
  민혁 → 물질만 배웠고 감정은 아예 없는 인간이 됨

빌런의 논리 (민혁 기준):
  "아버지가 나를 그렇게 키웠다. 사랑한다는 말 한 번 없이 돈만 줬으면서
   이제 와서 진심을 원한다고? 그건 아버지 잘못이다."
  → 독자가 민혁에게 50%는 이해하게 만드는 포인트.
```

## 욕망과 두려움

```
표면적 욕망: 자식들이 자기를 진심으로 대하는지 확인하고 싶다.
진짜 욕망:   죽기 전에 딱 한 번, 수아에게 "미안하다"고 말하고 싶다.
가장 큰 두려움: 수아가 자신을 진심으로 용서해주었는데 자신이 먼저 죽어 그것을 오래 누리지 못하는 것.
               (= 이미 시한부임을 알고 있기 때문에 더욱 절박함)
```

## 성장 아크 (12단계 서사 기준)

```
[변화 전]
1단계 (일상 세계): 900억 자산가. 생일날 혼자 케이크 앞에 앉음. "내가 뭘 잘못했나."
2단계 (소명):      시한부 선고. "죽기 전에 한 번만 진짜 아버지가 되고 싶다."
3단계 (조건 부재): 자식들에게 사과하면 '약해 보인다'는 두려움. 직접 접근 불가.

[변화 과정]
4~6단계 (전개): 가짜 파산 실행. 민혁에게 문전박대. 수아의 반지하 발견.
7~8단계 (위기·시련): 수아가 전 재산을 봉투에 넣어줌. 세계관 붕괴.
  - 이 시점 변화 저항(항상성): "내가 이 돈을 받으면 질 것 같다. 아직 아버지로서 자존심이..."
    → 하지만 봉투를 돌려줄 수가 없다. 수아가 맨발로 쫓아나왔기 때문에.

[변화 후]
9~10단계 (보상·정면승부): 가짜 파산을 공개. 민혁을 법정에 세움. 수아 앞에 무릎.
11단계 (각성):    "내 평생 벌어온 900억이 이 봉투보다 가벼웠다."
12단계 (귀환):    수아의 손을 잡고 여명의원 로비를 걸어나가며 "이제 진짜 집으로 가자."
                   (시한부지만 남은 날을 딸과 함께 살기로 결심)
```

## 보이스 시트 (Voice Sheet)

```
문장 리듬: 극도로 짧고 끊김. 접속사 거의 없음. 침묵이 대사 역할.
자주 쓰는 표현: "됐다." / "그래서?" / "알아서 해." / (말 없이 찻잔을 내려놓음)
절대 쓰지 않는 표현: "사랑한다" / "미안하다" (45화 이전까지는 절대 불가)
격식체 / 반말: 모든 상대에게 반말. 극적 순간에만 존댓말 전환 (효과 극대화)
감정이 격해질 때: 더 조용해짐. 말이 줄어들수록 감정이 큰 것.
```

```
평소:           "됐어. 나가."
화났을 때:       (찻잔 내려놓는 소리. 3초 침묵.) "...다시 한 번 말해봐."

```
