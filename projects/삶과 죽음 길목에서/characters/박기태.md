# 캐릭터 프로필 — 박기태 (EP1 주인공)

> 에피소드 1 "1000억보다 무거운 것"의 주인공. **뿌리를 찾는 군주.**

---

## 인물 형상화 6가지 특징 체크

```
[v] 고정관념 활용: 성공한 중소기업 회장이라는 외피 — 하지만 실체는 6.25 전쟁 고아 출신으로
    평생 홀로 살아남아야 했던 노인. 아무도 그의 출신을 모른다.
    → 독자가 처음엔 차갑고 고집스러운 부자 노인으로 보다가, 배경이 드러나며 연민으로 역전.

[v] 신체적 특징: 오른손 네 번째 손가락 마디가 굳은살로 두꺼워져 있다.
    고아원 시절과 젊은 시절 막노동의 흔적. 1000억 회장이 된 후에도 사라지지 않음.

[v] 말투와 행동: 말수가 적음. 꼭 필요한 말만 짧게. 생일날 혼자 설렁탕을 앞에 두고
    한 시간 동안 아무것도 하지 않는다 — 전화를 기다리면서.

[v] 상황 설정: 위암 말기, 6개월 시한부 진단(독자는 45화까지 모름). 증상은 체중 감소,
    식욕 부진, 간헐적 상복부 통증. 가짜 파산을 기획하지만 — 확인한 것은 진심이 없다는 사실.
    시한부임을 알기 때문에 "이제 두려울 게 없다" — 가짜 파산도, 절연도, 기부도 가능한 것.

[v] 특별한 능력: 사람을 읽는 눈 — 평생 사업에서 단련된 능력.
    하지만 자식에 대해서는 이 능력을 쓰지 않으려 했다. 믿고 싶었으므로.

[v] 태생적 특징: 이름 '기태(基泰)' — 터를 닦아 태평을 이룬다는 뜻.
    고아원 원장이 지어준 이름. 그래서 이 이름이 기부 협약서에 적힐 때의 무게가 다르다.
```

## 외형

```
체형: 키 173cm, 마른 체형. 한때 다부졌던 몸이 최근 몇 달 사이 눈에 띄게 줄었다.
     위암 말기로 인한 체중 감소 — 하지만 아무에게도 말하지 않음.
외모 특징: 백발. 이마의 주름이 깊다 — 걱정이 아니라 오랜 결단의 흔적처럼 보인다.
           식사량이 크게 줄었다. 설렁탕 국물도 절반을 남긴다.
주로 입는 옷/스타일: 평소 단정한 양복 (습관). 가짜 파산 이후엔 낡은 점퍼와 면 바지.
특징적인 외형 포인트:
  - 오른손 네 번째 손가락 마디의 굳은살 (고아원·막노동 시절 흔적)
  - 지갑이 매우 얇다. 카드 한 장과 낡은 편지 봉투 하나. 현금은 없다.
  - 가끔 상복부를 손으로 짚는 버릇 — 통증을 억누르는 것. 아무도 눈치채지 못함.
  - 매달 홀로 병원을 다닌다. 약을 받아오지만 처방전을 버린다.
```

## 성격 및 말투

### 핵심 성격 키워드

```
오랜 고독 / 자존심 / 믿고 싶은 마음 / 기다림 / 결단하면 흔들리지 않음
```

### 말투 특징

```
- 존댓말/반말: 모두에게 반말. 말 자체가 적음.
- 자주 쓰는 표현: "됐다." / "그래서?" / (말 없이 창밖을 본다.)
- 감정이 격해질 때: 반대로 더 조용해짐. 목소리가 낮아질수록 더 심각한 상태.
- 절대 쓰지 않는 표현: "사랑한다" — 평생 한 번도 자녀에게 말한 적 없음.
  그 대신 생일 때 통장에 돈을 넣어뒀다. 그게 그의 방식이었다.
- 생일날 혼자일 때: 설렁탕 국물을 한 숟가락 떠다가 내려놓고 창밖을 본다.
  그게 전부다. 무너지지 않는다. 그냥 조용히 결심한다.
```

### 대화 예시 (AI 학습용)

```
평소:               "됐어. 나가."
자식이 지원 요청할 때: (말 없이 수표를 꺼내 테이블 위에 놓는다.)
생일날 혼자:         (설렁탕 앞에 앉아 창밖을 본다. 핸드폰을 만지작거린다. 내려놓는다.)
파산 후 자식들의 괄시: (아무 말 없이 자리를 뜬다. 문 밖에서 한 번도 돌아보지 않는다.)
기부 서류 서명 시:   (볼펜을 천천히 내려놓으며 창밖을 본다. 처음으로 작게 웃는다.)
```

## 배경 스토리

```
출생과 유년:
  - 1955년생 (소설 현재: 70세)
  - 6.25 전쟁 이후 부모를 잃음 (당시 수십만 명의 전쟁 고아 중 한 명)
  - 서울 서부 지역 고아원에 수용. 시설 환경이 열악했고 구호물자 횡령이 일상.
  - 고아원 원장 한 명이 기태를 각별히 여겨 이름을 지어줬다 — "기태야, 잘 살거라."
  - 고아원 퇴소 후 10대 시절: 신문팔이, 구두닦이로 생계. 사회적 냉대와 편견 속에 성장.
  - 연고 없다는 이유로 취업 차별. 결혼 상대로 기피됨.

성장과 자수성가:
  - 20대: 건설 현장 막노동으로 종잣돈 마련.
  - 30대: 소규모 건자재 도매업 창업 ('기태산업').
  - 40~50대: 중소기업으로 성장. 지방 건설사 협력업체 네트워크 구축.
  - 와이프를 일찍 잃고 큰아들 민혁, 막내딸 수아를 혼자 키움.
  - 자식 교육에 모든 것을 쏟았지만 감정 표현은 서툴렀음.
    "사랑한다"는 말 대신 돈을 줬고, "미안하다"는 말 대신 더 많이 줬다.

현재 상황:
  - 70세 생일 전 주: 자식들이 찾아와 알랑방구를 피우며 지원을 요청.
  - 70세 생일 당일: 전화 한 통 없음. 혼자 설렁탕을 앞에 두고 앉아 결심.
  - 가짜 파산 공표 이후: 자식들의 태도가 180도 변함. 괄시와 무시.
  - 고아원 재단 이사장과 접촉 중. 전 재산 환원 계획.

숨겨진 비밀:
  - 위암 말기, 6개월 시한부 진단 (독자에게 45화까지 비밀)
    → 위암은 노인 남성에서 흔하며, 말기 증상: 체중 감소, 식욕 저하, 상복부 통증, 피로감.
    → 기태는 항암 치료를 거부했다. "남은 시간 동안 해야 할 일이 있다."
  - 지갑 속 낡은 편지: 고아원 원장이 퇴소 날 써준 편지 "기태야, 잘 살거라."
    이 편지가 기부 협약서에 서명할 때 다시 꺼내진다 (85화).
  - 유언 공증: 기부 결심 후 변호사를 통해 공증된 유언장 작성 완료.
    자식들이 유류분 청구 소송을 제기할 수 있다는 것을 알면서도 진행.
    "싸우고 싶으면 싸워라." — 말은 하지 않지만 이 각오가 있음.

트라우마:
  - "연고 없는 사람은 믿어주지 않는다"는 편견 속에서 자랐음.
    그래서 자식에게 돈을 줬다 — 연고 대신 돈으로 관계를 만들려 했던 것.
    그 방식이 결국 자식들에게 돈만 보는 눈을 심어줬다는 아이러니.

시한부 선고 이후 심리 변화 (자료조사 근거):
  - "이제 두려울 게 없어." — 죽음을 앞두면 사회적 두려움이 사라진다 (임상 관찰).
  - 삶의 재평가: "진짜 중요한 게 뭔지 이제 알았어."
  - 자식들에 대한 냉철한 평가 가능: 평소에는 모른 척했던 자식들의 행동이 선명하게 보임.
  - 가짜 파산은 시한부 이후의 관계 검증 욕구 — "죽기 전에 진짜 나를 사랑하는 사람이 있나."
  - 파산 후 자식들의 반응을 보고: "없구나." → 충격보다 체념. 이미 짐작하고 있었으므로.
  - 재산 처분 결정: 시한부 환자가 의미 있는 곳에 기부하려는 욕구 — 고아원 환원은 삶의 완결감.
```

## 양면성 & 입체성

```
외적 면모:
  냉혹한 중소기업 회장. 돈 주는 것 외엔 아무것도 없는 아버지.
  가짜 파산 이후엔 자식들에게 '짐이 된 늙은이'.

내적 과거:
  고아원에서 혼자 살아남아야 했던 소년이 어른이 된 것.
  '약하면 죽는다'는 생존 철학이 감정 표현을 차단함.
  그러나 이 노인은 70년 동안 딱 한 가지를 원했다 — 가족.
  아이러니하게도 가족을 원했기 때문에 가족을 망쳤다.

충돌 지점:
  친척 모임에서 아들 민혁이 "이제 아버지가 우리한테 짐이 된다"고 말하는 순간.
  기태는 아무 말 없이 자리를 뜬다.
  문 밖으로 나와 지갑을 꺼내 낡은 편지를 꺼내 본다. "기태야, 잘 살거라."
  그리고 고아원 재단에 전화를 건다.
```

### 히어로-빌런 양면성 (자식들과의 관계)

```
공유하는 트라우마 (자식들과):
  기태는 자식들에게 돈만 줬다. 감정 표현이 없었다.
  자식들은 아버지에게서 돈 외의 것을 배운 적이 없다.
  → 둘 다 '감정 없이 돈으로 관계 맺는 방식'을 반복하고 있다.

운명을 가른 선택:
  기태 → 가난과 외로움 속에서도 '기태산업'을 일으킴 (결핍에서 자수성가)
  민혁·수아 → 풍요 속에서 오히려 '더 받고 싶은' 욕망만 자람

빌런의 논리 (자식들 기준):
  "아버지가 우리한테 사랑한다는 말 한 번 없이 돈만 줬잖아요.
   이제 와서 진심을 원한다고요? 그건 아버지 잘못이에요."
  → 독자가 이 말에 일부 고개를 끄덕이게 만드는 포인트.
  → 하지만 생일날 전화 한 통조차 없다는 사실로 설득력이 사라짐.
```

## 욕망과 두려움

```
표면적 욕망: 자식들이 자신을 진심으로 대하는지 확인하고 싶다.
진짜 욕망:   죽기 전에 딱 한 번, 자신이 나온 고아원 원장의 편지에 답하고 싶다.
             "잘 살았습니다."라고.
가장 큰 두려움: 자신이 평생 쌓아온 것이 결국 아무것도 아니었다는 것.
               (이미 확인 중. 파산 후 자식들의 반응이 그것을 증명하는 중.)
```

## 성장 아크 (12단계 서사 기준)

```
[변화 전]
1단계 (일상 세계): 70세 생일 직전. 자식들이 지원을 요청하러 찾아온다. 기태는 수표를 건넨다.
2단계 (소명):      생일 당일, 전화 한 통 없음. 혼자 설렁탕. 결심.
3단계 (조건 부재): 자식들에게 직접 "진심으로 대해달라"고 말하는 것은 불가. 자존심.

[변화 과정]
4~6단계 (전개): 가짜 파산 실행. 자식들의 태도가 돌변.
7~8단계 (위기): 자식들의 괄시가 극에 달함. "짐이 된다"는 말을 듣는 장면.
  - 변화 저항(항상성): "설마 내 자식들이 이럴 리 없어. 조금만 더 기다려보자."
  - 하지만 결국 고아원 원장의 편지를 꺼내 보는 순간 포기가 아닌 결단을 선택.

[변화 후]
9~10단계 (보상·정면승부): 고아원 재단에 전화. 기부 결심. 자식들과 마지막 대면.
11단계 (각성):    "내가 어디서 왔는지 내 새끼들은 몰랐지. 나는 안다."
12단계 (귀환):    기부 협약서에 서명. 처음으로 작게 웃는다.
                   자식 없는 노인의 얼굴에서 — 처음으로 고독이 빠져나간 자리가 보인다.
```

  평소: "됐어. 나가."
