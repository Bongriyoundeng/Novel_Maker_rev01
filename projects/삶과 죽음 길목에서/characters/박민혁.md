# 캐릭터 프로필 — 박민혁 (EP1 갈등 유발자)

> 에피소드 1 "900억보다 무거운 것"의 갈등 유발자. **마키아벨리적 후계자 / 아버지를 수단으로 본 장남.**

---

## 인물 형상화 6가지 특징 체크

```
[v] 고정관념 활용: '성공한 장남' = 가문의 자랑이라는 기대 전복.
    → 실제로는 아버지의 이름으로 쌓은 성공. 아버지가 없으면 자신도 없다는 것을 알기에 더 집착함.

[v] 신체적 특징: 항상 비싼 시계를 차고 다님 (아버지에게 받은 첫 번째 선물 — 아이러니).
    시계는 그가 아버지의 유산임을 증명하는 동시에, 그가 아버지를 얼마나 도구화했는지를 상징함.

[v] 말투와 행동: 표면적으로 차분하고 논리적. 숫자와 근거로 말함.
    하지만 감정이 격해질 때 아버지에게 반말로 전환. "아버지가 뭘 안다고."

[v] 상황 설정: 아버지의 가짜 파산 소식에 가장 먼저 재산 은닉에 착수.
    동생 수아를 오래전부터 홀대해왔으며, 아버지의 사랑을 독점하려 했음.

[v] 특별한 능력: 법적 허점과 재산 이전 구조를 꿰뚫는 능력.
    변호사·회계사를 활용해 자신의 행동 증거를 지우는 데 능숙함.

[v] 태생적 특징: 이름 '민혁(敏赫)' — 날카롭고 빛나는 사람. 이름처럼 날카롭지만, 그 날카로움이 가족을 향함.
```

## 성격 및 말투

### 핵심 성격 키워드

```
냉철한 계산 / 표면적 효성 / 독점 욕망 / 아버지에 대한 의존과 혐오 / 동생 배제
```

### 말투 특징

```
- 존댓말/반말: 아버지에게 평소 존댓말. 격해지면 반말로 전환.
  → 이 전환이 민혁의 '민낯'이 드러나는 신호.
- 자주 쓰는 표현: "근거가 있어야 합니다." / "현실적으로 생각하면." / "제가 알아서 할게요."
- 감정이 격해질 때: 말수는 줄어들고 더 논리적인 척하지만 목소리가 낮아짐.
- 절대 쓰지 않는 표현: "수아는..." — 동생 이름을 가능한 한 입에 올리지 않음.
```

### 대화 예시 (AI 학습용)

```
평소:          "아버지, 회사 입장을 먼저 정리해야 합니다."
아버지에게:    "감정적으로 결정하시면 안 됩니다." (차갑고 논리적)
수아 언급 시:  (0.5초 멈춤 후) "그 얘기는 지금 중요하지 않습니다."
격해질 때:     "아버지가 뭘 안다고. 내가 다 만들었잖아."
들통날 때:     "...이게 왜." (처음으로 말문이 막힘)
```

## 배경 스토리

```
과거:
  - 어린 시절부터 아버지의 첫째로서 특별 대우를 받음
  - 아버지의 사업을 물려받기 위해 인생 전체를 투자함
  - 막내 수아가 태어난 뒤 아버지의 관심이 분산되자 내심 분노
  - 수아를 의도적으로 소외시킨 것은 어릴 때부터 시작된 패턴
  - 아버지 회사를 실질적으로 운영하게 된 30대부터 재산 구조를 파악·관리

현재 상황:
  - 아버지의 가짜 파산 소식을 진짜로 믿고 재산 은닉에 착수
  - 법인 우회 이전, 부동산 명의 변경 등 구체적 행동 중
  - 아버지가 여명의원에 입소한 것을 '기회'로 판단

숨겨진 비밀:
  - 아버지의 시계를 받은 날이 사실 수아의 생일이었음 (아버지가 수아 대신 민혁에게 선물)
  - 이 사실을 기태가 45화에서 밝힘 — 민혁이 그 날을 아직 자랑스럽게 여긴다는 것이 반전
```

## 양면성 & 입체성

```
외적 면모: 성공한 장남. 아버지를 '모시는' 척하는 능숙한 외형.

내적 과거: 아버지의 인정이 자신의 존재 근거였음.
           평생 "아들이라면 이래야 해"를 주입받았고, 그것이 지금의 자신을 만들었음.
           수아를 배제한 건 생존 본능 — 아버지의 사랑이 분산되면 자신이 사라진다고 믿었음.

충돌 지점: 아버지가 가짜 파산임을 알고, 자신이 '시험당했다'는 것을 깨달았을 때.
           처음으로 자신의 행동이 '자식으로서의 선택'이었음을 직시해야 하는 순간.
```

### 히어로-빌런 양면성

```
공유하는 트라우마 (주인공 박기태와):
  둘 다 "가족을 지키기 위해서"라는 논리로 행동함.
  기태는 딸을 지키려 가짜 파산을 택했고,
  민혁은 가문을 지키려 재산 은닉을 택했음.
  방법이 다를 뿐, 출발점은 같은 '두려움'.

운명을 가른 선택:
  기태는 위기 앞에서 "진심이 무엇인가"를 물었고,
  민혁은 위기 앞에서 "어떻게 살아남을 것인가"만 물었다.

빌런의 논리:
  "나는 평생 아버지를 위해 살았다. 수아는 그러지 않았다.
   그런데 왜 아버지는 나보다 수아를 더 믿는 것처럼 행동하는가?
   나는 정당한 것을 가져가는 것뿐이다."
  → 이 논리는 독자에게 설득력은 있지만 공감은 없음.
  → 설득력과 공감의 분리 — 이것이 민혁을 진짜 빌런으로 만드는 요소.
```

## 욕망과 두려움

```
표면적 욕망: 아버지의 재산을 지키는 것.
진짜 욕망:   아버지에게 "네가 맞았다"는 말을 듣는 것.
             평생 아버지의 인정을 원했으나, 방법이 틀렸음.
가장 큰 두려움: 아버지가 자신보다 수아를 선택하는 것.
               단순한 재산 문제가 아니라, 자신의 존재 가치가 부정되는 것.
```

## 주인공과의 관계

```
초기 관계: 아버지(박기태)와 표면적으로 가까운 장남. 사업을 물려받은 후계자.
갈등 요인: 기태의 가짜 파산 → 민혁은 이를 진짜로 믿고 재산 은닉 착수
관계 변화 방향: 가짜 파산이 공개되며 민혁의 행동 전모가 드러남
결말에서의 최종 관계: 기태가 "내 눈을 봐라"며 모든 증거 제시 → 민혁 퇴장.
                      수아가 후계자로 지명되며 관계 단절. 용서는 없음, 단지 끝.
```
