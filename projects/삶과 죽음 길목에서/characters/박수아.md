# 캐릭터 프로필 — 박수아 (EP1 핵심 조력자)

> 에피소드 1 "900억보다 무거운 것"의 핵심 조력자. **희생적 수호자.**

---

## 인물 형상화 6가지 특징 체크

```
[v] 고정관념 활용: '버림받은 딸' = 원망과 복수심 가득할 것이라는 기대 전복.
    → 오히려 아버지가 찾아오자 가장 정성껏 맞이함. 독자의 예상을 완전히 뒤집음.

[v] 신체적 특징: 항상 맨발이거나 실내화. 돈이 없어 신발이 낡았지만 아버지에게 줄 돈을
    모으기 위해 새 신발을 사지 않음. 맨발로 쫓아나오는 장면이 핵심 Show 포인트.

[v] 말투와 행동: 짧고 따뜻한 말투. 상대의 말 끊지 않음. 하지만 울지 않는다 —
    절박할수록 더 조용하게 웃는다. (이게 더 슬프다.)

[v] 상황 설정: 3년 전 남편 사망 + 6살 아들 지운 양육 + 퇴거 위기 반지하 거주.
    → 누가 봐도 도움받아야 할 처지인데 오히려 아버지를 도우려 함.

[v] 특별한 능력: 타인의 고통을 자기 고통보다 먼저 감지하는 공감 능력.
    → 아버지의 눈빛에서 '시험'이 아닌 '외로움'을 먼저 읽는다.

[v] 태생적 특징: 이름 '수아(秀兒)' — 뛰어난 아이. 아이러니하게도 집안에서 가장
    뛰어난 사람이지만 가장 가난하게 살고 있다.
```

## 외형

```
체형: 165cm, 마른 편. 피곤함이 몸에 배어있지만 눈빛이 맑다.
외모 특징: 손이 트고 갈라져 있음 (배달·설거지 노동). 머리는 항상 급하게 묶음.
주로 입는 옷/스타일: 낡은 플리스 점퍼와 청바지. 아들 지운이 그린 그림이 냉장고에 붙어있는 집.
특징적인 외형 포인트: 맨발 또는 낡은 슬리퍼. 신발이 항상 문 밖에 있고 자신은 신지 않는다.
```

## 성격 및 말투

### 핵심 성격 키워드

```
따뜻한 침착함 / 절박한 삶 속의 유머 / 원망 없음 / 헌신 / 맑음
```

### 말투 특징

```
- 존댓말/반말: 아버지에겐 존댓말(오랜 단절 때문에 어색한 공손함). 아들 지운에겐 친구처럼 반말.
- 자주 쓰는 표현: "괜찮아요." / "있어요, 충분해요." / "지운아, 손 씻어."
- 감정이 격해질 때: 웃음. 역설적으로 웃음이 먼저 나온다. 울지 않으려는 의지.
- 절대 쓰지 않는 표현: "왜 그때 그랬어요" — 과거에 대한 원망 표현을 하지 않음.
```

### 대화 예시 (AI 학습용)

```
평소:         "지운아, 밥 다 됐어. 손 씻고 와."
화났을 때:    (화를 내지 않는다. 대신 더 조용해짐. "알겠어요."가 가장 강한 거절.)
당황했을 때:  "어... 잠깐만요, 정리할게요." (반지하 방 치우며)
아버지에게:   "진지 드셨어요? 저 고등어 있는데... 많지는 않아요."
극도로 위기일 때: (봉투를 쥐어주며 맨발로 뛰어나와) "아빠는 늙었으니까, 우리가 벌면 돼요."
```

## 배경 스토리

```
과거:
  - 20대에 아버지 반대를 무릅쓰고 결혼 → 의절
  - 남편과 행복하게 살았으나 3년 전 교통사고로 사망
  - 아들 지운(6세)과 단둘이 반지하 거주
  - 그래픽 디자이너였지만 배달 알바가 수입이 나아서 직업 전환

현재 상황:
  - 건물주로부터 재개발로 인한 퇴거 명령서 수령 (3개월 후 나가야 함)
  - 아들 지운이 천식 있어 지하 습기가 건강에 안 좋은 걸 알지만 이사비 없음
  - 남편 보험금 + 아들 학자금으로 모은 돈이 전재산 = 봉투에 담기는 금액

숨겨진 비밀:
  - 죽은 남편의 영정 사진 옆에 아버지와 함께 찍은 오래된 사진이 있다.
    아버지가 찾아왔을 때 이 사진을 가리려 했지만, 기태가 먼저 발견함.
  - 아들 지운은 할아버지를 사진으로만 알고 있는데, 기태가 처음 들어왔을 때
    "할아버지다!"라고 반겼다. 수아는 아들에게 할아버지 이야기를 나쁘게 하지 않았음.

트라우마:
  - 의절 소식을 들었을 때, 아버지가 자신의 결혼을 반대한 게 아니라
    '딸이 없어도 된다'고 판단한 것처럼 느껴졌던 상처.
  - 하지만 남편이 죽은 후 아버지에게 연락하고 싶었지만 먼저 연락할 수가 없었음.
```

## 양면성 & 입체성

```
외적 면모: 가난한 미혼모 과부. 모든 것이 부족함.

내적 가치: 물질적 빈곤이 영혼을 잠식하지 못한 사람. '나'보다 '다른 사람'이 먼저.

충돌 지점: 봉투를 쥐어주는 순간.
  이건 영웅적 결단이 아니다. 수아 입장에서는 '당연한 것'이다.
  아버지가 없으면 추울 것 같아서. 그게 전부.
  → 자신의 전 재산을 주면서도 "우리가 벌면 되잖아요"라고 말하는 것이
    이 인물의 가장 핵심적인 충돌 지점 — 빈곤 속의 풍요.
```

## 욕망과 두려움

```
표면적 욕망: 지운과 함께 지하 아닌 곳에서 살고 싶다.
진짜 욕망:   아버지가 자신을 다시 '딸'이라고 불러주는 것.
가장 큰 두려움: 아버지가 자신처럼 차가운 길바닥으로 내몰려 인간적 존엄을 잃는 것.
               자신이 아버지를 돕지 못한 채 아버지가 먼저 떠나는 것.
```

## 성장 아크 (12단계 서사 기준)

```
[변화 전]
1단계: 반지하에서 지운과 조용히 버티는 일상.
2단계: 아버지가 초라한 모습으로 문 앞에 나타남 (소명 = 선택의 기로).
3단계: "도와줄 돈도 없고, 아버지가 받아줄지도 모르겠다" (무력감).

[변화 과정]
4~6단계: 고등어 한 토막으로 아버지를 대접. 자신의 밥을 덜어드림.
          죽은 남편 영정 앞에서 아버지가 오열하는 장면 목격.
7~8단계: 퇴거 명령서를 아버지가 발견. 자신의 위기가 드러남.
  - 항상성: "이건 제 일이에요. 아버지 걱정 안 하셔도 돼요." (도움 거부)

[변화 후]
9단계: 봉투를 쥐어줌. "아빠는 늙었으니까 우리가 벌면 돼요."
11단계: 아버지의 진짜 시험이었음을 깨닫고 처음으로 운다.
12단계: 아버지 손을 잡고 여명의원 로비를 걸어나감.
        "이제 진짜 집으로 가자"는 아버지 말에 말없이 고개를 끄덕이며
        아버지의 낡은 외투 자락을 꼭 쥔다.
```
