# 캐릭터 프로필 — 박수아 (EP1 막내딸)

> 에피소드 1 "1000억보다 무거운 것"의 막내딸. **착한 척하다 외면하는 자식.**

---

## 인물 형상화 6가지 특징 체크

```
[v] 고정관념 활용: '반지하에 사는 가난한 막내딸' = 불쌍한 피해자, 순수한 사람이라는 기대 전복.
    → 실제로는 아버지의 돈에 의존하며 살아왔고, 파산 후에는 자기 살기 바빠 외면한다.
    → 민혁처럼 적극적 악의는 없지만, 결과적으로 아버지를 실망시키는 것은 같다.

[v] 신체적 특징: 손이 트고 거칠다 (두 개의 아르바이트). 피곤한 눈.
    아버지를 만날 때만 립스틱을 바르는 습관 — 단정하게 보이려는 무의식적 노력.

[v] 말투와 행동: 말이 많고 친근한 편. 아버지 앞에서 살갑게 굴지만
    생일 당일엔 전화를 깜빡 잊는다. 바쁘다는 핑계로.

[v] 상황 설정: 반지하 거주 / 아르바이트 두 개 / 이혼 후 혼자 살고 있음.
    → 경제적으로 힘들기 때문에 아버지에게 의존하는 것이 '당연한 것'이 되어버렸음.

[v] 특별한 능력: 상황을 자기에게 유리하게 설명하는 능력.
    "바빠서요." / "몸이 안 좋았어요." — 항상 그럴듯한 이유가 준비되어 있다.

[v] 태생적 특징: 이름 '수아(秀兒)' — 뛰어난 아이. 아버지가 기대를 걸었던 막내딸.
    그 기대가 오히려 수아를 부담스럽게 만들었고, 아버지와의 거리감이 생긴 원인.
```

## 외형

```
체형: 165cm, 마른 편. 피곤함이 배어 있다.
외모 특징: 손이 트고 거칠다. 아버지를 만날 때만 립스틱을 바른다.
주로 입는 옷/스타일: 낡은 플리스 점퍼와 청바지.
특징적인 외형 포인트: 핸드폰을 자주 확인한다. 알림이 많다. 그러나 아버지 생일날은 확인을 못 한다.
```

## 성격 및 말투

### 핵심 성격 키워드

```
살갑고 친근한 외면 / 자기 합리화 / 의존적 / 무의식적 무관심 / 뒤늦은 죄책감
```

### 말투 특징

```
- 존댓말/반말: 아버지에게 존댓말 (친근한 어투). "아버지, 저예요~" 식의 가벼운 존댓말.
- 자주 쓰는 표현: "바빠서요." / "몸이 좀 안 좋았어요." / "그나저나 저 요즘 진짜 힘든데..."
  → 아버지 안부를 묻다가 어느새 자기 이야기로 넘어가는 패턴.
- 감정이 격해질 때: 울음. 금방 울고 금방 그친다. 눈물이 자기 주장의 도구가 될 때도 있다.
- 파산 소식 후: 처음엔 걱정하는 척 연락하다가, 1주일 후부터 연락이 뜸해짐.
  → "어떻게 해드려야 할지 모르겠어서요..." (자기 합리화)
```

### 대화 예시 (AI 학습용)

```
생일 전주 방문:    "아버지, 오랜만이에요. 얼굴이 좀 수척하신 것 같은데. 그나저나 저 요즘
                    월세가 좀 밀렸거든요, 이번 달만 좀 도와주시면 안 될까요?"
파산 직후 전화:    "아버지 괜찮으세요? 어떻게 된 거예요? 저한테 왜 말을 안 하셨어요."
1주일 후:         (연락 없음)
3주일 후, 다시 연락: "아버지, 저 요즘 몸이 너무 안 좋아서... 아버지 어떠세요? (짧은 통화 후) 저 좀 쉬어야 할 것 같아서요."
기태가 고아원에 기부한다는 말을 들었을 때: "아버지 그게 무슨 말이에요! 우리한테 남겨주셔야죠!"
```

## 배경 스토리

```
과거:
  - 30대 초반에 결혼했으나 수년 전 이혼. 이유는 잘 알려지지 않음.
  - 전 직장: 소규모 인테리어 회사 직원 (현재 퇴직).
  - 아버지로부터 크고 작은 경제적 도움을 받아왔음 (월세, 생활비 등).
    아버지가 돈을 주는 것을 당연하게 여겨왔음.

현재 상황:
  - 반지하 거주. 월세 두 달 연체.
  - 아르바이트 두 개. 간신히 생활.
  - 생일 전주에 아버지를 찾아가 알랑방구를 피우고 월세 지원을 요청.
  - 정작 아버지 생일 당일엔 바빠서 전화를 깜빡 잊음.
  - 파산 소식 후 처음엔 걱정하는 척 연락하다가 점점 연락이 뜸해짐.
  - 친척 모임에서 민혁이 "아버지가 짐이 됐다"고 말할 때 적극적으로 반대하지 않음.

내면:
  - 아버지가 자신을 기대했다는 것을 알고 있고, 그 기대에 미치지 못한다는 죄책감이 있음.
  - 하지만 그 죄책감이 행동으로 이어지지 않는다 — 자기 살기도 바쁘다는 현실이 있으므로.
  - 아버지와 절연을 통보받았을 때 처음으로 자신이 아버지에게 어떻게 대했는지를 돌아봄.
  - 하지만 이미 늦었다.
```

## 양면성 & 입체성

```
외적 면모: 살갑고 친근한 막내딸. 생활이 어려운 가난한 딸. 불쌍해 보이는 처지.

내적 과거: 아버지의 경제적 지원을 당연하게 여겨왔고, 파산 후에는
           '어떻게 해드려야 할지 모르겠어서'라는 핑계로 외면했다.
           민혁처럼 악의적이지 않지만, 결과적으로 같은 선택을 한다.

충돌 지점: 기태가 고아원 재단 기부 결정을 통보하는 순간.
  수아는 처음으로 목소리를 높인다. "우리한테 남겨주셔야 하잖아요!"
  기태가 아무 말 없이 전화를 끊는다.
  수아는 전화기를 들고 한참을 아무것도 하지 못한다.
  → 독자는 이 순간 수아에 대한 연민과 비판을 동시에 느낀다.
```

### 민혁과의 차이

```
박민혁 (장남):
  - 아버지 파산 후 적극적으로 등을 돌리고 "짐이 된다"고 말하는 악의적 행동
  - 계획적이고 명확한 배신

박수아 (막내딸):
  - 파산 후 연락이 뜸해지고, 모임에서 민혁에게 반대하지 않는 소극적 방관
  - 악의보다는 무관심과 자기 합리화
  → 두 자식은 다른 방식으로 아버지를 실망시키지만, 결과는 같다.
  → 수아를 무조건 나쁜 사람으로 그리지 않되, 아버지가 절연을 결심하기에 충분한 근거 제공.
```

## 욕망과 두려움

```
표면적 욕망: 생활의 안정. 아버지의 지원.
진짜 욕망:   아버지와 좋은 관계를 유지하면서 도움도 받고 싶다 (두 가지를 동시에).
가장 큰 두려움: 아버지와 완전히 단절되는 것 — 그것이 자기 잘못임을 인정해야 하는 순간이 두렵다.
```
