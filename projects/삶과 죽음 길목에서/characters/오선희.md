# 캐릭터 프로필 — 오선희 (EP3 주인공)

> 에피소드 3 "천 원짜리 물 한 병"의 주인공. **단단한 벽에서 흐르는 강물로.**

---

## 인물 형상화 6가지 특징 체크

```
[v] 고정관념 활용: 냉혹한 여성 CEO의 전형 — 실제로는 32세에 남편을 잃고 혼자 두 아들을 키운
    생존자. 차가움은 방어기제이지 본질이 아니다.

[v] 신체적 특징: 항상 빨간 볼펜을 꽂고 다님 (보고서에 빨간 줄 치는 습관).
    회의실에서 누군가의 보고서에 빨간 줄을 치는 순간이 그녀의 '지배' 신호.

[v] 말투와 행동: "결론부터 말해라. 사장은 바쁜 사람이다." — 군더더기 없는 비즈니스 언어.
    하지만 강아름이 가져온 아들의 동화책 앞에서 처음으로 말문이 막힌다.

[v] 상황 설정: 건강 이상 징후(혈압·당뇨 악화). 10년간 아들과 단절.
    아들이 없는 삶에 익숙해졌다고 생각하지만, 회사 창고에 아들의 유치원 그림을 넣어두었다.

[v] 특별한 능력: 숫자를 통해 사람의 진심을 읽는 능력.
    → 강아름의 지출 내역서(마지막 천 원으로 물 한 병)에서 진심을 읽는다.

[v] 태생적 특징: 이름 '선희(善姬)' — 착하고 아름다운 여성. 이름과 정반대의 삶을 살아왔다.
    결말에서 처음으로 이름값을 하는 사람이 됨.
```

## 외형

```
체형: 키 162cm, 단정하고 절제된 체형. 나이보다 젊어 보이지만 눈가가 피곤하다.
외모 특징: 흰 머리를 염색해 관리. 항상 단정한 정장. 빨간 볼펜은 왼쪽 윗주머니에.
주로 입는 옷/스타일: 네이비 또는 차콜 정장. 절대 화려하지 않음. 기능적 의복.
특징적인 외형 포인트: 빨간 볼펜. 회의실에서는 절대 내려놓지 않는다.
                       쓰러진 후 처음으로 볼펜을 잃어버린다.
```

## 성격 및 말투

### 핵심 성격 키워드

```
냉혹한 생존자 / 억눌린 모성 / 완벽주의 / 고집 / 균열의 시작
```

### 말투 특징

```
- 존댓말/반말: 모든 직원에게 존댓말 (하지만 차갑게). 직급 상관없이 "씨" 호칭.
  → 아들에게는 반말(과거형 대화 기억에서만).
- 자주 쓰는 표현: "결론부터." / "근거가 뭐야." / "다시."
- 감정이 격해질 때: 볼펜으로 서류를 가볍게 탁탁 치기 시작함 (위험 신호).
- 절대 쓰지 않는 표현: "고마워" / "미안해" — 30년 동안 한 번도 쓰지 않음.
  → 결말에서 강아름에게 처음 "고맙다"라고 말하는 것이 클라이맥스.
```

### 대화 예시 (AI 학습용)

```
평소:         "결론부터. 다시."
화났을 때:    (볼펜 탁탁.) "근거가 뭐야. 숫자로 가져와."
당황했을 때:  "...잠깐." (처음으로 볼펜을 멈춤. 3초 침묵.)
강아름에게:   "불쌍해서 데려온 거 아니야. 실력으로 증명해봐."
쓰러진 후:    (물병 받으며) "...고맙다." (두 글자. 처음으로.)
```

## 배경 스토리

```
과거:
  - 32세에 남편 사망. 두 아들(큰아들 오재현 40대, 작은아들 오재원 38세)을 혼자 키움
  - "사람을 믿지 마라, 숫자만 믿어라"는 철칙으로 회사 키움
  - 아들들을 후계 도구로 대하며 "1등이 아니면 의미 없다" 압박
  - 10년 전, 작은아들 재원과 큰 다툼 후 연락 두절 (이유는 38화까지 비밀)
  - 큰아들 재현은 회사에 있지만 모자 관계는 차갑고 비즈니스적

현재 상황:
  - 건강검진에서 심장 + 혈압 문제 이상 소견
  - 여명의원 인근 병원을 다니기 시작 (여명의원과 연결되는 계기)
  - 강아름이 신입 사원으로 입사한 직후 상황 전개

숨겨진 비밀:
  - 작은아들 재원과 다툰 이유: 재원이 "어머니 때문에 인생이 망가졌다"며 독립 선언.
    선희는 그것을 '배신'으로 받아들였지만, 실제론 재원의 말이 맞았음.
  - 회사 창고에 재원의 유치원 그림이 보존되어 있다.
    선희도 왜 버리지 못하는지 모른다.
  - 강아름의 고아원이 선희가 20년간 익명 후원한 곳과 일치 (72화까지 비밀).

트라우마:
  - 남편이 죽은 날 아들들이 울고 있을 때 선희는 울지 못했다.
    '울면 무너진다'고 생각해서. 그 날 이후 30년간 울지 않음.
    → 강아름이 처음으로 선희 앞에서 선희 이름을 부르며 울 때 선희의 뭔가가 깨진다.
```

## 양면성 & 입체성

```
외적 면모: 냉혹한 여성 CEO. 빨간 볼펜으로 사람을 평가하는 관리자.

내적 과거: 32세에 홀로 남겨진 어머니. 살아남기 위해 감정을 꺼버린 것뿐.
           아들의 유치원 그림을 30년 동안 버리지 못하는 사람.

충돌 지점: 강아름이 자신의 마지막 천 원으로 선희에게 물을 사다준 순간.
  선희는 숫자로 사람을 읽는다. 아름의 지출 내역에서 그 천 원의 의미를 이해한다.
  처음으로 볼펜을 내려놓는다.
```

### 히어로-빌런 양면성

```
공유하는 트라우마 (아들 재원과):
  둘 다 남편/아버지의 부재라는 결핍에서 출발.
  선희 → 결핍을 지워버리기 위해 더 강해짐 (감정 차단)
  재원 → 어머니의 냉혹함이 두 번째 결핍이 됨

운명을 가른 선택:
  선희가 "살아남으려면 차가워야 한다"고 선택한 날,
  재원은 "더 이상 이 집에서 살아남고 싶지 않다"고 선택.

빌런의 논리 (재원 기준):
  "어머니는 아버지 없는 자식이 불쌍한 게 아니라,
   어머니 있는 자식인데 아버지보다 더 외롭게 컸다.
   어머니가 사업으로 증명한 건 성공이 아니라, 내가 필요 없다는 것이었다."
```

## 욕망과 두려움

```
표면적 욕망: 회사를 잘 굴리는 것.
진짜 욕망:   자식보다 내가 먼저 변해야 한다는 것을 인정하는 것.
             (하지만 이것을 자신에게 인정하는 데 63화가 걸림)
가장 큰 두려움: 재원이 다시 찾아왔을 때 그것이 '진심'인지 '계산'인지 구분할 수 없는 것.
               자신이 변하기 전에 죽는 것.
```

## 성장 아크 (12단계 서사 기준)

```
[변화 전]
1단계: 회장실 + 병원 왕복. 아들 없는 일상이 '정상'이라 생각.
2단계: 건강 이상 소견 + 강아름 입사 (두 가지 균열의 시작).
3단계: "내가 바뀔 이유가 없다. 나는 틀리지 않았다."

[변화 과정]
4~6단계: 강아름의 업무 능력 인정. 아름의 고아원 출신을 알게 됨.
          "불쌍해서 데려온 거 아니야."
7~8단계: 쓰러짐. 아름의 마지막 천 원 물병.
  - 항상성: "고맙다는 말은 약해 보이는 것이다. 절대 안 한다."
    → 하지만 목이 탄다. 물이 목구멍으로 넘어간다.

[변화 후]
9단계: 재원이 달려옴. 그 눈에서 '계산'을 읽지만 아름의 눈에서 '진심'을 읽음.
11단계: "내가 30년간 틀렸던 것들이 있다."
12단계: 강아름이 내민 물병을 받으며 "고맙다." — 두 글자. 처음으로.
```
