# 캐릭터 프로필 — 오재원 (EP3 갈등 유발자)

> 에피소드 3 "천 원짜리 물 한 병"의 갈등 유발자. **어머니가 만든 상처 / 돌아온 아들의 진심은 무엇인가.**

---

## 인물 형상화 6가지 특징 체크

```
[v] 고정관념 활용: '10년 만에 돌아온 아들' = 회개하고 돌아온 탕자의 기대 전복.
    → 오선희는 재원의 눈에서 '계산'을 읽는다. 하지만 독자는 끝까지 알 수 없음.
    → 진심인가, 계산인가 — 이 불확실성이 EP3의 핵심 긴장.

[v] 신체적 특징: 어머니를 닮은 눈. 하지만 같은 눈으로 전혀 다른 것을 봄.
    어머니가 볼펜을 쥐듯, 재원은 커피잔을 쥐고 있음 — 같은 불안, 다른 도구.

[v] 말투와 행동: 어머니처럼 논리적. 하지만 어머니와 달리 감정이 새어나옴.
    "어머니"를 부르는 목소리가 일관되지 않음 — 때로는 낮고, 때로는 떨림.

[v] 상황 설정: 쓰러졌다는 소식에 달려왔지만, 타이밍이 너무 정확함.
    오선희는 10년간 아들의 소재조차 몰랐는데, 입원 소식은 어떻게 알았는가?

[v] 특별한 능력 (결핍으로서의 능력): 어머니의 약점을 정확히 알고 있음.
    어머니 곁에서 자란 사람이기 때문에. 그 지식이 무기가 될 수도, 화해가 될 수도 있음.

[v] 태생적 특징: 이름 '재원(才源)' — 재능의 근원. 어머니가 직접 지은 이름.
    "네가 이 집안의 근원이 되길 바란다"는 뜻으로 지었지만,
    어머니는 정작 재원을 집안의 도구로 키웠음 — 이름과 현실의 간극이 재원의 상처.
```

## 외형

```
체형: 178cm, 어머니 오선희의 골격을 닮음.
외모 특징: 어머니와 닮은 눈매. 하지만 눈에서 피로감이 있음 — 10년의 무게.
주로 입는 옷/스타일: 평범한 캐주얼. 10년 전 결별 이후 '성공한 척'을 하지 않음.
특징적인 외형 포인트: 어머니를 닮은 눈.
                       어머니 앞에서 커피잔을 두 손으로 쥐는 버릇 (불안할 때).
```

## 성격 및 말투

### 핵심 성격 키워드

```
억압된 상처 / 어머니에 대한 사랑과 분노 동시 존재 / 불확실한 동기 / 10년의 침묵 / 돌아온 이유
```

### 말투 특징

```
- 존댓말/반말: 어머니에게 존댓말. 하지만 10년의 공백 때문에 어색함이 남음.
- 자주 쓰는 표현: "어머니." (단 한 글자. 뭔가를 이어 말하려다 멈추는 패턴)
- 감정이 격해질 때: 어머니와 비슷하게 논리적이 되려 하지만, 목소리가 떨림.
- 절대 쓰지 않는 표현: "죄송합니다" — 재원은 사과하지 않음. (어머니도 그랬기 때문에)
```

### 대화 예시 (AI 학습용)

```
첫 재회:       "어머니." (그리고 멈춤. 10년치가 걸려서 다음 말이 안 나옴)
어머니에게:    "제가 왔습니다." (거리감. 가깝고도 먼 말투)
강아름에게:    "어머니 상태가 어떤지 알 수 있습니까?" (정보를 구하는 눈)
선희 눈을 피할 때: (커피잔을 두 손으로 쥠. 시선이 내려감.)
결정적 순간:   "...어머니가 다치셨다고 해서." (다음 말 없음. 그것뿐.)
```

## 배경 스토리

```
과거:
  - 어린 시절: 아버지 부재 + 어머니의 극도의 성공 압박 속에서 성장
  - 큰아버지(오재현)는 어머니를 닮아 사업에 순응했지만, 재원은 그러지 못했음
  - 20대: 어머니의 회사에 들어갔지만 "어머니 덕분에 취직한 사람"으로 취급받음
  - 10년 전: "어머니 때문에 인생이 망가졌다"는 말과 함께 독립 선언
    → 어머니는 이것을 '배신'으로 받아들였지만, 재원의 말은 틀리지 않았음

현재 상황:
  - 어머니가 쓰러졌다는 소식을 듣고 달려옴 (55화~)
  - 10년의 공백을 어떻게 채울 수 있는지, 채울 수 있기는 한지 모름
  - 오선희는 재원의 눈에서 '계산'을 읽는다
    → 재원이 정말 계산적인가, 상처받은 채 돌아온 것인가 — 결말까지 불확실

숨겨진 내면:
  - 어머니 회사 창고에 자신의 유치원 그림이 있다는 것을 모름
  - 어머니가 30년간 자신의 그림을 버리지 못했다는 것을 알게 되는 순간 (73화~)
    → 이 순간이 재원의 분기점 — 계산이 있었든 없었든, 뭔가가 무너짐

트라우마:
  - "어머니는 아버지 없는 자식이 불쌍한 게 아니라,
     어머니 있는 자식인데 아버지보다 더 외롭게 컸다.
     어머니가 사업으로 증명한 건 성공이 아니라, 내가 필요 없다는 것이었다."
  - 이 말이 재원의 자기 이해 전체이자, 10년 침묵의 이유.
```

## 양면성 & 입체성

```
외적 면모: 쓰러진 어머니 소식에 정확한 타이밍에 나타난 '돌아온 아들'.
           오선희는 계산으로 읽고, 강아름은 판단을 보류함.

내적 과거: 어머니에게 버려진 것이 아니라, 어머니로부터 도망쳤음.
           도망친 것이 어머니를 사랑하지 않아서가 아니라,
           사랑하기 때문에 더 이상 상처받을 수 없어서였음.

충돌 지점: 어머니가 창고에 자신의 유치원 그림을 보존해두었다는 것을 아는 순간.
           '필요 없는 아들'이라고 믿어왔는데, 그것이 틀렸을 수도 있다는 균열.
```

### 히어로-빌런 양면성

```
공유하는 트라우마 (주인공 오선희와):
  둘 다 '남편/아버지의 부재'에서 출발한 결핍.
  선희 → 결핍을 지워버리기 위해 더 강해짐 (감정 차단)
  재원 → 어머니의 냉혹함이 두 번째 결핍이 됨

운명을 가른 선택:
  선희는 살아남기 위해 차가워졌고,
  재원은 더 이상 차가워지지 않기 위해 떠났다.
  둘 다 틀리지 않았다. 그것이 이 관계를 비극으로 만드는 이유.

빌런의 논리 (불확실한 악의로서):
  "어머니가 나를 필요 없어했다면, 나도 어머니를 필요 없어할 권리가 있다.
   하지만 쓰러졌다는 소식에 달려온 건... 왜였는가."
  → 재원 자신도 자신의 동기를 정확히 모름.
  → 이 불확실성이 EP3에서 가장 인간적인 설정.
```

## 욕망과 두려움

```
표면적 욕망: (겉으로 드러나지 않음. 행동으로만 추측 가능.)
진짜 욕망:   어머니에게 "내가 있어도 된다"는 것을 인정받는 것.
             사업 성과로가 아니라, 그냥 아들로서.
가장 큰 두려움: 돌아왔는데 어머니에게 다시 '계산'으로만 보이는 것.
               사실은 아무 계산 없이 달려온 것인데, 그것을 증명할 수 없는 것.
```

## 주인공과의 관계

```
초기 관계: 10년 연락 두절. 존재만 알고 있는 아들.
갈등 요인: 쓰러진 어머니 소식에 정확한 타이밍 등장 → 선희가 '계산'으로 읽음
관계 변화 방향: 어머니 창고의 유치원 그림 → 재원의 동기에 균열
결말에서의 최종 관계: 오선희의 결말은 강아름과의 장면.
                      재원은 결말에 없음 — 선희는 '딸'이 아닌 '직원'을 선택함.
                      재원과의 화해는 열린 결말. 독자의 해석에 맡김.
```
