# 캐릭터 프로필 — 이덕수 (EP2 갈등 유발자)

> 에피소드 2 "가죽 손가방"의 갈등 유발자. **기록 파괴자 / 아버지의 기억을 도구로 쓴 장남.**

---

## 인물 형상화 6가지 특징 체크

```
[v] 고정관념 활용: '노부모를 요양원에 모신 효자' = 사회적으로 이해받는 선택처럼 보이는 전복.
    → 실제로는 아버지를 격리시켜 자산을 통제하려는 기획된 유기.
    → "최고급 요양원에 모셨습니다"는 말이 가장 완벽한 알리바이가 됨.

[v] 신체적 특징: 항상 명함을 먼저 내미는 습관 (자신의 직함이 곧 자신임).
    아버지 앞에서만 명함을 꺼내지 않는다. (아버지가 직함보다 더 오래된 존재이기 때문.)

[v] 말투와 행동: 부드럽고 공손한 말투. 모든 말이 '아버지를 위해'로 포장됨.
    하지만 내용은 항상 아버지의 선택권을 제거하는 방향.

[v] 상황 설정: 아버지의 치매 진단을 위조해 강제 성년후견인 신청을 준비 중.
    가죽 손가방 안의 내용물이 자신의 모든 계획을 무력화할 수 있다는 것을 직감.

[v] 특별한 능력: 법적·행정적 절차를 이용한 장기 전략 수립.
    단기 폭력이 아닌 제도를 통해 부모를 무력화하는 '합법적 폭력'의 전문가.

[v] 태생적 특징: 이름 '덕수(德秀)' — 덕이 있고 뛰어난 사람. 아버지가 직접 지어준 이름.
    이름 안에 아버지의 기대가 담겨 있음 — 그 이름을 짊어진 채 아버지를 배신하는 아이러니.
```

## 외형

```
체형: 175cm, 약간 배가 나온 체형. 50대 중소기업 대표의 전형적 외형.
외모 특징: 항상 단정하게 빗은 머리. 포근해 보이는 얼굴 — 악의가 드러나지 않는 외모.
주로 입는 옷/스타일: 정장 + 캐주얼 사이. 고급이지만 과시하지 않음.
특징적인 외형 포인트: 명함 케이스. 언제나 왼쪽 안주머니.
                       아버지 앞에서는 꺼내지 않는다.
```

## 성격 및 말투

### 핵심 성격 키워드

```
부드러운 폭력 / 합법 뒤의 악의 / 아버지에 대한 죄책감 / 가면 / 자기 합리화
```

### 말투 특징

```
- 존댓말/반말: 아버지에게 항상 공손한 존댓말. 단 한 번도 반말 없음.
  → 이 공손함이 오히려 더 섬뜩함. 감정이 없기 때문에 흔들리지 않음.
- 자주 쓰는 표현: "아버지 편하시게 해드리려고요." / "다 아버지 위해서입니다." / "제가 처리할게요."
- 감정이 격해질 때: 더 조용해지고 더 느리게 말함. 속도가 줄수록 위험.
- 절대 쓰지 않는 표현: "제가 틀렸습니다." — 덕수의 어휘에 없는 단어.
```

### 대화 예시 (AI 학습용)

```
평소:          "아버지, 여기서 편히 지내시면 됩니다."
아버지에게:    "그 가방은 제가 맡아두겠습니다. 잃어버리시면 안 되니까요."
간호사에게:    "아버지가 요즘 기억이 많이 흐려지셨어요. 혹시 이상한 말씀 하시면 연락 주세요."
장부 발견 시:  (손이 멈춤) "...이게 여기 있었군요." (목소리 변화 없음. 그것이 공포.)
단죄 장면:     "저는 아버지 위해서 한 겁니다." (끝까지 포기하지 않는 논리)
```

## 배경 스토리

```
과거:
  - 아버지의 기대를 받고 자란 장남. 하지만 아버지의 사업 방식(기록·신뢰 중심)을 답답하게 여겼음
  - 30대에 아버지 회사에서 독립해 자신의 건설사를 차림
  - 아버지의 도움 없이 성공했다고 믿지만, 실제로는 아버지의 인맥·신용을 물려받은 것
  - 치매 초기 징후가 보이는 아버지를 보면서 '상속 시기'를 계산하기 시작

현재 상황:
  - 아버지를 여명의원에 입소시키고 가죽 손가방의 존재를 위협으로 판단
  - 의사를 통해 치매 진단서 확보 시도 (이태석이 정상임을 알면서도)
  - 성년후견인 신청 서류를 준비 중
  - 어머니 경자의 연금 통장도 이미 관리 명목으로 통제 중

트라우마:
  - 아버지의 장부에 '차남 민수 생활비'가 기록되어 있다는 것을 어린 시절에 목격.
    (아버지가 덕수 몰래 차남을 도왔다는 사실 — 65화 비밀)
  - 이것이 덕수의 '아버지에 대한 배신감'의 씨앗.
```

## 양면성 & 입체성

```
외적 면모: 아버지를 최고급 요양원에 모신 효자. 방문도 규칙적으로 옴.

내적 과거: 아버지가 자신을 사랑했지만, 차남에게도 같은 사랑을 줬다는 것을 평생 받아들이지 못함.
           "나만의 아버지"를 갖고 싶었음 — 이 결핍이 '독점'으로 발현됨.

충돌 지점: 장부가 법원에 제출되고, 자신의 모든 행동이 기록으로 증명되는 순간.
           아버지가 50년간 모든 것을 기록해두었다는 사실 — 덕수는 아버지가 '증거를 만들고 있었다'는 것을
           그 순간에야 이해함.
```

### 히어로-빌런 양면성

```
공유하는 트라우마 (주인공 이태석과):
  둘 다 '무언가를 독점하고 싶었던' 사람.
  태석은 진실을 기록해 지키려 했고,
  덕수는 재산을 기록에서 지우려 했음.

운명을 가른 선택:
  같은 '기록의 힘'을 봤지만,
  태석은 기록으로 진실을 수호했고,
  덕수는 기록을 파괴함으로써 진실을 지우려 했다.

빌런의 논리:
  "아버지는 나를 믿었어야 한다. 50년 동안 내가 곁에 있었는데.
   차남은 어디 있었나? 그런데 왜 아버지는 나를 시험하는 것인가?
   가죽 손가방 하나로 내가 지운 것들이 다 돌아온다는 게 말이 되는가?"
  → 이 논리에는 '상처받은 장남'의 설득력이 있음.
  → 하지만 그 상처가 어머니의 연금을 빼앗는 행동으로 이어지는 순간 설득력이 사라짐.
```

## 욕망과 두려움

```
표면적 욕망: 아버지의 재산을 안전하게 이전하는 것.
진짜 욕망:   아버지가 자신을 '가장 신뢰한 아들'로 봐주는 것.
             그 인정을 얻지 못하자 인정 대신 재산을 가져가기로 결심한 것.
가장 큰 두려움: 아버지의 장부에 자신에 대한 실망이 기록되어 있는 것.
               실제로 장부 마지막 페이지에 자신의 이름이 있을까봐 열어보지 못함.
```

## 주인공과의 관계

```
초기 관계: 아버지를 요양원에 입소시킨 '효자' 장남.
갈등 요인: 가죽 손가방 → 장부의 존재 → 진실의 증거
관계 변화 방향: 치매 조작 기도 → 강제 후견인 신청 → 장부 법원 제출로 전모 공개
결말에서의 최종 관계: "넌 내 아들이었어" — 이태석의 과거형 단절.
                      이름으로 불리지 않고 '아들이었던 사람'으로 끝남.
```
