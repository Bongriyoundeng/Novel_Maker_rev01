# 캐릭터 프로필 — 이태석 (EP2 주인공)

> 에피소드 2 "가죽 손가방"의 주인공. **기록하는 전략가.**

---

## 인물 형상화 6가지 특징 체크

```
[v] 고정관념 활용: '요양원의 말 없는 노인' = 무력하고 소외된 존재라는 기대 전복.
    → 실제로는 50년간 기록을 이어온 지성적 전략가. 침묵은 무능이 아닌 준비.

[v] 신체적 특징: 낡은 가죽 손가방을 항상 왼손에 들고 다님. 잠 잘 때도 침대 옆 바닥에 놓음.
    누가 가방을 만지려 하면 그때만 눈빛이 달라진다.

[v] 말투와 행동: 하루 평균 10문장 이하. '의사 표현 없는 노인'으로 분류될 정도.
    하지만 말 한마디가 결정적일 때만 나온다. "넌 내 아들이었어." (과거형)

[v] 상황 설정: 치매 '기운'이 실제로 있음 (초기). 하지만 기록 능력은 완벽히 유지.
    아들 덕수는 이를 치매로 위장하려 함.

[v] 특별한 능력: 50년간 모든 거래와 관계를 기록한 가족 장부.
    → 아날로그 기록이 디지털 위조를 이기는 서사.

[v] 태생적 특징: 이름 '태석(泰石)' — 태평한 바위. 흔들리지 않는다.
    아들에게 버려진 요양원에서도 장부를 든 채 흔들리지 않는다.
```

## 외형

```
체형: 키 168cm, 단단한 노인. 걸음이 느리지만 허리는 꼿꼿.
외모 특징: 흰 머리칼을 항상 단정히 빗어 넘김. 요양원 환자복 위에 반드시 카디건을 걸침.
주로 입는 옷/스타일: 요양원 환자복 + 베이지색 낡은 카디건 (20년 된 것).
특징적인 외형 포인트: 왼손에 항상 낡은 갈색 가죽 손가방. 자물쇠가 달려 있다.
                       손가방의 가죽이 닳아서 안쪽 천이 보이는 자리가 있지만 바꾸지 않는다.
```

## 성격 및 말투

### 핵심 성격 키워드

```
전략적 침묵 / 기록에 의한 정의 / 아내 수호 / 단호한 단절 / 마지막 자존심
```

### 말투 특징

```
- 존댓말/반말: 아내 경자에게만 존댓말. 나머지는 반말 또는 침묵.
- 자주 쓰는 표현: 말을 거의 하지 않음. "그래." / "알았어." 가 전부.
  결정적인 순간: "화내면 지는 거다." (자기 자신에게 하는 말)
- 감정이 격해질 때: 더 조용히 장부를 꺼낸다. 기록하는 행위 자체가 감정 표현.
- 절대 쓰지 않는 표현: 자식에게 "사랑한다" — 표현이 없는 세대.
  단, 아내 경자에게는 이름을 부른다. "경자야." (이것이 그의 애정 표현 전부)
```

### 대화 예시 (AI 학습용)

```
평소:         "그래." (긍정도 부정도 이 두 글자로)
아들 덕수에게: (말 없음. 장부를 꺼내 펼침. 덕수가 얼굴을 굳힘.)
아내 경자에게: "경자야, 왔어?" (두 단어. 이 두 단어에 50년이 담겨 있음)
간호사에게:   "가방은 내가 든다."
결정적 순간:  "넌 내 아들이었어." (과거형. 단절 선언.)
```

## 배경 스토리

```
과거:
  - 결혼 첫날부터 일기+가계부 형식으로 가족 장부를 기록하기 시작
  - 중소 건재상 50년 운영. 직원 이름과 봉급, 모든 거래를 손으로 기록
  - 아들 덕수가 가업을 이어받았지만 부도 위기에 여러 번 빠짐
  - 매번 빚을 갚아줬고, 그 모든 금액이 장부에 기록되어 있음
  - 3년 전부터 치매 초기 증상이 나타나기 시작 (아직 경미한 수준)

현재 상황:
  - 아들 덕수가 "치매가 심해지셔서"라는 명목으로 여명의원에 강제 입소
  - 실제로는 부모의 아파트와 예금을 관리하려는 목적
  - 아내 경자는 덕수가 면회를 차단하여 한 달째 못 오는 중
  - 장부에는 덕수의 위조 행위를 뒷받침할 모든 원본 기록이 있음

숨겨진 비밀:
  - 장부의 첫 페이지: "19XX년 X월 X일. 오늘부터 기록한다."
    결혼 첫날 썼던 것. 마지막 페이지는 아직 비어 있다.
  - 장부에는 의절한 차남(이덕훈)에게 몰래 보낸 생활비 기록도 있다.
    덕수는 이 사실을 모른다. 65화에서 차남이 증인으로 등장.

트라우마:
  - 자신이 기록을 남기는 이유: "내가 여기 있었다"는 것을 증명하기 위해.
    아버지가 흔적 없이 떠났고, 그래서 자신은 흔적을 남기기로 다짐한 인생.
```

## 양면성 & 입체성

```
외적 면모: 요양원의 조용하고 말 없는 노인. 치매 의심 환자.

내적 과거: 50년간 기록해온 가족 역사의 수호자.
           겉으로는 무력해 보이지만 실제로는 가장 강력한 무기를 들고 있다.

충돌 지점: 덕수가 가방을 빼앗으려 할 때.
  평소 조용하던 태석의 눈빛이 처음으로 바뀐다.
  "이 가방은 내가 든다." — 이 다섯 글자가 그의 50년 의지를 담은 전부.
```

### 히어로-빌런 양면성 (아들 덕수와의 관계)

```
공유하는 트라우마:
  태석은 아버지에게 아무것도 물려받지 못한 사람 (= 결핍에서 자수성가).
  덕수는 아버지에게 모든 것을 물려받으면서도 감사함 없이 자랐음.
  둘 다 '표현하지 않는 가족'에서 왔지만, 결과가 반대.

운명을 가른 선택:
  태석 → 결핍을 극복하려 기록과 노동으로 일어섬
  덕수 → 풍요 속에서 오히려 더 쉬운 길(부모 착취)을 선택

빌런의 논리 (덕수 기준):
  "아버지가 50년간 한 번도 '수고했다'는 말 한 마디 없었다.
   장부만 들고 다니며 내 실수를 기록했다. 그게 아버지의 사랑이었냐.
   그 장부가 결국 나를 죽이는 칼이 됐다." — 50%는 이해 가능.
```

## 욕망과 두려움

```
표면적 욕망: 아내 경자와 다시 같은 공간에 있고 싶다.
진짜 욕망:   장부가 증거가 되어 경자를 지키고, 기록이 자신의 존재 증명이 되는 것.
가장 큰 두려움: 기록 능력을 잃는 것 (치매가 심해지면 기억이 사라짐).
               그 전에 경자에게 장부를 전달하지 못하는 것.
```

## 성장 아크 (12단계 서사 기준)

```
[변화 전]
1단계: 요양원 304호. 매일 장부를 쓰고 경자를 기다린다.
2단계: 덕수가 후견인 신청을 시작함 (위기의 소명).
3단계: 혼자이고 말도 통하지 않는 환경. 치매 의심 딱지.

[변화 과정]
4~6단계: 간호사 나현수(장례지도사)가 태석이 쓴 장부를 우연히 봄. 조력자 연결.
7~8단계: 덕수가 가방을 빼앗으려 시도. 위기.
  - 항상성: 장부를 보여주면 가족이 완전히 끝난다는 두려움. "기다리면 돌아오겠지."

[변화 후]
9단계: 장부를 법원에 제출하기로 결심.
11단계: "기록은 죽지 않는다. 나는 여기 있었다."
12단계: 창가에서 경자의 손에 장부를 쥐어주며 눈을 감는다.
        "이제 당신이 증인이야."
```
