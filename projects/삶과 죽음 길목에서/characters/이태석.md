# 캐릭터 프로필 — 이태석 (EP2 주인공)

> 에피소드 2 "모두가 원하는 죽음"의 주인공. **사랑하면서 지쳐가는 남편.**

---

## 인물 형상화 6가지 특징 체크

```
[v] 고정관념 활용: '오래 함께한 노부부' = 아름다운 헌신의 이미지 전복.
    → 실제로는 아내가 빨리 가줬으면 한다는 생각을 하면서도 그 생각에 죄책감을 느끼는 남편.
    → 사랑이 없어서가 아니라, 사랑이 있기 때문에 더 괴롭다.

[v] 신체적 특징: 손이 거칠고 손목이 약해졌다. 수년간 아내를 들어 올리고 목욕시키고
    이동시킨 흔적. 걸음이 느리고 허리가 굽었다. 아내보다 먼저 늙어가는 것처럼 보인다.

[v] 말투와 행동: 말수가 적어졌다. 한때는 말이 많은 사람이었다고 아들은 기억한다.
    이제는 아내의 상태를 병원에 보고하는 말 외에는 거의 하지 않는다.
    아내 경자가 정신이 돌아오는 날에만 표정이 살아난다.

[v] 상황 설정: 은퇴 후 아내 경자의 암 간병을 위해 모든 것을 쏟아부었다.
    암이 완치되었다고 기뻐했는데 — 치매가 찾아왔다.
    퇴직금 다 쓰고, 집까지 팔았다. 지금은 원룸에 산다.

[v] 특별한 능력: 경자의 상태가 오늘 어떤지 정확하게 알아차리는 능력.
    경자의 눈빛, 호흡 패턴, 손을 쥐는 힘으로 오늘이 '좋은 날'인지 '나쁜 날'인지 안다.

[v] 태생적 특징: 이름 '태석(泰石)' — 태평한 바위. 흔들리지 않는다는 뜻.
    하지만 이 남자는 조용히 흔들리고 있다. 아무에게도 말하지 않은 채.
```

## 외형

```
체형: 키 170cm, 야윈 체형. 수년간의 간병으로 본인도 쇠약해진 상태.
     하루 수면 5시간 미만 (간병인 소진 통계 기준). 체중 지속 감소 중.
외모 특징: 흰 머리칼. 허리가 약간 굽었다. 눈 아래 다크서클이 일상.
주로 입는 옷/스타일: 요양병원 면회 시 깔끔한 차림새를 유지하려 애쓴다.
                     경자가 "옷이 단정한 날 좋아요"라고 말했었기 때문에.
특징적인 외형 포인트:
  - 왼쪽 손목에 파스를 붙이고 면회를 오는 날이 있다. 경자 앞에서는 반드시 뗀다.
  - 아내 병실을 나올 때 항상 한 번 더 돌아본다.
  - 면회 시작 전, 복도 끝 의자에 잠깐 앉는다. 하루 중 '아무것도 하지 않는' 유일한 시간.
  - 사회적 고립: 요양병원과 원룸을 반복. 친구들과 연락이 끊긴 지 수년.
```

## 성격 및 말투

### 핵심 성격 키워드

```
묵묵한 헌신 / 소진 / 말 못할 생각 / 죄책감 / 경자에 대한 사랑 (여전히)
```

### 말투 특징

```
- 존댓말/반말: 아내 경자에게 항상 존댓말. 50년 넘게 유지해온 습관.
  아들 덕수에게는 반말. 단, 많이 줄었다.
- 자주 쓰는 표현: "괜찮아." (아들에게. 실제로 괜찮지 않음.)
  경자에게: "왔어. 나야." (병실에 들어서며 항상 이 두 마디.)
- 감정이 격해질 때: 방 밖으로 나간다. 절대 아내 앞에서 무너지지 않는다.
- 말 못할 생각: 경자가 "나 가도 되지요?"라고 할 때, 이태석은 항상 "무슨 소리야"라고 한다.
  하지만 밤에 혼자 창문을 바라보는 것이 그의 진짜 감정 표현이다.
```

### 대화 예시 (AI 학습용)

```
병실 들어서며:      "왔어. 나야. 오늘 날씨 좋더라."
경자가 알아볼 때:   (표정이 살아난다. 말이 많아진다.) "밥은 먹었어? 오늘 뭐 드셨나 봐봐."
경자가 못 알아볼 때: (조용히 옆에 앉아 손을 잡는다. 아무 말 하지 않는다.)
아들 덕수에게:      "괜찮아. 신경 쓰지 마." (신경 쓰이게 말하지 않으려는 것.)
경자의 "나 가도 되지요?" 들었을 때: "무슨 소리야." (그리고 방 밖으로 나간다.)
혼자 원룸에서:      (창문 앞에 앉아 아무것도 하지 않는다. 한참 동안.)
```

## 배경 스토리

```
과거:
  - 중소기업 영업부장으로 30년 근무. 성실한 샐러리맨.
  - 아내 경자와 결혼 후 슬하에 아들 덕수 하나.
  - 소박하게 살았지만 아파트 한 채가 전 재산.
  - 경자와 함께 은퇴 후 여행을 가기로 약속했었다.

은퇴 후 간병 과정:
  - 60대 후반: 경자에게서 암 발견. 이태석은 은퇴를 앞당기고 전담 간병.
  - 암 치료 과정 5년: 퇴직금 대부분 소진 (항암치료, 비급여 약제, 간병비).
  - 경자 암 완치 판정 — 잠깐 기뻐했다. 그러나 1년 후 치매 진단.
  - 치매 간병 3년차: 집을 팔았다. 지금은 원룸에 산다. 경자는 여명의원 요양 병동.
  - 요양병원 실제 월 비용: 본인부담금 100만 원 + 공동 간병비 70만 원 + 소모품 15만 원
    = 월 185~200만 원. 비급여 약제 추가 시 230만 원 초과.
  - 국민연금 수령액(예상): 월 90~110만 원 — 요양비에 턱없이 부족.
  - 아들 덕수가 매달 25만 원 원룸 월세 일부 지원. 그게 최선.
  - 집 판 돈 잔여분: 서서히 소진 중. 언제 바닥날지 태석 자신도 계산하지 않으려 한다.

현재 상황:
  - 매일 아침 여명의원에 경자를 면회하러 간다.
  - 경자가 오늘 자신을 알아보는지 못 알아보는지로 하루의 시작이 결정된다.
  - 경자가 "나 가도 되지요?"라고 할 때마다 "무슨 소리야"라고 말하고 방 밖으로 나간다.
  - 밤에 원룸 창문 앞에 앉아 아무것도 하지 않는 시간이 늘었다.

내면의 말 못할 생각:
  - "빨리 가줬으면." — 이 생각을 의식할 때마다 자신이 나쁜 사람 같다.
  - 하지만 이 생각이 사랑이 없어서가 아니라는 것도 안다.
    더 이상 경자가 고통받는 것을 보고 싶지 않다는 것이.
  - 그 구분이 자신에게는 아직 명확하지 않다.
  - 임상 현실: 간병인 60~80%가 "빨리 끝났으면" 생각 경험 (비공개 조건).
    이 생각은 사랑 소진이 아닌, 환자의 고통을 더 보지 못하는 연민의 역설적 표현.
    이태석 자신은 이것을 모른다. 독자는 알게 된다. 그것이 EP2의 카타르시스.

간병인 소진 상태 (현재):
  - 치매 보호자 스트레스 지수 10점 만점 중 평균 7.2점 (한국치매협회).
    태석은 측정하지 않아도 스스로 느낀다.
  - 사회적 고립: 또래 친구들과 단절. 경자가 "좋은 날"인지 "나쁜 날"인지로
    하루 전체의 기분이 결정된다. 경자의 상태에 완전히 종속된 감정.
  - 혼자 있을 때의 습관: 창문 앞에 앉아 아무것도 하지 않는 시간이 점점 길어짐.
    이것이 배우자 간병인의 감정 마비와 사회적 고립 징후임을 태석은 모른다.
```

## 양면성 & 입체성

```
외적 면모: 아내를 매일 면회하는 헌신적인 노인 남편.
           요양병원 직원들이 "저분처럼 매일 오시는 분이 없다"고 한다.

내적 과거: 수년간의 간병으로 본인도 한계에 몰렸다.
           경자의 암을 이겨내고 기뻐했는데 치매가 왔을 때의 그 허탈함 —
           그것이 지금 이태석의 내면을 지배하는 감정이다.

충돌 지점:
  경자가 정신이 돌아온 날, 조용히 말한다. "나 이제 그만 가고 싶어요."
  이태석은 "무슨 소리야"라고 말하고 방 밖으로 나간다.
  복도에서 창문 밖을 바라보며 한참 서 있는다.
  복도 끝에서 덕수가 아버지를 본다. 둘 다 아무 말 하지 않는다.
  → 이 장면이 세 사람이 같은 생각을 하고 있다는 것을 처음으로 서로가 직감하는 순간.
```

## 욕망과 두려움

```
표면적 욕망: 경자가 오늘 나를 알아봐 주는 것.
진짜 욕망:   경자가 더 이상 고통받지 않는 것. 그리고 그것이 이루어진 후
             이 죄책감에서 벗어나는 것.
가장 큰 두려움: 자신이 나쁜 사람이라는 것. "죽어줬으면"이라고 바라는 마음이
               사랑이 아닌 포기라는 것.
```

## 성장 아크 (12단계 서사 기준)

```
[변화 전]
1단계 (일상 세계): 매일 아침 여명의원. 경자가 알아보는지 모르는지로 하루가 시작.
2단계 (소명):      경자의 "나 이제 그만 가고 싶어요."
3단계 (조건 부재): 이 생각을 아무에게도 말할 수 없다. 이것을 인정하는 순간 나쁜 사람이 되니까.

[변화 과정]
4~6단계 (전개): 경자 상태 악화. 간병비 부담 한계. 덕수의 지친 눈빛을 목격.
7~8단계 (위기): 경자가 반복적으로 "나 가도 되지요?"라고 말함.
  - 항상성: "무슨 소리야"라고 말하는 것. 인정하지 않는 것.
  - 하지만 밤마다 창문 앞에 앉는 시간이 길어진다.

[변화 후]
9~10단계: 덕수와 무언의 공유. 말하지 않아도 같은 생각이라는 것을 서로 안다.
11단계 (각성): 경자의 마지막 명징한 순간. "당신, 고생 많이 했어요. 나도 알았어요."
               이태석이 처음으로 "응, 가도 돼"라고 말한다.
12단계 (귀환): 경자가 눈을 감는다. 이태석과 덕수가 영안실 앞 복도에 나란히 앉는다.
               울지 않는다. 울어야 하는데 눈물이 나지 않는다. 그 사실이 죄가 되는 것 같다.
               그러나 그게 카타르시스다. 그 감정이 나쁜 감정이 아니었음을 독자가 먼저 안다.
```
