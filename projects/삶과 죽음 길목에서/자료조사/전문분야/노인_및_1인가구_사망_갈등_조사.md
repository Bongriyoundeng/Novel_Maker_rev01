# 전문분야 연구 노트 — 고령층·1인 가구 사망 실태 및 가족 갈등

> **용도:** 삶과 죽음 길목에서 — 임종 심리, 고독사, 상속·간병 갈등 장면 집필
> **출처:** 한국 보건의료 통계, 고독사 실태 자료, 법원 소송 데이터 기반

---

## 조사 주제

```
주제:        대한민국 고령층 및 1인 가구의 사망 실태와 가족 갈등
소설 내 활용 목적: 임종 전 심리 묘사, 고독사 현장, 가족 갈등(상속·간병·연명치료) 장면의
             현실감 있는 묘사 및 서사적 갈등 설계
관련 화수:   전 화수 (핵심 배경 자료)
```

## 핵심 규칙 / 사실

> 소설에서 틀리면 독자가 즉시 알아채는 정보

```
1. 고독사 사망자(2024년 3,924명) 중 남성 81.7% — 특히 50~60대 남성이 다수
   (실직·이혼으로 관계망이 무너진 상태)

2. 1인 가구는 다인 가구보다 전체 사망 위험 25%, 65세 이전 조기 사망 위험 35% 높음

3. 임종 전 '가족에게 짐이 되는 것'을 죽음 자체보다 더 두려워한다 (85%)
   → 연명의료 중단 찬성률 88% (본인 기준)
   → 그러나 가족 64%는 반대 (도덕적 비난·죄책감 우려)

4. 유류분 반환 청구 소송 6.5배 폭증 — 자산가만의 문제가 아님
   (생전 증여 불평등, 부양 기여도 인정 범위가 핵심 쟁점)

5. 80세 이상 사망자의 54%가 집중 — 폐렴(3위)·알츠하이머(4위) 비중 압도적
   여성 알츠하이머 사망률 남성의 2.1배
```

## 일반적 오해 vs 실제

> 독자가 흔히 알고 있는 것과 실제가 다른 경우 — 이 차이를 서사에 활용

| 일반적 오해 | 실제 | 서사 활용 방법 |
|-----------|------|-------------|
| 고독사는 가족이 없는 사람만 겪는다 | 가족이 있어도 시신 인수 거부 급증 (장례비 부담 + 단절된 관계) | 가족 있음에도 방치된 죽음 → 죄책감 갈등 |
| 임종 앞에서는 가족이 화해한다 | 연명치료 결정 과정에서 오히려 형제 분열 | "왜 포기하냐"vs"이게 아버지 뜻이야" 충돌 |
| 상속 분쟁은 큰 재산이 있을 때만 | 소액 자산에서도 유류분 소송 6.5배 증가 | 집 한 채를 두고 자녀 간 법정 싸움 |
| 간병하는 자식이 당연히 더 받는다 | 법적 기여도 인정 범위 좁음 → 독박 간병 자녀의 억울함 | "나만 10년 간병했는데 똑같이 나누라고?" |

```
[이 전문분야가 소설의 세계관 규칙에 영향을 주는 부분]
- 현대 한국 사회 배경 → 모든 제도·법률·통계가 현실 그대로 적용
- 사망 원인·절차·분쟁 유형이 독자에게 '이건 실제 있을 법한 일'이라는 신뢰 기반

[세계관이 이 전문분야를 어떻게 변형하는가]
- 판타지 요소 없음. 현실의 냉혹함 자체가 드라마틱한 장치
- 법·제도의 허점(유류분, 시신 인수 거부 등)이 악역 없는 갈등을 만드는 핵심
```

## Show Don't Tell 묘사 소재

```
[전문가만이 보이는 디테일]
→ 장례지도사: 유가족의 눈물보다 먼저 냉장고 온도와 안치 예약 가능 여부를 확인한다
→ 요양보호사: 노인의 이름이 아니라 호실 번호와 투약 시간으로 사람을 기억한다
→ 상속 전문 변호사: 위로 전에 제일 먼저 사망일자와 유언장 존재 여부를 확인한다

[전문가의 신체 반응·습관]
→ 고독사 현장 특수청소 업자: 현장 들어가기 전 아무 말 없이 마스크 끈을 두 번 조인다
→ 오래 간병한 자녀: 병원 복도를 걸을 때 자동으로 발소리를 줄인다

[비전문가와 전문가의 차이를 보여주는 행동]
→ 처음 임종을 맞은 가족: 숨이 멎은 후에도 계속 이름을 부른다
→ 숙련된 간호사: 마지막 숨이 끊기기 직전 조용히 커튼을 친다
```

## 갈등 설계 포인트 (서사 활용)

```
[A. 연명치료 결정 씬]
상황: 아버지가 연명치료 중단 의사를 밝혔는데 큰아들이 반대
갈등: "포기하는 거야" vs "이게 아버지 뜻이야"
감정 핵심: 죄책감 vs 체면 vs 실제 사랑의 충돌

[B. 독박 간병 폭발 씬]
상황: 10년 간병한 딸, 사망 후 형제와 상속 균등 분배 통보받음
갈등: 법적으론 균등하지만 감정적으론 불가능한 공정
감정 핵심: "내 청춘을 돈으로 환산하면 얼마냐"

[C. 고독사 발견 씬]
상황: 이웃 신고로 경찰이 먼저 진입, 가족은 나중에 연락받음
갈등: 경찰·집주인·가족의 이해관계 충돌, 시신 인수 거부 선택 직전
감정 핵심: 죄책감과 도망치고 싶은 욕망의 공존

[D. 얌체 상속 씬]
상황: 생전 연락 끊었던 자녀가 사망 소식에 특별연고자로 재산 청구
갈등: 평생 모신 사람 vs 법적 권리를 가진 사람
감정 핵심: 법과 도리의 괴리
```
