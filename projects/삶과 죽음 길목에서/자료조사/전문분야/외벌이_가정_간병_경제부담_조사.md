# 외벌이 가정 간병 경제 부담 조사

> EP2 "모두가 원하는 죽음" — 이덕수 캐릭터 (45세, 중소기업 사무직 + 주말 대리운전)의 경제적 현실 묘사 근거 자료

---

## 1. 외벌이 가정의 경제 현실 (2023~2024년 기준)

### 중소기업 사무직 40대 남성 평균 임금
- 제조업·서비스업 중소기업 사무직 (경력 15~20년, 대리~과장급): 월 **280~350만 원** (세전).
- 세후 실수령: 월 **240~310만 원** 수준.
- 외벌이로 4인 가족 부양 + 부모 지원 시: 생활비 외 여유분 매우 제한적.

### 4인 가족 월 생활비 (서울·수도권 기준)
| 항목 | 금액 |
|------|------|
| 주거비 (전세 대출 이자 또는 월세) | 60~80만 원 |
| 식비·생활용품 | 60~80만 원 |
| 교육비 (초등학생 1명) | 30~50만 원 |
| 교통비·통신비 | 20~30만 원 |
| 보험료 | 15~25만 원 |
| 기타 잡비 | 20만 원 |
| **합계** | **약 200~280만 원** |

- 세후 300만 원 수령 시: 생활비 후 잔여 **20~100만 원** 수준.
- 부모 지원 (월세 일부 + 용돈): 추가 20~50만 원 → **적자이거나 극히 빠듯**.

## 2. 요양병원 비용과 가족 부담

### 이태석(부)의 요양병원 비용 (월 기준)
| 항목 | 금액 |
|------|------|
| 요양병원 본인부담금 | 80~120만 원 |
| 간병비 (공동 간병 기준) | 60~90만 원 |
| 기저귀·소모품 | 10~20만 원 |
| 기타 비급여 | 10~30만 원 |
| **합계** | **약 160~260만 원** |

### 이태석의 수입 상황
- 전직 영업부장, 은퇴 후: 퇴직금 대부분 암 간병에 소진.
- 집 매각 후 원룸 거주: 보증금 + 월세 60~70만 원.
- 국민연금 수령 (예상): 월 80~120만 원 (가입 기간·소득에 따라 다름).
- **결론**: 국민연금 수령액으로 요양병원비 충당 불가. 자녀 지원 필수.

### 이덕수의 실제 부담 계산
- 이덕수가 아버지에게 지원하는 금액 (원룸 월세 일부): 월 20~30만 원.
- 실질적으로 가능한 지원 한도: 자신의 생활비를 줄이거나 투잡 수입으로 충당.

## 3. 대리운전 투잡 현실

### 주말 대리운전 실태
- 주말 대리운전 시간대: 금·토요일 저녁 9시~새벽 3시.
- 건당 수입: 기본 콜 1.5만~2.5만 원. 장거리는 더 많음.
- 주말 1박 기준 (8~10시간 근무): 8~15만 원 수입.
- 월 주말 대리운전 수입 (주 2회 기준): **약 40~60만 원**.

### 투잡의 물리적 한계
- 주말: 토·일요일 → 대리운전 + 부모님 면회.
- 금요일 저녁: 대리운전.
- 평일: 회사 근무 8~9시간.
- **수면 시간**: 주말 귀가 후 새벽 3~4시, 면회를 위해 오전 일찍 기상.
- 이덕수의 실질 주간 노동 시간: **70~80시간 이상**.

### 대리운전 기사의 현실
- 사회적 시선: 중소기업 직원이 대리운전한다는 것을 아내 외에 알리지 않는 경우 많음.
- 후드 티 착용: 얼굴 노출 최소화. 아는 사람과 마주칠까 봐.
- 심야 대리운전 시 조우하는 풍경: 술 취한 승객, 심야 도심, 새벽의 적막.
- 이 경험이 이덕수의 내면 독백 배경이 될 수 있음.

## 4. 외벌이 가정 간병 부담 통계

### 성인 자녀의 부모 부양 부담

| 구분 | 수치 |
|------|------|
| 부모 부양 때문에 여가·취미 포기한 경험 | 40대의 약 45% |
| 부모 부양 비용으로 자녀 교육비 감소 경험 | 약 30% |
| 부모 부양으로 직장 내 성과 저하 경험 | 약 25% |
| 부양 부담으로 배우자와 갈등 경험 | 약 35% |

### 부모 부양과 자녀 양육의 이중 부담 ('샌드위치 세대')
- 40~50대: 부모 부양과 자녀 양육을 동시에 책임지는 '샌드위치 세대'.
- 특히 외벌이 가정: 경제적 압박이 극심.
- 배우자의 이해와 지지가 있을 경우에도 **가정 내 긴장감** 증가.

## 5. 이덕수 가정의 내부 긴장

### 아내의 입장
- 아내는 이덕수의 상황을 알고 이해하려 하지만:
  - 주말마다 남편이 없음 (면회 + 대리운전).
  - 집안 수입이 항상 빠듯함.
  - 초등학생 자녀 혼자 돌봄.
  - "언제까지 이렇게 살아야 하나" 불안감.
- 아내의 침묵: 직접 불만을 표현하지 않지만 이덕수가 느끼는 집안 공기.

### 이덕수의 죄책감 이중 구조
- 부모에 대한 죄책감: "더 자주 못 가드려서."
- 아내·자녀에 대한 죄책감: "가정에 더 신경 못 써서."
- 두 방향의 죄책감이 동시에 압박.
- 어느 쪽을 선택해도 다른 쪽에 미안한 상황 → 만성 피로와 무력감.

## 6. 수입과 지출 시뮬레이션 (이덕수 케이스)

### 월 수입
| 항목 | 금액 |
|------|------|
| 중소기업 사무직 세후 | 280만 원 |
| 대리운전 (월 기준) | 50만 원 |
| **합계** | **330만 원** |

### 월 지출
| 항목 | 금액 |
|------|------|
| 4인 가족 생활비 | 230만 원 |
| 아버지 원룸 월세 지원 | 25만 원 |
| 예비비·비상금 | 20만 원 |
| **합계** | **275만 원** |

- **잔여**: 약 55만 원. 저축 불가 수준.
- 갑작스러운 지출(자녀 병원비, 차 수리 등) 발생 시: 즉시 적자.
- 아버지 비용이 더 늘어나면 (요양비 상승, 추가 약제비 등): 대리운전 횟수 늘리거나 대출.

## 7. 집필 활용 노트

### 이덕수의 경제적 현실을 보여주는 장면 제안
- 편의점에서 계산할 때 잔액 확인하는 장면.
- 대리운전 중 심야 도로 → 이때 어머니 생각이 나는 내면 독백.
- 병실 앞에서 숨을 고를 때의 내면: 피로 + 죄책감 + 사랑이 뒤섞인 감정.
- 아내와 주방에서 나누는 짧은 대화: 말하지 않아도 아내가 아는 것들.

### 투잡의 의미
- 이덕수에게 대리운전은 단순 수입원이 아님.
- 심야에 혼자 운전하는 시간 = 감정 처리의 공간.
- 아무도 없는 새벽 도로에서 이덕수가 처음으로 "어머니"를 소리 내어 부르는 장면 가능.

### 아내와의 관계
- 아내는 악인이 아님. EP2에 악인 없음.
- 아내도 이해하려 하지만 한계가 있음.
- 두 사람의 팽팽함: 직접 싸우지 않지만 서로가 서로에게 뭔가를 참고 있다는 것을 앎.
- 경자 임종 후 이덕수가 집에 돌아왔을 때 아내가 아무 말 없이 밥을 차려주는 장면 가능.
