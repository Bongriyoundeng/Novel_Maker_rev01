|------|------|

| 나이 | 14세 (중학교 2학년) |

## 능력 상세

```
[0.1초 공중 부양]

발동 조건: 점프의 최고점에서 의식적으로 "지금 멈춰"라고 생각하면 발동.
           초반엔 발동이 불안정해서 0.05초밖에 안 됨.

감각 묘사:
  - 중력이 잠깐 사라진 느낌. 공중에서 "어? 아직 안 내려가네?" 수준.
  - 발동 시 본인은 엄청나게 멋있다고 생각함.
  - 실제로 겉으로 보면 "점프력 좋은 애" 수준.

실전 활용:
  - 블로킹: 상대가 슛 릴리즈하는 순간까지 공중에서 기다렸다가 블록
  - 레이업: 수비 손이 내려가는 0.1초를 기다렸다가 릴리즈
  - 점프볼: 상대보다 약간 오래 공중에 있음 → 타이밍 우위

실수 에피소드:
  - 4화: "멋있게 하려다가" 0.1초를 너무 의식한 나머지 레이업 미스
  - 17화: 피로 상태에서 발동 → 착지 비틀거림 → 손예찬: "괜찮아?"
```

---

## 외형

```
체형: 171cm (다섯 명 중 가장 키 큼). 팔이 길고 점프력이 원래부터 좋은 편.
외모 특징: 항상 뭔가 하려는 표정. 눈이 좀 큼. 웃을 때 눈이 가늘어짐.
           운동하면 얼굴이 금방 빨개짐.
주로 입는 옷: 형광색 계열 좋아함. "잘 보여야 한다"는 철학.
특징적 포인트: 말을 엄청 많이 함. 침묵을 못 견딤.
```

## 성격 및 말투

### 핵심 성격 키워드
```
시끄러움 / 낙천적 / 인정욕구 / 알고 보면 팀 걱정 제일 많이 함 / 무한 긍정
```

### 말투 특징
```
- 말 많고 빠름. 혼자 대화를 이어가는 경우도 있음.
- 자주 쓰는 표현: "진짜", "아니 근데", "야야야", "솔직히", "멋있지 않냐"
- 흥분하면 말이 더 빨라짐. 가끔 본인이 무슨 말 하는지 잊어버림.
- 분위기 읽을 줄 앎. 진짜 팀이 힘들 땐 혼자 조용해짐.
```

### 대화 예시
```
평소:
  강준서: "야야야 나 아까 뜬 거 봤어? 0.1초 뜬 거 맞지?"
  차윤호: "응."
  강준서: "응? 진짜? 야 나 미쳤다 진짜로."

농구 잘 됐을 때:
  "야야야 이거 우리 진짜 되는 거 아니야? 솔직히 됐잖아."

농구 안 됐을 때:
  "아 씨 아니 근데 내가 아까 뜨긴 했어. 그 타이밍이 문제였지."

신규현한테 (신규현이 말 없을 때):
  "야 규현아 너 살아있냐. 숨 쉬고 있어?"

이준서 슛 들어갔을 때:
  "야 역시 준서 슛이 좋다니까." (알고 있지만 모른 척)

팀 진짜 힘들 때 (드물게 진지하게):
  "...야. 우리 할 수 있잖아. 아까도 됐잖아."
```

## 배경 스토리

```
과거:
  초등학교 때 육상부였음. 점프력이 타고났다는 소리를 들어왔음.
  농구는 차윤호 따라서 시작. 근데 생각보다 재미있어서 계속 함.

현재 상황:
  공부는 평균 이하. 학원 다니긴 하는데 집중을 못 함.
  부모님이 "농구 그만하고 공부하라"는데 겉으론 "알겠습니다"하고 안 함.

능력 발현 후 반응:
  다섯 명 중 가장 흥분함. "야야야 나 유성 맞은 거 아니야? 내가 선택받은 거잖아!"
  나머지 넷: "..."
  근데 정작 능력 쓰는 건 제일 서툼.
```

## 농구 스타일

```
포지션: SF (스몰 포워드) — 높이와 기동력 담당
주특기: 레이업 (0.1초 부양으로 수비 타이밍 흐트러뜨림), 블로킹
약점: 슛이 엄청 불안정함. 원래 외곽 슛이 잘 안 됨.
     패스 판단이 느림 (차윤호가 미니맵으로 커버해줌).

능력 활용 예:
- 레이업 코스에서 수비가 따라붙었을 때 공중에서 0.1초 더 기다리다 릴리즈
- 블로킹 시도 시 타이밍을 0.1초 늦춰 상대 손이 내려간 후 블록 성공
```

## 성장 아크

```
1단계 (도입):
  "나 0.1초 뜬다! 야 나 최고 아니야?" — 아직 기본기가 없다는 걸 모름.

3단계 (위기):
  진짜 잘하는 팀을 만나서 0.1초 부양이 아무 의미 없는 상황을 경험.
  처음으로 혼자 연습 시작. 차윤호한테는 "그냥 심심해서"라고 함.

5단계 (결말):
  "0.1초가 의미 있으려면 그 전에 내가 거기 있어야 했어."
  처음으로 경기 끝나고 조용히 앉아서 쉬는 장면 (말없이 웃음).
```
