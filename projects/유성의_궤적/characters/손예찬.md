# 캐릭터 프로필: 손예찬

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 이름 | 손예찬 |
| 나이 | 14세 (중학교 2학년) |
| 성별 | 남 |
| 직업/직위 | 중학생 / 팀 볼 핸들러 |
| 소속 | 유성단 |
| 능력 | 지면 마찰력 자유 조절 (Ultimate Traction) |
| 등장 회차 | 1화 |

---

## 능력 상세

```
[지면 마찰력 자유 조절 — Ultimate Traction]

발동 조건: 의식적으로 발바닥에 집중하면 발동.
           발동 레벨: 2단계
           - MAX (마찰력 2배): "그립 MAX" 상태. 급정지·폭발적 첫 발.
           - ZERO (마찰력 0): "슬라이드" 상태. 미끄러지듯 이동 가능.
           기본 상태: 일반 마찰력 (능력 OFF).

감각 묘사:
  - MAX: 발이 코트에 떡 붙는 느낌. 급정지 시 몸이 한 번에 멈춤.
  - ZERO: 얼음판 위에 선 느낌. 발이 마음대로 미끄러짐.
  - 본인 표현: "MAX는 달라붙는 느낌, ZERO는 매끈한 느낌."

실전 활용:
  - MAX: 드리블 중 갑자기 멈추기 → 수비수가 혼자 앞으로 나가버림
         급출발 (정지 상태에서 폭발적인 첫 발)
  - ZERO: 슬라이딩으로 빠른 이동 (수비 시 포지셔닝), 혹은 수비 피하기

자주 일어나는 사고:
  - ZERO 조절 실패 시 → 본인이 미끄러짐
  - 이게 시즌 내내 반복됨. 팀원들은 이제 "예찬이 또 미끄러진다"가 일상.
  - 15화: 결정적 장면에서 ZERO 남용 → 넘어짐 → 팀 분위기 최악

한계:
  - MAX 상태에서 점프 시 발이 너무 붙어있어서 오히려 점프력이 줄어듦.
  - 코트 상태(물기, 먼지)에 따라 효과 편차가 있음.
  - 마찰력 MAX 1분 초과 시 발바닥이 화끈거림.
```

## 외형

```
체형: 163cm (다섯 명 중 키가 가장 작음). 근데 발이 빠름.
외모 특징: 항상 뭔가 꾸미려는 흔적이 있음. 헤어에 신경 씀.
           뛸 때 발걸음이 특이하게 빠름. "발이 빠른 애"로 통함.
           넘어질 때 리액션이 크고 시끄러움.
주로 입는 옷: 좀 더 힙한 스타일 지향. 학교 체육복도 소매를 걷음.
특징적 포인트: 말을 빠르게 함. 농담을 많이 함. 분위기 띄우는 역할.
              자주 미끄러지는 것 때문에 "예찬아 괜찮아?"가 팀 내 유행어.
```

## 성격 및 말투

### 핵심 성격 키워드
```
장난끼 / 쾌활 / 분위기 메이커 / 실수를 웃음으로 넘김 / 속은 겁쟁이
```

### 말투 특징
```
- 빠르고 유쾌함. 농담을 습관처럼 함.
- 자주 쓰는 표현: "아씨", "야 잠깐만", "이게 아닌데", "일단 해보자"
- 넘어질 때마다 나오는 말: "아씨!" (반사적으로)
- 진지한 상황을 농담으로 흐트러뜨리는 경향 → 가끔 분위기 망침.
- 근데 정작 본인이 팀에서 제일 겁쟁이. 중요한 순간에 손이 떨림.
```

### 대화 예시
```
평소:
  "야 나 오늘 진짜 잘할 것 같아. 느낌이 와."
  강준서: "매날 그러잖아."
  손예찬: "아니 진짜로 이번엔 진짜야."

능력 성공 시:
  "야 봤어? 내가 딱 멈췄잖아. 그 수비 혼자 앞으로 가버렸잖아."

능력 실패 (미끄러짐) 시:
  "아씨!" (바닥에서)
  차윤호: "...나는 못 봤다."
  강준서: "예찬아 괜찮아?"

중요한 경기 전:
  "야 나 배 아파." (진지하게)
  이준서: "먹은 거 있어?"
  손예찬: "아니 긴장돼서."

농담 시:
  신규현 (조용히 있을 때): "규현아 너 또 잔상 보는 거야?"
  신규현: "..."
  손예찬: "나도 잔상 봐주라. 수학 시험 답지 잔상."

실수 후:
  "야 근데 아까는 진짜 잘될 뻔했잖아. 타이밍이 살짝..."
  차윤호: "됐어."
```

## 배경 스토리

```
과거:
  달리기가 빠른 편이었음. 다섯 명 중에 체육 성적이 제일 좋음.
  근데 농구는 드리블은 좀 되는데 나머지가 다 기본 수준.

현재 상황:
  공부는 중간 수준. 딱히 문제없음.
  부모님이 "뭘 그렇게 뛰어다니냐"고 하지만 그냥 넘어감.
  집에서 혼자 마찰력 조절 연습을 하다가 방바닥에서 미끄러져서 부모님한테 들킴.
  → "왜 방에서 스케이트 타냐"는 질문을 받음. "운동이요."

유성 사건 당일:
  넘어지면서 가장 크게 소리 질렀음 (다섯 명 증언 일치).
  분화구 앞에서 "야 우리 여기 사진 찍자"고 함 → 차윤호: "가자."

능력 발현 후 반응:
  처음에 "야 나 이거 미끄러지는 거야 아니면 안 미끄러지는 거야?"
  이후 한동안 ZERO 연습하다가 복도에서 미끄러짐 → 선생님한테 불려감.
```


## 농구 스타일

```
포지션: PG (포인트 가드) — 볼 핸들링 & 돌파 담당
주특기: 드리블 돌파, 급정지 페이크 (마찰력 MAX 활용)
약점: 슛이 약함. 공중에서 하는 플레이가 약함 (키도 작고 점프도 평범).
     ZERO 조작이 불안정 → 자주 미끄러짐.

능력 활용 예:
- 드리블 중 급정지 (마찰력 MAX) → 수비수가 관성으로 앞으로 나감 → 돌파
- 수비 포지셔닝 시 슬라이딩 (마찰력 ZERO) → 빠른 이동
- 단, ZERO 실패 시 본인이 미끄러짐 → "아씨!"
```

## 성장 아크

```
1단계 (도입):
  "ZERO가 재밌는데 자꾸 넘어짐." — 능력 컨트롤 불안정.

3단계 (위기):
  15화 미끄러짐 사고 후 → 자신감 바닥. MAX만 씀 (안전하게).

4단계 (절정):
  결승에서 처음으로 ZERO를 제대로 씀 → 폭발적인 돌파.
  손이 떨리지 않았던 것을 혼자 느낌.

5단계 (결말):
  "겁나는 건 여전한데 그게 사라지길 기다리다가는 아무것도 못 하더라."
  마지막에 또 미끄러짐 → "아씨!" → 모두 웃음.
```
