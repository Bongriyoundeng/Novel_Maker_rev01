# 캐릭터 프로필: 신규현

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 이름 | 신규현 |
| 나이 | 14세 (중학교 2학년) |
| 성별 | 남 |
| 직업/직위 | 중학생 / 팀 수비 담당 |
| 소속 | 유성단 |
| 능력 | 0.2초의 '예지 잔상' (Reaction Time) |
| 등장 회차 | 1화 |

---

## 능력 상세

```
[0.2초의 예지 잔상 — Reaction Time]

발동 조건: 상대방을 집중해서 보면 자동 발동. 의식적으로 끄지 않는 이상 켜짐.

감각 묘사:
  - 상대가 크로스오버를 하거나 슛을 쏘기 직전에,
    0.2초 뒤에 그 선수가 어디 있을지 아주 흐릿한 잔상이 보임.
  - 마치 포토샵 레이어가 하나 더 겹쳐 보이는 느낌.
  - 잔상은 반투명하고 희미함. 확신이 아닌 힌트.

실전 활용:
  - 수비: 크로스오버 페이크에 안 속음 → 진짜 방향으로 미리 발이 나감
  - 스틸: 패스 방향 예측 → 가로채기
  - 1:1 수비에서 거의 뚫리지 않음

실수 에피소드:
  - 9화: 잔상이 두 명 동시에 보이다가 혼란 → 아무것도 못 함
  - 25화: 극도로 긴장한 상태에서 잔상이 흔들려 반대 방향으로 발이 나감
```

---

## 외형

```
체형: 165cm, 마른 편. 체력이 의외로 좋음 (수비 전문이라 계속 뛰어야 함).
외모 특징: 눈을 항상 가늘게 뜨고 있음. 뭔가를 분석하는 표정.
           표정 변화가 적음. 웃을 때도 미소 수준.
           가끔 너무 집중해서 보다가 상대를 무섭게 만들기도 함.
주로 입는 옷: 무난한 색깔. 브랜드 따위 없음.
특징적 포인트: 말수가 가장 적음. 있는 듯 없는 듯 있다가 갑자기 핵심 한 마디.
```

---

## 성격 및 말투

### 핵심 성격 키워드
```
관찰형 / 과묵 / 신중 / 한 마디가 핵심임 / 감정 표현이 서툼
```

### 말투 특징
```
- 말이 매우 적음. 대화에서 말 한 마디 하면 그게 핵심임.
- 자주 쓰는 표현: "...", "응", "아니", "그거 아닐걸"
- 강준서가 말 많이 하면 그냥 놔둠. 대신 틀린 것만 짧게 교정.
- 감정 표현이 서툼. 기쁨도 슬픔도 잘 드러내지 않음.
- 가끔 엉뚱한 관찰을 꺼냄. "야 저 팀 5번이 왼발을 더 자주 씀."
```

### 대화 예시
```
평소:
  강준서: (10분째 혼자 얘기 중)
  신규현: "...그거 아닐걸."
  강준서: "뭐가?"
  신규현: "다."

관찰 중:
  "저 선수 오른쪽으로 가려고 할 때 왼쪽 어깨가 먼저 움직여."
  (아무도 안 물어봤는데 갑자기)

칭찬받았을 때:
  "...응."

팀이 힘들 때:
  "할 수 있어." (짧게, 근데 이 한 마디가 팀원들한테 제일 크게 들림)

강준서가 시끄러울 때:
  (아무 말 없이 강준서 머리를 가볍게 한 번 침)

능력 경험 직후:
  "...저거 또 보였다."
```

## 배경 스토리

```
과거:
  특별히 뭔가를 잘한다는 소리를 들어본 적이 없음.
  공부도 평균, 운동도 평균. 그냥 차윤호랑 어릴 때부터 친해서 같이 다님.

현재 상황:
  수비를 유독 잘 함. 왜인지는 본인도 몰랐는데, 알고 보면 상대를 관찰하는 게 습관이었음.
  능력 생긴 이후로 수비가 급격히 좋아짐.

유성 사건 당일:
  가장 먼저 분화구를 살펴봄. 혼자 한 바퀴 더 돌아보다가 "없네" 하고 옴.

능력 발현 후 반응:
  강준서처럼 호들갑 안 떪. 며칠 동안 혼자 관찰하다가 팀원들한테 조용히 말함.
  "야, 상대가 움직이기 전에 뭔가 보이는 것 같아."
  강준서: "뭐가?" / 신규현: "잔상 같은 거."
```

## 욕망과 두려움

```
표면적 욕망: 딱히 없음. 그냥 같이 하면 됨.
진짜 욕망: 팀원들한테 "야 규현이 수비 진짜 든든하다"는 말이 듣고 싶음.
           (이걸 본인도 잘 모름)
가장 큰 두려움: 아무것도 안 보이는 상황 (극도의 긴장 상태에서 잔상이 차단될 때).
               "내가 이것도 없으면 뭐하는 거지?"
```

## 농구 스타일

```
포지션: SG (슈팅 가드) — 수비 및 압박 전문
주특기: 1:1 수비 (잔상으로 방향 예측), 스틸
약점: 공격이 약함. 슛을 잘 못 쏨. 드리블도 기본 수준.
     오펜스 리바운드 참여를 거의 안 함 (포지션 잡는 것보다 수비 복귀를 우선).

능력 활용 예:
- 상대가 크로스오버 준비 중 잔상 발동 → 미리 발이 나감 → 스틸
- 패스 타이밍 예측 → 패스 라인에 미리 들어가서 가로채기
```

---

## 성장 아크

```
1단계 (도입):
  "뭔가 보이는 것 같은데 맞는지 모르겠음." — 능력을 믿지 못하는 상태.

3단계 (위기):
  긴장으로 잔상이 안 보이는 경험 → "나 지금 아무것도 안 보여."
  처음으로 차윤호한테 털어놓음.

5단계 (결말):
  "잔상이 안 보여도 상대를 보면 알게 됐어. 그게 몸으로 익혀진 것 같아."
  팀원들한테 처음으로 먼저 말 건넴: "야. 잘했어."
```
