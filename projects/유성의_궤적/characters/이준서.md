## 능력 상세

```
[림 자석 0.5도 보정]

발동 조건: 이준서 본인이 슛을 던진 공이 림 근처(반경 약 30cm)에 도달했을 때 자동 발동.
           본인의 의지와 무관하게 항상 켜져 있음 (ON/OFF 불가).

감각 묘사:
  - 본인은 아무것도 느끼지 못함.
  - 공이 "어떻게든 들어가는" 경험을 반복하면서 자신감이 올라가는 것뿐.
  - 실제로는 아슬아슬하게 림을 맞고 나올 공이 0.5도 보정되어 인으로 떨어짐.

실전 활용:
  - 림을 건드리는 슛은 대부분 들어감
  - 3점 성공률이 비정상적으로 높아 보임 (실제 폼은 별로임에도)
  - 아주 가까이 있으면 팀원 슛에도 미약하게 발동됨 (팀원은 모름)

핵심 문제:
  - 이준서 본인이 능력이 있다는 걸 모름 (아니, 알지만 슛이 원래 좋다고 생각함).
  - 실제 폼 완성도가 낮아서 능력 없으면 성공률이 30%도 안 됨.
  - 21화에서 이 진실이 팀원들에 의해 (어쩌다) 공개됨.

능력 한계:
  - 림을 완전히 빗나간 공은 보정 불가. "림이라도 맞아야" 발동.
  - 집중력 분산 시 보정 범위가 0.3도로 감소.
```

## 외형

```
체형: 168cm. 손이 큼. 슛을 쏠 때 자세가 나름 폼 잡은 척하는 게 보임.
외모 특징: 자신감 있는 표정. 항상 살짝 웃는 얼굴.
           슛 폼이 어딘가 이상한데 본인은 모름. 팀원들은 알고 있음.
주로 입는 옷: 조금 더 챙겨 입는 편. "슈터는 멋있어야 한다"는 철학.
특징적 포인트: 슛을 던지고 나서 항상 자세를 1초쯤 유지함. 폼 잡음.
```

## 성격 및 말투

### 핵심 성격 키워드
```
자신감 / 허당 / 인정욕구 / 알고 보면 열심히 함 / 충격 받으면 오히려 더 열심히 함
```

### 말투 특징
```
- 밝고 자신감 있음. 자랑을 많이 함.
- 자주 쓰는 표현: "나 슛 좋잖아", "내가 원래", "진짜로", "솔직히"
- 지적받으면 잠깐 멈추고 나서 "...알아, 근데" 하고 이어감.
- 충격적인 진실을 들었을 때: 잠깐 조용해지다가 → "...그럼 더 열심히 하면 되잖아."
- 팀원들한테 밥을 잘 삼. "야 떡볶이 내가 살게."
```

### 대화 예시
```
슛 들어갔을 때:
  "야 내 슛이 원래 좋잖아. 다들 봤지?"
  (손 포즈 1초 유지)

슛 안 들어갔을 때:
  "아 타이밍이 살짝... 근데 아깝지 않았냐. 거의 들어갈 뻔했잖아."

강준서한테:
  "야 준서야 나 슛 폼 어때 보여?"
  강준서: "좋은데?" (사실 좀 이상하지만 모른 척)

21화 진실 공개 후 (팀원들이 능력 얘기를 어쩌다 꺼내게 됨):
  "...뭐? 내가 슛이 원래 좋은 게 아니라고?"
  (잠깐 침묵)
  "...그럼 더 하면 되잖아."

연습 중:
  "야 나 5천 개 쏘면 슛 좋아지겠지?"
  차윤호: "10만 개."
  이준서: "...몇 개?"

팀이 이겼을 때:
  "야 떡볶이 내가 살게!" (항상)
```

## 배경 스토리

```
과거:
  농구를 좋아한 건 맞는데 딱히 잘하진 않았음.
  그런데 공원에서 슛을 쏘면 신기하게 잘 들어가서 "나 슛 좋은 것 같아"라고 생각해옴.
  (능력이 이미 발현되어 있었을 가능성이 있음 — 확인 불가)

현재 상황:
  공부가 팀 중에서 제일 아슬아슬함. 최저학력제 위기가 가장 현실적인 서브 플롯.
  수학은 진짜 안 됨. "이 문제 내 슛처럼 쏘면 안 들어가냐"는 발언을 함.

유성 사건 당일:
  제일 크게 놀라서 뒤로 넘어짐. 자기가 놀랐다는 걸 부끄러워함.
  분화구 앞에서 "야 여기 소원 빌면 되는 거 아니야?" 함 → 차윤호: "..."

능력 발현 후 반응:
  처음엔 자기 슛이 원래부터 좋았던 거라고 생각함.
  차윤호가 미니맵으로 공의 궤적이 이상하게 휜 걸 보고 뭔가 이상함을 느끼지만
  확신을 못 해서 말 못 함.
```

```

진짜 욕망: 능력이 아닌 진짜 슛 실력으로 들어가는 경험을 하고 싶다.
           (21화 이후 이게 진짜 욕망임을 자각함)
가장 큰 두려움: 능력 없으면 자기가 아무것도 아닌 거.
               (21화에서 이걸 직면하고 무너지다가 → 오히려 더 열심히 함)
```

## 농구 스타일

```
포지션: SF (스몰 포워드) — 슈터 롤
주특기: 중거리 슛 & 3점 (능력 덕분에 성공률 이상하게 높음)
약점: 드리블이 약함. 1:1 공격이 거의 없음.
     수비 포지셔닝이 느림 (공격에 집중하다보니).
     실제 슛 폼 완성도: 낮음 (21화 이전까지는 본인만 모름).

능력 활용 예:
- 아슬아슬하게 림을 건드린 공이 어떻게든 들어감
- "야 준서 슛이 왜 이렇게 잘 들어가냐" → "내가 원래 잘해."
```

## 성장 아크

```
1단계 (도입):
  "나 슛 좋아. 다들 봐." — 스스로 확신. 팀원들 모른 척.

2단계 (전개):
  최저학력제 위기. 국어 영어 수학 평균 40% 맞추기 도전. 서브플롯의 긴장감.

3단계 (위기/21화):
  어쩌다 진실 공개. "...그럼 더 하면 되잖아." → 혼자 연습 시작.

5단계 (결말):
  능력 없는 상황에서도 림에 맞는 슛을 던짐 → 들어감 (진짜 실력으로).
  아무 말 안 하고 혼자 씩 웃음.
```
