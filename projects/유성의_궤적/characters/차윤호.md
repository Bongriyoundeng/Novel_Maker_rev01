## 능력 상세

```
[미니맵 — Court Vision]

발동 조건: 의식적으로 "지금 다 어디 있지?"라고 생각하는 순간 자동 발동.
           처음엔 통제가 안 됐는데 5화쯤부터 어느 정도 켜고 끌 수 있게 됨.

감각 묘사:
  - 눈에 보이지 않는 등뒤·측면 선수들의 위치가 머릿속에 '점'으로 나타남.
  - 점은 색깔이 없음. 그냥 위치만 알려줌. 팀원/상대 구분은 직관으로 해야 함.
  - 마치 1인칭 게임에서 미니맵이 켜진 느낌.

실전 활용:
  - 노룩 패스: 뒤에 달려오는 강준서 위치 알고 패스
  - 상대 수비 위치 파악 후 드리블 방향 결정
  - 1:1 상황에서 도움 수비 오는 타이밍 예측

한계 & 실수 에피소드:
  - 15화: 미니맵 켰다가 점이 너무 많이 보여서 멍때림 → 공 뺏김
  - 23화: 긴장 상태에서 발동 안 됨 → 처음으로 "능력 없는 나"를 경험
```

---

## 외형

```
체형: 167cm, 마르지만 나름 탄탄함. 어깨가 또래보다 조금 넓음.
외모 특징: 눈빛이 차분한 편. 감정이 얼굴에 잘 안 드러남.
           운동하면 땀이 엄청 많이 남. 앞머리가 이마에 철벅 달라붙음.
주로 입는 옷: 학교 체육복 혹은 헐렁한 반팔 + 반바지.
              색깔 조합 개의치 않음 (어머니가 사준 거 그냥 입음).
특징적 포인트: 농구공을 항상 들고 다님. 손이 커서 한 손으로 공을 잡음.
```

## 성격 및 말투

### 핵심 성격 키워드
```
침착함 / 은근 승부욕 / 과묵한 척이지만 사실 할 말 많음 / 책임감
```

### 말투 특징
```
- 기본 반말. 낯선 사람에겐 짧은 존댓말.
- 자주 쓰는 표현: "해보면 되지", "알잖아", "그래서", "됐어"
- 말버릇: 말 끝을 흐림. "...뭐 어쩌라고" "...일단 해봐" 스타일.
- 감정 격해지면 오히려 더 조용해짐. 대신 행동으로 보여줌.
- 절대 안 씀: "대박!", "헐!" — 그냥 "오." 한 마디.
- 중학생이라 가끔 어설픈 말투 나옴. "야 이거 되는 거 맞지?" 수준.
```

### 대화 예시
```
평소:
  강준서: "야 나 진짜 0.1초 뜬다니까."
  차윤호: "응." (드리블하면서)

농구 상황:
  "강준서 왼쪽. 지금."

능력 발동 전 혼잣말:
  "...다 보인다."

당황했을 때:
  "...잠깐만." (멈추고 상황 정리 중)

화났을 때 (드물게):
  "그러니까 아까부터 그 얘기를 하고 있었잖아."

손예찬이 또 미끄러졌을 때:
  "...나는 못 봤다."

이준서가 "내 슛이 원래 좋아" 할 때:
  (아무 말 안 하고 신규현과 눈빛 교환)
```

## 배경 스토리

```
과거:
  초등학교 때부터 농구를 좋아했음. 근데 특별히 잘하진 않았음.
  "그냥 좋아서 하는" 수준이었는데, 그게 지금도 마찬가지임.

현재 상황:
  중학교 2학년. 공부는 평균. 특별히 잘하는 것도 없음.
  강준서·신규현·이준서·손예찬이랑 항상 붙어다님. 사실상 그게 전부.

유성 사건 당일:
  제일 덜 놀랐음. (아니 사실 제일 많이 놀랐는데 겉으로 안 냄)
  분화구 보고 제일 먼저 "일단 가자"고 함.

능력 발현 후 반응:
  처음에 "착각인 거 아니야?" 했다가, 계속 경험하면서 받아들임.
  "뭔지는 모르겠는데 일단 농구에 쓰면 되잖아."
```

## 욕망과 두려움

```
표면적 욕망: 농구를 잘하고 싶다. 이기고 싶다.
진짜 욕망: "우리 팀이 잘됐으면 좋겠다." (근데 이걸 말로 꺼내지 못함)
가장 큰 두려움: 본인이 아무것도 못해서 팀이 지는 상황.
               능력만 믿다가 결정적 순간에 아무것도 없는 것.
```

## 농구 스타일

```
포지션: PG (포인트 가드) — 패스와 경기 조율 담당
주특기: 노룩 패스, 수비 포지셔닝 (미니맵 덕분)
약점: 슛이 별로임. 특히 3점은 거의 안 들어감. (이준서한테 요청하는 이유)
     드리블 기술이 아직 기본 수준.

능력 활용 예:
- 등 뒤에서 달려오는 강준서 위치 잡고 노룩 패스 → 팀원들 "어 어떻게 알았어?"
- 상대 도움 수비 오기 전에 드리블 방향 바꿈 → "야 쟤 눈 여러 개야?"
```

## 성장 아크

```
1단계 (도입):
  "능력 생겼으니까 써보자." — 아직 한계를 모름.

3단계 (위기):
  능력이 안 켜지는 순간 처음으로 "나 지금 뭐하고 있지?" 생각함.

5단계 (결말):
  "보이는 것보다 내가 직접 움직이는 게 중요했어."
  처음으로 경기 끝나고 먼저 씩 웃음.
```
