# 마이크로 요약 — 1화

---

## 델타 타일 (다음 화 인수인계용)

```
참여 인물:
  차윤호, 강준서, 신규현, 이준서, 손예찬 (5인 전원)

일어난 사건:
  1. 야간 공원 어설픈 3:2 농구 중 유성 낙하 목격. 다섯 명 전원 충격.
  2. 분화구 탐색 — 지름 3m, 유성 흔적 없음. 그냥 귀가.
  3. 다음날 차윤호가 공원에서 혼자 드리블 중 첫 미니맵 경험.
     강준서 위치가 머릿속에 '점'으로 느껴짐. 반사적 패스 성공.
  4. 저녁 편의점 재집결. 강준서가 먼저 "공중에서 안 내려가는 느낌" 꺼냄.
  5. 차윤호 등 돌려 노룩 패스 시도 → 성공.
  6. 신규현: "어제 그거 때문인 거 아니야." — 다섯 명 침묵.

감정적 결과:
  차윤호: "착각이겠지"에서 "...모르겠다"로. 아직 능력이라 확신 못 함.
  강준서: 자기 경험(0.1초 부양)을 말했지만 다른 넷에게 확인 못 받음.
  신규현: 유성과 이상한 경험의 연관성을 가장 먼저 짚음.
  이준서·손예찬: 차윤호 노룩 패스를 "운이 좋은 거 아니야?"로 받아들이는 상태.

다음 화로 넘어간 궁금증:
  - "차윤호 방금 한 게 진짜 능력인 건가, 운인 건가?"
  - "강준서의 0.1초 부양도 실제 능력인가?"
  - "다른 세 명한테도 뭔가 있는 건가?"
  - "유성은 어디 갔나?"
```

## 확정된 세계관 사실

```
[이번 화에서 새로 확정된 사실]
- 유성 낙하 일시: 여름 방학 중 밤 10시 12분 (뉴스 보도 기준)
- 낙하 지점: 동네 공원 옆 공터. 분화구 지름 약 3m.
- 유성 실물은 회수 불가능 (현장에 없음). 다음날 공원 관리원이 펜스 설치.
- 차윤호 첫 미니맵 발동 조건: 의식적으로 집중 + 주변에 사람이 있을 때.
  점의 형태로 방향·거리감이 느껴짐. 발동 후 자연 소멸.
- 강준서의 0.1초 부양: 1화에서 본인이 언급만. 타인 확인 미완.
```

## 캐릭터 상태 업데이트

```
차윤호:
  현재 상태: 능력 반신반의. "착각이겠지"에서 "...모르겠다"로 이동.
  마지막 위치: 편의점 평상 앞.
  다음 화 진입 감정: 찜찜함 + 알아보고 싶은 마음.

강준서:
  현재 상태: 자기 경험을 말했지만 확인 못 받음. 차윤호 노룩 패스에 당혹.
  마지막 위치: 편의점 앞, 공 들고 서 있음.
  다음 화 진입 감정: "야 이거 진짜인 것 같은데"에 가까워진 상태.

신규현:
  현재 상태: 가장 먼저 유성과 능력의 연관성 짚음. 자기 능력 발현 여부 미확인.
  마지막 위치: 편의점 평상.
  다음 화 진입 감정: 조용한 관찰 모드.

이준서:
  현재 상태: 차윤호 패스를 "운 좋은 거"로 받아들임. 자기 능력 미발현.
  마지막 위치: 편의점 평상.

손예찬:
  현재 상태: 상황 파악 중. 자기 능력 미발현.
  마지막 위치: 편의점 평상.
```
