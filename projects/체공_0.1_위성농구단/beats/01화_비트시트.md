```
[도입 상황]
밤 11시, 텅 빈 동네 공원 농구코트.
차윤호와 강준서가 1:1을 하고 있다.
별다를 것 없는 여름밤.
    ↓ [하지만]
하늘에서 뭔가 떨어진다.
눈에 보일 정도의 빛줄기. 공원 외곽 수풀에 처박히는 소리.
    ↓ [하지만]
두 사람은 3초 쳐다보다가 농구를 계속한다.
"야 저거 뭐야." / "몰라. 슛이나 해."
    ↓ [그러므로]
공원이 떠들썩해지기 전에 두 사람은 집에 간다.
("야 경찰 오기 전에 가자" — 강준서)
    ↓ [그러므로]
다음날 아침 — 차윤호가 침대에서 일어나려다 이상한 감각을 느낀다.
눈을 떴는데, 강준서가 문자를 보내기 0.2초 전에 이미 핸드폰이 울릴 것 같다는 느낌.
그리고 실제로 울린다.
    ↓ [하지만]
'그냥 우연이겠지' 하고 넘어간다.
    ↓ [그러므로]
저녁에 다시 공원에서 만난 두 사람.
강준서도 "나 오늘 좀 이상해. 뭔가 자꾸 보여" 라며 찜찜해함.
    ↓ [하지만]
확인하려고 1:1을 시작하는 순간, 차윤호가 드라이브 들어가다 점프했을 때
공중에서 시간이 잠깐 늘어난 느낌을 받는다.
착지 후 두 사람 동시에: "야."
    ↓ [그러므로]
1화 끝: 두 사람이 서로를 보며 "...이거 어제 그거 때문인 거 아니야?"
```

```
- "이 두 사람은 이걸 어디에 쓸 생각인가?"

주인공 내면 변화:
  차윤호: "별거 아니다" → "...뭔가 이상한 게 생겼다" (아직 능력으로 인정 안 함)
  강준서: "나 천재인가봐" (이미 능력으로 확신 중)

독자 감정 목표: 가벼운 흥미 + 두 캐릭터의 케미 파악 + "다음에 어떻게 쓰지?" 기대감
```

## 장면 구성 (Scene Breakdown)

### Scene 1 — 밤 11시 동네 공원 (1화의 60%)
```
장소: 동네 공원 농구코트 (야간 조명 2개 중 1개 고장난 상태, 반쪽 조명)
등장인물: 차윤호, 강준서
비트: 평범한 심야 1:1 중 위성 낙하 목격 → 3초 구경 후 농구 재개

감정 흐름:
  - 두 사람 특유의 가벼운 신경전으로 시작
  - 위성 낙하 목격 시: 잠깐의 정적 + 황당함
  - 농구 재개: 아무 일 없었다는 듯이 — 이 태도가 두 캐릭터의 성격을 압축해서 보여줌

묘사 포인트:
  - 코트 바닥에 튀는 공 소리, 운동화 끽 소리, 여름밤 매미 소리
  - 하늘에 빛줄기 → 수풀 쪽에서 쿵 소리 + 흙먼지
  - 강준서가 공 잡고 하늘 보는 자세, 차윤호는 자세 유지한 채 옆눈으로만 봄
  - "경찰 오기 전에 가자"고 먼저 말한 건 강준서지만 먼저 뛴 건 차윤호
```

### Scene 2 — 다음날 오전, 차윤호 방 (1화의 15%)
```
장소: 차윤호 자취방 or 집 침대 위
등장인물: 차윤호 (단독)
비트: 첫 번째 능력 힌트 — 강준서 문자 오기 0.2초 전에 울릴 것 같다는 감각

감정 흐름:
  - 처음엔 그냥 우연이라 생각
  - 두 번째 비슷한 상황 (엄마가 부르기 직전, 뭔가 알 것 같은 느낌) → 찜찜해짐
  - "그냥 잠을 잘 잤나" 하고 넘김

묘사 포인트:
  - 좁은 방, 여름 더위, 선풍기 소리
  - 핸드폰 화면 — 강준서: "야 공원 나와"
  - 차윤호의 내적 독백은 짧고 건조하게
```

### Scene 3 — 저녁, 다시 공원 (1화의 25% — 클리프행어 포함)
```
장소: 공원 농구코트 (오늘은 조명 두 개 다 켜져 있음)
등장인물: 차윤호, 강준서
비트: 두 사람이 각자의 이상한 감각을 말하다가 → 1:1 중 차윤호 첫 체공 연장 경험

감정 흐름:
  - 강준서가 먼저 "나 오늘 좀 이상해" 꺼냄 → 차윤호도 반응
  - 확인하려고 바로 1:1 시작 (말보다 행동이 먼저인 두 사람)
  - 차윤호가 드라이브 점프 → 공중에서 시간 늘어나는 감각 → 착지
  - 두 사람 동시에 "야."
  - 강준서: "야 이거 어제 그거 때문인 거 아니야?"
  - 차윤호: (한 박자 후) "...모르겠어. 다시 해봐."

클리프행어:
  다시 점프 → 이번엔 확실히 느껴진다.
  차윤호가 착지하면서 강준서를 보는 눈빛 — 말 없이 끝.
  (독자: "이게 진짜 초능력이야??" 상태로 1화 종료)
```

## 집필 핵심 지침

```
톤 절대 원칙:
- 무겁지 않게. 위성 낙하도 "야 저게 뭐야" "몰라 슛이나 해" 수준으로.
- 두 캐릭터의 케미가 첫 화의 핵심. 사건보다 대화 리듬이 먼저.
- 능력 묘사는 거창하지 않게 — "뭔가 공중에서 시간이 늘어난 것 같았다" 수준.

Show, Don't Tell 포인트:
- 차윤호의 진지함 → 대사 아닌 행동으로 (먼저 드리블, 먼저 자세 잡기)
- 강준서의 낙천성 → 황당한 상황에서도 웃는 것보다 말이 먼저 나오는 것
- 두 사람의 친밀도 → 설명 없이 호흡으로 보여줌 ("야." 한 마디로 통하는 관계)

리듬:
- Scene 1은 빠르게, 대화 중심으로 속도감 있게.
- Scene 2는 짧고 건조하게 (차윤호 단독 장면은 과하게 묘사하지 않음).
- Scene 3는 후반부로 갈수록 대사가 줄고 행동과 감각 묘사로 채움.

금지:
- "갑자기", "그리고 나서", "정말로", "굉장히"
- 위성 낙하를 과도하게 드라마틱하게 묘사
- 능력을 처음부터 너무 깔끔하게 설명하는 것 (독자도 함께 어리둥절해야 함)
```

## 서사 감사관 검수 체크리스트

- [ ] 두 캐릭터의 말투가 보이스 시트와 일치하는가?
- [ ] 위성 낙하 장면이 가볍고 자연스럽게 처리되었는가?
- [ ] 능력 힌트가 너무 명확하지 않게 (자연스럽게) 심어졌는가?
- [ ] 1화 끝에 독자가 "다음 화 보고 싶다"고 느낄 훅이 있는가?
- [ ] Show, Don't Tell 원칙이 지켜졌는가?
- [ ] 무거운 설명 없이 두 사람의 관계가 전달되었는가?
