## 외형

```
체형: 178cm, 다부지고 빠른 몸. 어깨는 차윤호보다 좁지만 순발력이 보임.
외모 특징: 귀여운 인상이라 만만해 보이지만 코트 위에서는 눈빛이 달라짐.
           땀을 많이 흘리는 체질. 항상 수건을 목에 걸고 다님.
주로 입는 옷: 밝은 색 농구복. "눈에 잘 띄어야 패스 받지"가 철학.
특징적 포인트: 왼손 손목에 파란 고무밴드 (사실 별 의미 없음. 이유 물어보면 "그냥요").
```

## 성격 및 말투

### 핵심 성격 키워드
```
낙천적 / 입이 가벼움 / 순발력 / 눈치 빠름 / 겉으론 덤덤한데 속으론 열정 폭발
```

### 말투 특징
```
- 반말 (차윤호에게). 모르는 사람에게도 반말에서 존댓말로 자연스럽게 왔다갔다.
- 자주 쓰는 표현: "야", "진짜?", "ㅋㅋ 아니 잠깐만", "오 씨", "됐고"
- 말버릇: 생각하면서 말하는 타입. 문장 중간에 "...아니" 로 방향 틀기.
- 감정 격해질 때: 갑자기 조용해지거나, 반대로 평소보다 두 배 빠르게 말함.
- 절대 안 쓰는 표현: 진지하게 긴 독백. 감정을 직접 이름 붙여서 설명하는 것.
```

### 대화 예시
```
평소:
  "야 저거 봐. 위성 아니야?"
  차윤호: "응."
  "...아 그래? 근데 왜 이렇게 가까이 떨어지냐. 야 슛 안 해?"

능력 처음 발동했을 때:
  "아니 방금 나 걔가 왼쪽으로 올 줄 알았어. 근데 진짜 왔어. 이게 무슨..."
  (0.5초 침묵)
  "야 나 천재인가봐."

능력 잘못 써서 실패했을 때:
  "아니 잔상이 두 개였어. 두 개. 어떻게 알아."

차윤호한테 잔소리 들을 때:
  "알아알아. 근데 들어갔잖아?"

경기 막판 집중할 때 (갑자기 말 없어짐):
  "..."
  (패스 하나로 상황 정리)
```

## 배경 스토리

```
과거:
  운동 신경은 좋았는데 딱히 뭘 제대로 배운 적 없음.
  초등학교 때 농구공 처음 잡은 뒤로 그냥 좋아서 혼자 함.
  차윤호랑은 공원에서 자연스럽게 친해진 케이스.

현재 상황:
  대학 진학 의지는 있지만 우선순위가 애매함.
  알바하면서 번 돈으로 농구화 샀다는 게 현재 인생 요약.

숨겨진 비밀:
  능력이 생기기 전에도 이미 "뭔가 보인다"는 감각이 약하게 있었음.
  근데 그게 초능력인지 감각인지 구분을 못 했음.
  (→ 비밀 3 복선: 위성이 강준서의 잠재 능력을 '깨운' 것일 수 있음 — 30화 이후 암시)

트라우마:
  한 번 팀 경기에서 "야, 너 그냥 혼자 하지 마"라는 말을 들은 적 있음.
  지금도 패스 타이밍에 지나치게 신경 씀.
```

## 욕망과 두려움

```
표면적 욕망: 이기는 게 재밌다. 잘하면 더 재밌다.
진짜 욕망: 차윤호와 함께 "와, 저 팀 뭐야"라는 소리를 듣는 것.
           혼자 잘하는 게 아니라 같이 잘하고 싶음.
가장 큰 두려움: 능력을 쓰다가 상대를 잘못 읽고 중요한 순간에 팀을 망치는 것.
               (이미 한 번 있었고, 그게 3단계 위기 원인 중 하나)
```

## 농구 스타일 & 능력 활용

```
포지션: PG / SG (콤보 가드)
주특기: 노룩 패스, 속공 전환, 상대 리듬 끊기
약점: 체력이 후반에 떨어지면 판단력 흐려짐. 슛 성공률이 전반>후반 편차 큼.

미래 잔상 주요 활용:
- 드리블 돌파 전에 수비 이동 방향 미리 읽어 역방향 패스
- 인바운즈 상황에서 팀원 이동 예측해 타이밍 패스
- 스틸 시도 — 상대가 어디로 드리블할지 미리 손 뻗기

잘못 쓰면:
- 잔상이 두 가지 방향으로 겹칠 때 잘못 읽어 엉뚱한 방향으로 패스
- 수비 예측 믿다가 실제로 다른 선택을 한 상대에게 스틸 당함
- 집중력 떨어진 상태에서 발동 → 잔상 왜곡 → 위치 감각 일시 상실
```

## 성장 아크

```
1단계 (도입):
  "능력이 있다 → 나 좀 되네?" — 과하게 자신감 넘침.

3단계 (위기):
  잔상을 잘못 읽어 결정적 패스 실패. 팀 패배.
  처음으로 "능력이 있어도 내가 못 쓰면 의미 없다"를 체감.

5단계 (결말):
  능력이 잠깐 봉인된 상태에서 '감각'만으로 타이밍 패스를 성공시킴.
  "어? 이거 능력 아니었는데."
```
