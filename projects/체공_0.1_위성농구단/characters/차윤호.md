## 외형

```
체형: 182cm, 군살 없는 탄탄한 체형. 어깨 넓음.
외모 특징: 눈빛이 좀 날카로운 편인데 실제 성격은 그렇게까지 날카롭지 않음.
           땀 흘릴 때 앞머리가 이마에 달라붙음 — 경기 장면에서 자주 등장.
주로 입는 옷: 농구 반바지 + 헐렁한 티셔츠. 색깔 조합에는 무신경.
특징적 포인트: 농구화가 항상 한 사이즈 크게 구겨져 있음. 강준서가 항상 지적함.
```

## 성격 및 말투

### 핵심 성격 키워드
```
진지함 / 은근 허당 / 승부욕 / 과묵한 척하지만 사실 할 말 많음 / 책임감
```

### 말투 특징
```
- 반말 사용 (강준서에게). 낯선 사람에겐 짧은 존댓말.
- 자주 쓰는 표현: "그래서", "...됐어", "해보면 되지", "알잖아"
- 말버릇: 말 끝을 잘 흐림. "...뭐 어쩌라고" 같은 식.
- 감정 격해질 때: 오히려 더 말이 없어짐. 대신 행동으로 보여줌.
- 절대 안 쓰는 표현: 과장된 감탄사 ("대박!", "헉!"). 그냥 "오." 정도.
```

### 대화 예시
```
평소:
  강준서: "야 저거 진짜 위성 아니야?"
  차윤호: "응." (슛 쏘면서)

농구 이야기할 때:
  "그 픽 타이밍이 0.3초 늦었어. 봤어?"

능력 발동 직전 (혼잣말):
  "...지금이다."

당황했을 때:
  "...잠깐만." (말 없이 상황 정리 중)

화났을 때 (드물게):
  "그러니까 내가 아까부터 그 얘기를 하고 있었잖아."

강준서가 또 사고쳤을 때:
  "...나는 아무것도 못 봤다."
```

## 배경 스토리

```
과거:
  고등학교 때 학교 농구팀에 들어가려 했으나 "기본기가 부족하다"는 이유로 떨어짐.
  그 후로 혼자 공원에서 연습. 강준서랑 친해진 것도 그때부터.

현재 상황:
  재수 중이지만 공부보다 농구에 더 많은 시간을 씀. 본인도 알고 있음.

숨겨진 비밀:
  사실 농구팀 탈락 이후로 "나는 재능이 없다"고 생각해왔음.
  능력이 생겼을 때 솔직히 '이거면 되겠다'고 생각한 것에 스스로 죄책감을 느낌.

트라우마:
  중요한 순간에 "기술이 아직 안 됩니다"라는 말을 들어온 것.
  그래서 능력에 기대고 싶은 자신과, 실력으로 인정받고 싶은 자신이 충돌함.
```

## 욕망과 두려움

```
표면적 욕망: 농구를 잘하고 싶다. 이기고 싶다.
진짜 욕망: 누군가에게 "야, 쟤 진짜 잘하네"라는 말을 듣고 싶다. 능력 없이도.
가장 큰 두려움: 능력이 없어지면 자기가 아무것도 아닌 것이 드러나는 것.
```

## 농구 스타일 & 능력 활용

```
포지션: SG / SF (스윙맨)
주특기: 드라이브-인 피니시, 중거리 풀업 점프샷
약점: 외곽 3점 성공률 불안정, 픽앤롤 수비 판단 느림

체공 연장 주요 활용:
- 드라이브 후 레이업에서 블로커 타이밍을 피하기 위해 공중에서 0.1초 더 버팀
- 풀업 점프샷에서 수비의 손이 내려가는 순간을 골라 릴리즈
- 3점 바운더리 근처에서 페이더웨이 발동 시 공중 시간 극대화

잘못 쓰면:
- 레이업 코스가 애매한 상태에서 체공 늘렸다가 오히려 수비에 더 오래 노출됨
- 피로 상태에서 발동 → 어지럼증으로 착지 비틀거림 (강준서에게 "야 괜찮아?" 소리 들음)
```

## 성장 아크

```
1단계 (도입):
  "능력 생겼다, 써보자" — 아직 능력과 실력의 경계를 고민 안 함.

3단계 (위기):
  중요한 경기에서 능력 과부하로 쓰러진 후 처음으로 진지하게 생각함.
  "나는 능력 없이도 할 수 있나?"

5단계 (결말):
  "할 수 있는 건 다 한다. 능력도 실력도." — 죄책감 없이 자기 걸 자기 것으로 수용.
  처음으로 경기 후에 먼저 웃음.
```
