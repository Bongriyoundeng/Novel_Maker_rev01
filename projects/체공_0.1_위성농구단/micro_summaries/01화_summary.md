## 마이크로 요약 (2문장)

```
차윤호와 강준서가 심야 공원 1:1 중 위성 낙하를 목격하고
별로 신경 쓰지 않은 채 귀가했으나, 다음날부터 이상한 감각이 발현되기 시작했다.
저녁 공원에서 다시 만난 두 사람은 각자의 감각을 확인하기 위해 1:1을 시도했고,
차윤호의 체공 연장과 두 사람의 미래 잔상 능력이 모두 실재함을 확인했다.
```

## 확정된 새 사실

```
세계관 / 설정 추가:
- 동네 공원 코트: 야간 조명 2개 (오른쪽이 자주 고장남)
- 위성 낙하 위치: 공원 북쪽 수풀
- 능력 발현 시점: 위성 낙하 다음날부터
- 차윤호 능력: 체공 연장(주력) + 미래 잔상(보조)
- 강준서 능력: 미래 잔상(주력) + 체공 연장(보조)
- 두 능력 모두 1화 기준 '규칙 미파악' 상태

밝혀진 정보: 없음 (비밀 1·2·3 모두 미공개 유지)

새 등장 장소: 동네 공원 농구코트, 차윤호 자취방, 동네 골목
```

## 미회수 복선 & 열린 궁금증

```
새로 심은 복선:
- 수풀에 처박힌 위성의 실체 (독자: "저게 진짜 뭐야?")
- 차윤호가 강준서의 잔상도 볼 수 있다는 것 (미래 잔상이 양방향임을 독자에게 힌트)

독자에게 남긴 궁금증:
- "이 능력, 의도적으로 쓸 수 있어?"
- "얼마나 강해질 수 있어?"
- "들키면 어떻게 돼?"

다음 화 인수 상황:
- 두 사람이 능력 확인 직후 상태. 규칙 파악 실험 시작 예정.
- "다시 해봐" 대사에서 바로 이어짐.
```

## 델타 타일 (2화 집필 시 사용)

```
[델타 타일 - 1화 완료 기준]

마이크로 요약:
심야 공원 농구 중 위성 낙하를 목격한 차윤호·강준서는 다음날부터 이상한 능력이
발현됨을 확인했다. 차윤호는 0.1초 체공 연장을, 강준서는 0.2초 미래 잔상을 주력으로
가지며, 둘 다 상대방 능력도 보조적으로 발동됨이 확인됐다.

변경된 인물 상태:
- 차윤호: 능력 발현 확인. 규칙 미파악. 위치: 동네 공원.
- 강준서: 능력 발현 확인. 규칙 미파악. 위치: 동네 공원.

새로 확정된 사실:
- 공원 북쪽 수풀에 위성(또는 불명 물체) 낙하
- 두 능력 모두 실재하며, 의도적 발동 가능 여부는 미파악

독자에게 열린 궁금증:
- 능력의 규칙(한도·부작용)이 뭔지
- 이걸 농구에 어떻게 쓸 건지
- 위성의 정체
```
