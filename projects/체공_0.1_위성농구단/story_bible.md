## 로그라인

```
동네 공원에서 심야 농구를 하다가 위성 낙하 현장을 목격한 차윤호·강준서,
다음날부터 "0.1초 체공 연장"과 "0.2초 미래 잔상"이라는 애매하지만 치명적인 초능력을 얻는다.
이 두 명이 동네 픽업 게임부터 전국 무대까지,
능력을 죄다 농구에 쏟아부으며 진짜 선수가 되어 가는 이야기.
```

## 5단계 마일스톤

### 1단계 — 도입 (1화 ~ 6화) : "위성이 뭔데"
- 반드시 일어나야 할 사건 (하지만/그러므로 인과):
  - 심야 공원 농구 중 하늘에서 위성이 떨어져 공원 한 켠에 처박힌다.
  - **하지만** 두 사람은 거의 신경 쓰지 않고 농구를 계속한다 ("야 저거 봤어?" "응 슛이나 해").
  - **그러므로** 다음 날부터 이상한 능력이 발현되기 시작하지만, 처음엔 우연이라 생각한다.
  - **하지만** 동네 픽업 게임에서 능력이 너무 노골적으로 터지며 "이거 진짜 초능력이다"를 확인한다.
  - **그러므로** 두 사람이 능력의 규칙을 파악하기 위해 실험을 시작한다.
- 독자에게 심어야 할 궁금증:
  - "위성이 진짜 우연히 떨어진 건가? 누가 보낸 건가?"
  - "이 능력 쓰다가 걸리면 어떻게 되지?"
  - "동네 수준에서 전국으로 진짜 갈 수 있어?"
- 클리프행어 포인트:
  6화 끝 — 구경하던 동네 고참 아저씨("원로")가 "너희 둘 이번 주말 3:3 대회 나올 수 있어?" 라고 묻는다.
  두 사람의 반응: (서로 눈빛 교환) "...어."
- 완료 여부: [ ]

### 2단계 — 전개 (7화 ~ 22화) : "이게 되네?"
- 반드시 일어나야 할 사건:
  - 3:3 동네 대회 → 5:5 지역 픽업 리그 참가 (팀 멤버 2~3명 추가 영입).
  - 능력의 규칙 발견: 한 번 발동에 에너지 소모. 남용 시 심한 어지럼증 + 다음날 능력 봉인.
  - 강준서가 능력을 잘못 읽어 대참사를 일으키는 코믹 에피소드.
  - 라이벌 등장: 재수 중인 전직 고교 에이스 "이재혁" — 능력 없이 순수 실력으로 두 사람을 압도.
  - 차윤호가 처음으로 "우리 진짜 기본기부터 다시 해야 하나?" 생각하기 시작.
- 주인공 성장 / 변화:
  능력을 "발동하면 짱"에서 "언제·어떻게 쓰느냐가 핵심"으로 인식 전환.
- 클리프행어 포인트:
  22화 끝 — 이재혁이 "너희가 그렇게 잘하면 전국 아마추어 대회 나와봐. 나도 나가거든" 하고 돌아선다.
  강준서: "전국이요?" / 차윤호: "...나가자."
- 완료 여부: [ ]

### 3단계 — 위기 (23화 ~ 36화) : "이게 안 되네?"
- 반드시 일어나야 할 사건:
  - 전국 아마추어 대회 예선 진출 (팀명 공식 등록: "위성농구단" — 강준서가 지었고 차윤호는 반대).
  - 예선에서 진짜 "연습만 한" 팀들을 만나며 기본기 차이를 처절하게 체감.
  - 차윤호가 중요한 경기에서 능력 과부하로 쓰러지는 사건 → 팀이 패배.
  - 두 사람 최대 갈등: "능력 쓰지 말고 실력으로만 가자" vs "쓸 수 있는 걸 왜 안 써".
  - 별개로: 위성 낙하를 조사하는 수상한 사람이 두 사람 주변을 맴돌기 시작 (가볍게, 코믹하게).
- 최대 위기 / 갈등:
  준결승 직전 차윤호가 능력 봉인 상태로 경기에 나서야 하는 상황.
- 클리프행어 포인트:
  36화 끝 — 준결승 상대가 이재혁의 팀임이 확인된다.
  강준서: "아 씨, 쟤잖아." / 차윤호: (손목 테이핑 감으며) "알아."
- 완료 여부: [ ]

### 4단계 — 절정 (37화 ~ 46화) : "가자"
- 반드시 일어나야 할 사건:
  - 이재혁과의 준결승 — 전반은 순수 실력 싸움, 후반에 능력을 최소한으로 절묘하게 융합.
  - 이재혁이 경기 후 두 사람을 인정하는 장면 (라이벌 → 서로 존중).
  - 결승: 전국 최강팀 상대. 초반 완전히 밀림.
  - 두 능력의 완벽한 시너지 순간 발현 — "차윤호가 체공 중 강준서의 잔상을 보고 노룩 패스".
- 클라이맥스 장면:
  잔여 시간 3초, 2점 차 역전 필요. 차윤호가 0.1초 체공 + 강준서의 잔상이 겹치는 순간.
  둘이 동시에 같은 것을 '느끼는' 장면.
- 클리프행어 포인트:
  46화 끝 — 버저 비터 직전 컷. "터졌어?" / "몰라. 지금 날고 있어."
- 완료 여부: [ ]

### 5단계 — 결말 (47화 ~ 50화) : "우리 진짜 됐다"
- 모든 복선 회수 목록:
  - 위성 낙하의 출처 (마지막 화에 코믹하게 공개: 정부 위성이 오작동으로 떨어진 것. 누구도 몰랐음.)
  - 강준서가 능력 쓰다가 실패했던 장면들의 의미 — 알고 보면 그게 기술의 연습이었음.
  - "원로 아저씨"가 두 사람의 첫 3:3을 지켜본 이유 (알고 보면 본인도 실력자였음).
- 정서적 보상 장치:
  우승 후 두 사람이 텅 빈 동네 공원으로 돌아와 다시 1:1을 하는 엔딩 장면.
- 독자 카타르시스 포인트:
  우승 + 스카우트 제안 + "그래서 능력이 없었어도 됐을 것 같아?" "...글쎄. 근데 재밌었잖아."
- 완료 여부: [ ]

## 초능력 시스템 규칙 (설붕 방지 핵심)

```
[능력 1] 체공 연장 (차윤호 주력 / 강준서도 보유)
- 점프 후 공중에 있는 순간, 최대 0.1초 체공 시간을 의도적으로 늘릴 수 있다.
- '원하는 시점'에 발동 가능 — 올라갈 때도, 정점에서도, 내려오기 직전에도.
- 발동 횟수 제한: 1경기(혹은 강도 높은 활동 기준) 최대 5회.
- 초과 사용 시 즉각 어지럼증 + 다음 발동까지 딜레이 2초 발생.
- 3회 연속 발동 시 그날 이후 능력 봉인 (다음날 자정에 해제).
- 겉보기엔 "점프력이 좋은 선수" 수준. 위협적으로 보이지 않음.

[능력 2] 미래 잔상 (강준서 주력 / 차윤호도 보유)
- 상대방 또는 팀원이 0.2초 후에 있을 위치가 '흐릿한 잔상'으로 보인다.
- 잔상은 희미하고 겹쳐 보임 — '정확한 예측'이 아닌 '가능성 힌트' 수준.
- 동시에 볼 수 있는 잔상: 최대 3명분. 코트 전체를 보면 흐릿해짐.
- 발동 횟수 제한: 1경기 최대 7회. 초과 시 두통 + 시야 왜곡.
- 3회 연속 발동 시 30초간 능력 봉인.

[공통 규칙]
- 극도로 피로하거나 긴장 상태에서 발동률 저하.
- 두 능력이 동시에 발동되면 시너지 (차윤호 체공 중 강준서 잔상 시야 공유) — 단, 50화 전까지 1~2회만 가능.
- 능력이 공식적으로 드러나면 안 됨. 겉보기엔 '그냥 엄청 감각 좋은 선수'.

[절대 금지 설정]
- 능력 발동으로 상대를 직접 다치게 하는 전개.
- 능력이 경기 결과를 혼자 만들어내는 전개 (항상 실력+능력 조합이어야 함).
- 능력이 심판이나 관중에게 명확히 목격되는 전개.
```
