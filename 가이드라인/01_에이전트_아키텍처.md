# 가이드라인 1: 에이전트 아키텍처 개요

## 왜 다중 에이전트 시스템인가?

AI가 생성한 소설이 "재미없다"는 평가를 받는 이유는 문장력 부족이 아닙니다.
**서사 구조의 통일성 결여**, **캐릭터 심리의 일관성 부재**, **통계적 평균값으로의 수렴**이 핵심 문제입니다.

하나의 AI에게 "재미있는 소설 써줘"라고 요청하는 방식은 이 문제를 해결하지 못합니다.
대신 각 역할에 특화된 에이전트들이 서로 견제하고 협력하는 **에이전틱 워크플로우**가 필요합니다.

---

## 4개 핵심 에이전트 역할

| 에이전트 | 파일 | 핵심 역할 | 주요 산출물 |
|---------|------|---------|----------|
| 플롯 건축가 | `agents/01_plot_architect.md` | 서사 구조 & 아크 설계 | 시놉시스, 비트 시트 |
| 세계관 제작자 | `agents/02_world_builder.md` | 설정 관리 & 바이블 유지 | 세계관 규칙, 업데이트된 story_bible |
| 산문 스타일리스트 | `agents/03_prose_stylist.md` | 실제 문장 생성 | 4단계 레이어 완성 원고 |
| 서사 감사관 | `agents/04_narrative_auditor.md` | 품질 검수 & 설붕 감지 | 감사 보고서 |

## 사령관(Commander)의 위치

```
사용자(사령관)
      │
      ├── 목표 설정 & 결말 확정
      ├── 에이전트 산출물 승인 / 반려
      ├── 결정 경계 항목 최종 결정
      └── 킬 스위치 수신 & 판단
```

사령관은 펜을 들지 않습니다. **명령·승인·검수**가 전부입니다.
사령관 운영 프로토콜은 `agents/00_commander.md` 참조.

## 표준 워크플로우 다이어그램

```
[프로젝트 시작]
      │
      ▼
플롯 건축가: 시놉시스 & 마일스톤 설계
      │
      ▼ [사령관 승인]
세계관 제작자: 세계관 규칙 + story_bible.md 초안
      │
      ▼ [사령관 승인]
      │
────────────────────────────────────────
[화별 반복 사이클]
      │
      ▼
플롯 건축가: 화별 비트 시트
      │
      ▼ [사령관 승인]
      │
      ▼
산문 스타일리스트: 4단계 레이어 집필
      │
      ▼
서사 감사관: 감사 보고서
      │
      ├─ 통과 ──────────────────────────┐
      │                                │
      └─ 수정 필요 → 산문 스타일리스트 수정 ─┤
                                       │
      ▼ [사령관 최종 확정]               │
세계관 제작자: story_bible.md 업데이트 ◄──┘
      │
      ▼
마이크로 요약 작성 (micro_summary.md)
      │
      ▼
다음 화 루프 시작
```

## 에이전트 간 정보 흐름

```
story_bible.md ◄──────────────────────── 세계관 제작자 (업데이트)
      │
      ├──► 플롯 건축가 (참조)
      ├──► 산문 스타일리스트 (참조)
      └──► 서사 감사관 (검증 기준)

beat_sheet.md ◄────── 플롯 건축가 (작성)
      │
      └──► 산문 스타일리스트 (집필 기준)

micro_summary.md ◄─── 매 화 완료 후 작성
      │
      └──► 다음 화 모든 에이전트 (컨텍스트)
```

## 에이전트 호출 판단 가이드

| 상황 | 호출할 에이전트 |
|------|--------------|
| 시놉시스가 없다 | 플롯 건축가 |
| 세계관 규칙이 필요하다 | 세계관 제작자 |
| 화별 비트를 짜야 한다 | 플롯 건축가 |
| 실제 원고를 써야 한다 | 산문 스타일리스트 |
| 원고 품질을 검토해야 한다 | 서사 감사관 |
| 설정 오류가 의심된다 | 세계관 제작자 + 서사 감사관 |
| 장기 연재 후 전체 점검이 필요하다 | 서사 감사관 (사후 감사) |
