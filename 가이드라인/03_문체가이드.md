# 가이드라인 3: 문체 가이드라인

## 핵심 원칙

> AI 특유의 기계적인 말투를 지우고 자연스러운 문장을 얻으려면 **페르소나**와 **제약 사항**을 섬세하게 조정해야 합니다.

---

## AI에게 줄 페르소나

```
당신은 10년 차 웹소설 전문 편집자이자 퇴고 전문가입니다.
기계적인 표현을 제거하고, 독자가 주인공에게 감정 이입할 수 있도록 문장을 매끄럽게 다듬는 것이 당신의 강점입니다.
인터넷 소설 특유의 빠른 호흡과 몰입감을 최우선으로 여깁니다.
```

---

## 문체 지침 상세

### 1. 대화와 행동 중심 (7:3 원칙)

**나쁜 예 (Tell - 설명 중심)**
```
지수는 화가 났다. 그녀는 민준이 자신을 배신했다는 사실을 알았고, 그것이 매우 분했다.
```

**좋은 예 (Show - 행동/대화 중심)**
```
지수는 컵을 테이블에 내려놓았다.
"괜찮아."
목소리가 너무 평온했다. 오히려 그게 이상했다.
민준이 한 발 뒤로 물러섰다.
```

### 2. 캐릭터별 고유 말투

각 캐릭터의 말투를 미리 정해두고 AI에게 고수하도록 지시합니다.

```
예시:
- 주인공 지수: 짧고 단호한 문장, 감정을 숨기는 편, 반말
- 남주 민준: 격식체와 반말 혼용, 차갑지만 가끔 솔직해짐
- 빌런 회장: 점잖은 척하는 존댓말, 위협적인 우회 표현
```

### 3. 인간적 불규칙성

AI에게 명시할 내용:
```
- 구어체와 신조어를 적절히 사용해줘
- 문장 길이를 불규칙하게 섞어줘 (짧은 문장 + 긴 문장)
- 감탄사, 말 잇기(...)도 활용해줘
- 완벽한 문법보다 자연스러운 리듬을 우선해줘
```

### 4. 호흡 조절

```
긴장 장면 → 문장 짧게 (1~5단어 문장 혼용)
감정 장면 → 문장 중간 길이
설명 구간 → 가급적 짧게, 대화에 녹여서
```

## AI 문체 교정 프롬프트

### 퇴고 요청
```
다음 원고를 퇴고해줘:
1. "매우", "굉장히", "정말로" 같은 부사 → 행동으로 대체
2. "~라고 생각했다", "~을 느꼈다" → 행동/묘사로 전환
3. 같은 단어 반복 → 다양한 표현으로 교체
4. 긴 설명 문단 → 대화나 짧은 장면으로 분산
5. 캐릭터 말투 일관성 검토

[원고 붙여넣기]
```

### 기계적 표현 제거 요청
```
다음 문장들을 더 자연스러운 한국어 웹소설 문체로 바꿔줘:
- "그는 상황을 분석하였다" → (더 자연스럽게)
- "그녀는 놀라움을 금치 못하였다" → (더 자연스럽게)
[추가 문장들]
```

## 피해야 할 AI 문체 패턴

```
X "그는 ~라고 말하였다" → O "그가 말했다" / O 직접 대화로
X "상황이 심각해져 가고 있었다" → O "상황이 심각해졌다"
X 단락마다 같은 길이의 문장 → O 길이를 불규칙하게
X 감정을 직접 서술 → O 신체 반응, 행동으로 표현
X 과거완료형 남용 → O 현재진행형, 과거형 혼용
```
