# 가이드라인 4: 한국 웹소설 장르 문법

> 한국 웹소설 플랫폼을 타겟으로 한다면, 해당 시장에서 검증된 **서사 구조와 독자 기대 심리**를 에이전트에게 학습시켜야 합니다.

---

## 2026년 시장 트렌드 키워드

| 장르 | 핵심 트렌드 |
|------|-----------|
| 판타지 / 회귀물 | 시스템창, 스탯 공개, 성장 보상 체계 시각화 |
| 로맨스 판타지 | 주체적 여주, 전문직 설정, 역하렘, 빌런 남주 |
| 현대 판타지 | 하이퍼-전문직, 헌터물 변형, 현실 감각 강조 |
| 무협 | 문파 역학, 협의(俠義), 성장 서사 |
| 공통 | 클리프행어, 중독성 루프, 사이다, 정서적 보상 |

## 장르별 문법 가이드

### 회귀 / 빙의 / 환생 (회빙환)

**독자 기대 심리:**
- 주인공이 미래 지식으로 과거의 실수를 바로잡는 카타르시스
- '이번에는 달라'는 주인공의 각오와 그것이 무너지는 순간의 긴장감
- 나비 효과로 인한 예상치 못한 변수

**에이전트 지침:**
```
[플롯 건축가에게]
회귀자 주인공이 미래 지식을 사용할 때마다 나비 효과로 인한
예상치 못한 변수를 발생시켜줘. 주인공이 모든 것을 완벽하게
통제하면 긴장감이 사라진다. 반드시 계획이 부분적으로 어긋나는
요소를 포함할 것.
```

---

### 시스템 / 상태창 장르

**독자 기대 심리:**
- 주인공의 성장이 수치로 가시화되는 쾌감
- 노력과 보상의 인과관계가 명확할 것
- 스킬 획득, 레벨업의 중독성 루프

**에이전트 지침:**
```
[세계관 제작자에게]
주인공의 스탯 체계를 별도로 관리해줘.
스탯 수치는 구체적일수록 좋다 (예: "근력 47 → 53" not "근력 증가").
주인공이 겪는 시련과 보상이 논리적으로 일치하도록 유지해줘.
독자가 성장 커브를 예측하면서도 예상을 뛰어넘는 보상이 있어야 한다.

[산문 스타일리스트에게]
시스템 메시지는 소설 내 UI처럼 별도 형식으로 표시해줘:
───────────────────
[스킬 획득]
■ 철벽의 의지 (Lv.1)
  효과: 방어력 +15%, 넉백 무효화
───────────────────
```

### 로맨스 판타지 (로판)

**독자 기대 심리:**
- 주체적이고 매력적인 여주인공
- 남주의 집착과 차가움, 그 이면의 취약함
- 계급/신분 차이가 만드는 갈등과 해소
- '사이다' 장면 (무례한 인물에게 통쾌하게 복수)

**에이전트 지침:**
```
[플롯 건축가에게]
로판 장르에서 '사이다' 장면의 설계 원칙:
1. 독자가 충분히 '화'가 쌓여야 한다 (악당을 2~3화 동안 미워하게 만들기)
2. 해소는 예상보다 크게, 예상치 못한 방식으로
3. 여주인공이 직접 해결하거나 주도적 역할을 해야 한다
   (남주가 대신 해결하면 반쪽짜리 사이다)

귀족제 사회에서 현대적 가치관을 가진 주인공이 겪는 딜레마를
부각시켜줘. 너무 쉽게 귀족 사회에 적응하면 현실감이 떨어진다.
```

### 무협

**독자 기대 심리:**
- 무공 수련과 성장의 과정
- 문파 간 복잡한 이해관계
- 협의(俠義) 정신과 강호의 도덕률

**에이전트 지침:**
```
[세계관 제작자에게]
무협 세계관 설계 시 주의사항:
- 무공 이름의 나열보다 각 문파의 '철학'과 '정신'을 반영할 것
- 강호 세계관 특유의 도덕률(협의): 약자 보호, 은원을 반드시 갚음,
  문파의 명예를 목숨보다 중히 여김
- 주인공이 협의를 실천하는 과정에서 발생하는 현실적 대가도 묘사
  (협의는 자기희생을 수반한다)
```

## 공통 문법: '사이다'와 '주무기'

### 사이다 (카타르시스) 설계
```
사이다의 구조:
  갈등 축적 (2~4화) → 독자 분노/답답함 극대화 → 예상 초월한 해소

사이다를 위한 빌런 설계:
- 단순히 나쁜 것이 아니라 독자가 '진짜 짜증나는' 방식으로 행동
- 주인공이 반격할 때까지 적당히 우위를 유지
- 완전한 패배는 절정 구간에 배치
```

### 중독성 루프 설계
```
매 화 끝에 독자가 느껴야 하는 것:
  "이 궁금증은 반드시 다음 화에서 해결해야 해"

루프 유지 공식:
  [이번 화 궁금증 해소] + [더 큰 새 궁금증 제시]

단순히 클리프행어로 끝내는 것이 아니라,
독자가 캐릭터의 다음 선택을 '예측하고 싶어하는' 상태를 만드는 것이 목표.
```
